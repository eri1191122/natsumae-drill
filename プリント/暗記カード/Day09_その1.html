英単語

Day09
問題1｜単語①
textbook
よみ・意味は？
テキストブック ＝ 教科書
text（本文）＋ book。-xt の t を忘れない
⚠ textの -xt を落とさない（texbook は誤）
確認できた？
問題2｜単語②
classroom
よみ・意味は？
クラスルーム ＝ 教室
class（クラス）＋ room（部屋）
⚠ ひとつながりに書く（class room は誤）
確認できた？
問題3｜単語③
garden
よみ・意味は？
ガーデン ＝ 庭・庭園
gar-den（gar では「ガー」と伸ばす）
⚠ gardon / graden と書かない
確認できた？
問題4｜単語④
beach
よみ・意味は？
ビーチ ＝ 砂浜・海岸
ea で「イー」の音（teach / each と同じパターン）
⚠ beech（ブナの木）と混同しない
確認できた？
問題5｜単語⑤
afternoon
よみ・意味は？
アフタヌーン ＝ 午後
after（あと）＋ noon（正午）。 -oo- で「ウー」
⚠ afternooon と o を3つ書かない
確認できた？

特訓ブリッジ

B-1・B-2
問題14｜B-1 三単現s
Mika ( carry ) the bag every day.（三単現に直せ）
→ Mika           the bag every day.
carries
子音＋y → y を i に変えて es（-ies）
play → plays（母音＋y はそのまま s）と区別
確認できた？
問題15｜B-2 否定命令文
「ここで走るな。」を英語に直せ。
→           run here.
Don't
否定命令文＝ Don't + 動詞の原形…（文頭 D は大文字）
命令文＝ Run here. / 否定命令文＝ Don't run here.
確認できた？
