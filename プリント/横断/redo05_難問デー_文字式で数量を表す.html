数①｜L4 難問　いくつかの数量が混ざった代金を文字式で表す
1個 a 円 のりんごと、1個 b 円 のみかんがあります。
(1) りんごを 3個、みかんを 5個 買ったときの 代金の合計 を、a, b を使った式で表しなさい。
(2) (1)の代金から x 円 の値引きをしてもらい、さらに残りに 10%の消費税 がかかります。支払う金額 を a, b, x を使った式で表しなさい。（カッコを使ってよい）
⓪ 「文字式で数量を表す」ってなに？

やさしく言うと：数がわからないところを文字（a, b, x）にして、文章のとおりに「かけ算・たし算・ひき算」でつないでいくこと。「1個a円のりんご3個」なら a×3＝3a。文字式では ×の記号は書かず、数を文字の前に書く。

① 消費税10%のかけ方
  • 10%増し（税込み）＝もとの金額の1.1倍。もとが M 円なら税込みは M×1.1＝1.1M（円）。
  • 「10%引き」なら0.9倍だけど、今回は 税は足すので1.1倍。
1個p円のペンを2本買って50円引き、そこに10%税がつくなら？ → 代金 2p、値引き後 (2p−50)、税込み (2p−50)×1.1＝1.1(2p−50)（円）。「値引きしてから税」の順にカッコでまとめる。
値引き前の代金に税をかけてしまうのが最頻ミス。先に x 円引いて、残った金額に1.1をかける。順番が大事。
ステップ1(1)（代金の合計）りんご：1個a円が3個＝a×3＝3a（円）。みかん：1個b円が5個＝b×5＝5b（円）。合計＝3a＋5b（円）。×は書かず数を前に。
ステップ2(2)（値引きする）(1)の代金 3a＋5b から x 円引く＝3a＋5b−x（円）。これが「税がかかる前の、払うもとの金額」。
ステップ3(2)（10%の税をかける）税込み＝もとの金額×1.1。もとは (3a＋5b−x) だから、かたまりをカッコでくくって (3a＋5b−x)×1.1。×を消して 1.1(3a＋5b−x)（円）。これが支払う金額。
確かめa＝100, b＝80, x＝50 を入れる。代金＝3×100＋5×80＝300＋400＝700。値引き後＝700−50＝650。税込み＝650×1.1＝715円。式 1.1(3a＋5b−x)＝1.1×(300＋400−50)＝1.1×650＝715。ぴったり一致。
3a＋5b を「8ab」や「8(a＋b)」とまとめない（りんごとみかんは別の値段。3a と 5b は文字がちがうのでたせない）。税を 3a＋5b−x×1.1 と書かない（これだと x にだけ1.1がかかってしまう）。かたまり全体にかけるからカッコが必要。1.1 を「10」や「0.1」にしない（10%増しは0.1倍ではなく1.1倍）。
「文章を文字式に直す」「かたまりにカッコをつけて全体にかける」は、このあとの 方程式・連立方程式・1次関数 の文章題で、式を立てる土台になる。とくに「値引きしてから税」のように 順番のある計算をカッコで表す力は、割合の文章題でずっと効いてくる。塾のテストでも、この「混ざった代金を1つの式に」はL4の定番。
こたえ(1) 3a＋5b（円）　(2) 1.1(3a＋5b−x)（円）

数②｜L4 難問　数量の関係を等式・不等式で表す（方程式は解かない）
1本 x 円 のジュースを 6本 買い、1000円 出します。
(1) ジュース6本の代金が ちょうど1000円 になるときの関係を、等式 で表しなさい。
(2) 6本買って おつりが200円以上残る ようにしたいとき、その関係を 不等式 で表しなさい。
⓪ 「等式」と「不等式」ってなに？

やさしく言うと：等式は「＝（イコール）」で、左と右がちょうど同じことを表す。不等式は「>・<・≧・≦」で、どちらが大きい（小さい）かを表す。今回は 方程式を解くのではなく、関係を式にするだけでOK。

① 「以上・以下・より大きい」の記号
  • ○以上（その数を入れる）＝ ≧。例：200円以上 → ≧200
  • ○以下（その数を入れる）＝ ≦。
  • ○より大きい・○より多い（その数を入れない）＝ >。「未満」は <。
「おつり」って何？ → 出したお金 − 代金。1000円出して代金が y 円なら、おつり＝1000−y（円）。これが「200円以上」なら 1000−y ≧ 200。
「以上」を > にしてしまうのが最頻ミス。「以上・以下」は その数もふくむから ≧・≦。「より大きい・未満」だけが >・<。
ステップ1(1)（代金を文字式に）1本x円のジュース6本の代金＝x×6＝6x（円）。
ステップ2(1)（等式にする）「代金がちょうど1000円」＝代金と1000が等しい＝6x＝1000。これが等式（※ここでは x は求めなくてよい。関係を表すだけ）。
ステップ3(2)（おつりを式に）おつり＝出したお金−代金＝1000−6x（円）。
ステップ4(2)（不等式にする）「おつりが200円以上」＝おつりが200と等しいか、それより多い＝1000−6x ≧ 200。「以上」は ≧（200もふくむ）。
確かめ(1)：もしx＝150なら6x＝900で1000にならない。x＝10006のとき6x＝1000で成り立つ。式 6x＝1000 は正しく「代金＝1000」を表す。(2)：x＝100なら6x＝600、おつり＝1000−600＝400で、400≧200は成り立つ（OK）。x＝140なら6x＝840、おつり160で、160≧200は成り立たない（NG）。式 1000−6x≧200 が「200円以上残る条件」を正しく表している。
(1)を「6x」だけで終えない（等式は「＝」が必要。6x＝1000 まで書く）。(2)で「以上」を > にしない（以上は ≧、ちょうど200もふくむ）。おつりを 6x−1000 と逆に書かない（おつり＝出したお金−代金＝1000−6x。引く順番に注意）。
「ことばの関係（ちょうど・以上・以下）を ＝・≧・≦ の式に直す」のは、このあとの 方程式・不等式・連立方程式 の文章題のいちばん最初のステップ。ここで「立式」だけ正確にできるようにしておくと、あとは解くだけになる。「以上＝≧・より大きい＝>」の区別は高校の数学Ⅰまでずっと使う大事な約束。
こたえ(1) 6x＝1000　(2) 1000−6x ≧ 200

数③｜★さらに挑戦（できたらでOK）　L4　規則を見つけて n を使った式で表す
下の図のように、マッチ棒で 正方形を横一列にならべて いきます。
正方形が1個のときマッチ棒は4本、2個のとき7本、3個のとき10本…と増えていきます。
(1) 正方形を n 個 ならべたとき、マッチ棒の本数を n を使った式で表しなさい。
(2) マッチ棒を 100本 使うと、正方形は何個できますか。
⓪ 「規則を n の式にする」考え方

やさしく言うと：「1個ふえるごとに、いくつふえるか」を見つけるのがコツ。正方形が1個ふえるたびにマッチ棒が 3本ずつふえている（4→7→10）。この「毎回ふえる数3」が、n の前につく数になる。

① 数えかたで式を作る
  • 毎回ふえる数：7−4＝3、10−7＝3 → 3本ずつふえる。
  • 最初の1個で4本。あと (n−1) 個ふえるたびに3本ずつ → 4＋3(n−1)。
  • または「左はじの縦1本＋正方形ごとに3本（縦1本＋上下2本）」と見て 1＋3n。どちらも同じ式になる。
n＝1, 2, 3 を 3n＋1 に入れて確かめる → 3×1＋1＝4、3×2＋1＝7、3×3＋1＝10。表とぴったり合えば、その式で正しい。
「正方形1個で4本だから 4n」としてしまうのが最頻ミス。それだと2個で8本になり、表の7本と合わない。となりの正方形は1辺を共有するから、ふえるのは3本ずつ。
ステップ1(1)（ふえる数を見つける）4→7→10 と、3本ずつふえている。だから n の前につく数は3（＝3n の形）。
ステップ2(1)（式を完成）3n だけだと n＝1で3本、でも本当は4本。1本たりないので＋1する。マッチ棒の本数＝3n＋1（本）。
（左はじの縦1本が最初からあって、正方形ごとに「上・下・右の縦」の3本がふえる、と見ると 1＋3n＝3n＋1 で納得できる。）
ステップ3(2)（100本のとき）本数の式＝100 とおく：3n＋1＝100。ここは 式に数を入れてさかのぼるだけ。3n＝100−1＝99、n＝99÷3＝33。正方形は 33個。
確かめn＝33 を 3n＋1 に入れる → 3×33＋1＝99＋1＝100（本）。ちょうど100本で一致。表の規則（4,7,10,…）も 3n＋1 ですべて合う（n＝1→4, n＝2→7, n＝3→10）。
本数を 4n としない（となりどうしで縦の棒を共有するので、1個ふえても4本はふえない。ふえるのは3本）。3n＋1 の「＋1」を忘れない（n＝1で4本になるよう調整する1本）。(2)で 100÷4＝25 などとしない（式 3n＋1＝100 から n を求める。99÷3＝33）。
「ならび方の規則を n の式にする」は、このあとの 関数（比例・1次関数）・数列の考え方 につながる超重要テーマ。「毎回ふえる数」が n の前の数（変化の割合）になる、という感覚は中2・中3でそのまま使う。図形を数えて式にする力は、高校入試の規則性問題でも頻出。ここに挑戦できたのは大きな一歩。
