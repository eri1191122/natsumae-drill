「いっしょにテニスをしましょう。」
✏️ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
📖 文法のせつめい（読んでから下の答えへ。わかる子は読みとばしてOK）

「〜しましょう」と相手をさそうときは Let's ＋ 動詞の原形。Let's は Let us を短くした形で、「（いっしょに）〜しよう」とよびかける言い方。
例：Let's go.（行こう）／Let's eat lunch.（昼ごはんを食べよう）

Let's のうしろは動詞の原形：かならず動詞のもとの形がくる。Let's play（○）／Let's plays・Let's playing（✕）。主語による -s も ing もつけない。

スポーツに the は不要：play tennis（○）。×play the tennis。楽器のときだけ the（play the piano）。

なぜ？Let's は「（私たちが）〜するのをゆるそう」＝「いっしょにやろう」という意味のかたまり。命令文のなかまなので、うしろの動詞はいつも原形で始める。

つながり：さそいへの返事は Yes, let's.（そうしよう）／ No, let's not.（やめておこう）。「Shall we 〜?」も同じ「さそい」の言い方として後で出てくる。

よくある間違い
✕ Let's plays tennis. → ○ Let's play tennis.（うしろは原形・-s をつけない）
✕ Let's playing tennis. → ○ Let's play tennis.（ing もつけない＝原形）
✕ Lets play tennis. → ○ Let's play tennis.（Let's の '（アポストロフィ）を落とさない）

ゴール🌸：Let's ＋ 原形の形で、相手をさそう文が書ける。

✍ よみとり確認：「〜しましょう」は（ ① ）＋ 動詞の（ ② ）／スポーツに the は（ ③ ）
① Let's ／ ② 原形 ／ ③ つけない
こたえLet's play tennis.

数①・②｜L1→L2 基礎〜標準　項と係数（ことばの確認）
3x − 5 という式について答えよう。
(1) この式の 項 をすべて書きなさい。
(2) x の係数 を答えなさい。
⓪ そもそも「項・係数」って何だっけ？

項（こう）：式を ＋でつないだ1つ分のかたまり。ひき算は「マイナスをたす」と見る。
3x − 5 は 3x ＋ (−5) と同じ → 項は 3x と −5 の2つ。符号（−）も項の一部。

係数（けいすう）：文字の前についている数。3x なら x の係数は 3（x が3個ぶん、という意味）。

① 同類項（どうるいこう）

意味：文字の部分が まったく同じ項どうし。3x と 2x は同類項（どちらも x）。3x と 3y は別もの（文字がちがう）。数だけの項（定数項）どうしも仲間。

なぜ大事：計算でまとめられるのは 同類項だけ。次の問題からは「同じ仲間を集めて、係数だけたし引き」する。

2x ＋ 3x は同類項 → x が2個＋3個＝5個 → 5x。
2x ＋ 3 は仲間がちがう → これ以上まとめられない。
3x − 5 の項を「3x と 5」としない（−5 のマイナスごと1つの項）。係数を「x」と答えない（係数は数の3）。
項とは式を ＋でつないだ1つ分が「項」。3x − 5 は 3x ＋ (−5) と同じだから、項は 3x と −5 の2つ。ひき算は「マイナスをたす」と見ると項が見つけやすい。
係数とは文字の前についている数が「係数」。3x の x の係数は 3。「3が x 何個ぶんか」を表す数。
同類項とは文字の部分が まったく同じ項どうしを「同類項」という。3x と 2x は同類項（どちらも x）。3x と 3y は別もの（文字がちがう）。次の問題でまとめるのはこの同類項だけ。
「項」「係数」「同類項」のことばは、このあとの式の計算・方程式でずっと使う共通語。とくに「同類項だけまとめられる」という考えが、次の同類項をまとめる計算の土台になる。塾でも最初にこのことばを正しく言えるか確認する。
(1)で項を 3x と 5 としない。−5 の マイナスも項の一部（符号ごと1つの項）。(2)で係数を x と答えない（係数は数の 3）。−5 は文字がないので「定数項」とよび、x の係数ではない。
こたえ(1) 項は 3x と −5　(2) x の係数は 3

数③・④・⑤｜L2 標準　同類項をまとめる（係数だけ計算）
次の式を計算して、できるだけかんたんにしよう。
(1) 2x + 3x　(2) 5a − 8a　(3) 4x + 3 − x + 2
考え方（同類項）同類項は 係数だけ計算して、文字はそのまま残す。x が何個ぶんかを数える感じ。
解説(1)2x + 3x は「x が2個 ＋ x が3個」＝x が5個 → 5x。文字 x はそのまま、数だけ 2＋3＝5。
解説(2)5a − 8a は係数を 5−8＝−3 → −3a。引くと負になることもある。文字 a はそのまま。
解説(3)4x + 3 − x + 2。x の仲間（4x と −x）と、数の仲間（3 と 2）を分けて集める。x：4−1＝3 → 3x。数：3＋2＝5。あわせて 3x + 5（これ以上はまとめられない）。−x は −1x のこと。
「同類項は係数だけ計算」は文字式の計算のいちばんの基本。このあとの加法・減法、分配法則、方程式（移項して同類項をまとめる）まで、ずっとこの考えを使う。今ここで「文字の仲間どうし・数の仲間どうしを集める」感覚をつかもう。
(1)を 5x² としない（たし算では指数は増えない。2x＋3x＝5x、かけ算 x×x のときだけ x²）。(3)で 3x と 5 をさらに 8x のようにまとめない（文字 x の項と、数だけの項は 別の仲間なので合体できない）。−x を 0 や x と書かず、係数は −1 と考える。
こたえ(1) 5x　(2) −3a　(3) 3x + 5

数⑥｜L2 標準　一次式の加法（たし算）
次の計算をしよう。
(2x + 3) + (x − 5)
ステップ1（かっこを外す）前が ＋ のかっこは、中の符号をそのまま外してよい。(2x + 3) + (x − 5) → 2x + 3 + x − 5。
ステップ2（同類項を集める）x の仲間：2x ＋ x ＝ 3x。数の仲間：3 − 5 ＝ −2。あわせて 3x − 2。
検算x に 1 を入れて確かめる。もとの式：(2×1+3)+(1−5)＝5+(−4)＝1。答えの式：3×1−2＝1。どちらも1で一致。合っている。
「かっこを外す → 同類項を集める」の2ステップは、このあとのひき算・方程式でもまったく同じ流れ。たし算は符号がそのままで一番やさしいので、ここで手順を体にしみこませると、次のひき算（符号に注意）がぐっと楽になる。
たし算なのに 符号を変えないこと（＋のかっこは中身そのまま）。x の項と数の項を 混ぜてたさない（2x ＋ 3 を 5x としない）。x どうし・数どうしで分けて計算する。

数⑦｜L2→L3 標準〜応用　一次式の減法（ひき算）★最重要
次の計算をしよう。
(5x − 2) − (3x − 7)
ステップ1（かっこを外す・最重要）前が − のかっこは、中の全部の符号を変えて外す。−(3x − 7) → −3x ＋ 7（3x は −3x に、−7 は ＋7 に）。式全体は 5x − 2 − 3x + 7。
ステップ2（同類項を集める）x の仲間：5x − 3x ＝ 2x。数の仲間：−2 ＋ 7 ＝ 5。あわせて 2x + 5。
検算x に 1 を入れて確かめる。もとの式：(5×1−2)−(3×1−7)＝3−(−4)＝3+4＝7。答えの式：2×1+5＝7。どちらも7で一致。合っている。
「− の前のかっこは中の符号を全部変える」は、このあとの方程式・分配法則で何度も出てくる最重要ルール。ここでつまずく中1がとても多いので、5x−2−3x+7 のように いったん全部の符号を直した式に書きなおすクセをつけると、方程式の移項でもミスしなくなる。塾の計算テストでもこの減法は定番。
(5x − 2) − (3x − 7) を 2x − 9 としない。これは −(−7) を −7 のままにした典型ミス。−(−7) ＝ ＋7 なので、数は −2 ＋ 7 ＝ ＋5。「− のかっこは中の 全部の符号を変える」＝3x も −7 も両方ひっくり返す。後ろの項（−7）の符号変えを忘れやすいので、かっこを外した式 5x − 2 − 3x ＋ 7 を必ず一度書いてから計算する。

数⑧｜★ちょいむず（L3）
次の計算をしよう。
2(x + 3) − (4x − 5) + 3x
ステップ1（分配でかっこを外す）前に数がついた 2(x + 3) は、2を中の両方にかける → 2×x ＋ 2×3 ＝ 2x ＋ 6。3 にかけ忘れないこと。
ステップ2（− のかっこを外す）前が − のかっこは 中の全部の符号を変える。−(4x − 5) → −4x ＋ 5（4x は −4x に、−5 は ＋5 に）。式全体は 2x ＋ 6 − 4x ＋ 5 ＋ 3x。
ステップ3（同類項を集める）x の仲間：2x − 4x ＋ 3x ＝ x。数の仲間：6 ＋ 5 ＝ 11。あわせて x + 11。x は係数 1 なので 1x とは書かない。
検算x に 2 を入れて確かめる。もとの式：2(2+3)−(4×2−5)+3×2 ＝ 10 − 3 + 6 ＝ 13。答えの式：2 ＋ 11 ＝ 13。どちらも13で一致。合っている。
これは「分配」「− のかっこ」「同類項まとめ」の3つを1問でぜんぶ使う、入試の基礎レベル（L3）。手順をひとつずつ書き出せば必ず解ける。難しく感じたら、まずは 2x ＋ 6 − 4x ＋ 5 ＋ 3x のように かっこを全部外した式を一度書いてから、x の仲間・数の仲間に分けて集めよう。
2(x+3) を 2x+3 としない（3にも2をかけて 2x+6）。−(4x−5) を −4x−5 としない（−5 が ＋5 に変わる＝後ろの符号変えを忘れやすい）。最後に x の項が 1 個ぶんでも x（1x とは書かない）。

数⑨｜今日のチャレンジ　L2 標準　並べる（順列）
赤・青・黄・緑・白 の 5色 から、ちがう 3色 を選んで 1列に並べます。
並べ方は全部で何通りありますか。
🔁 これは6/23にやった「場合の数（並べる）」のもう一度だよ。忘れたころにやると、本当に身につく。今日はそのチャレンジ1問。
1番目5色のうち どれでもよい → 5通り。
2番目1番目で使った色は使えない → 残り 4通り。
3番目使った2色以外 → 残り 3通り。3色で止める（4番目はかけない）。5 × 4 × 3 ＝ 60 → 60通り。
検算（数えなおし）1番目を「赤」に決めると、残り4色から2色を並べる＝4×3＝12通り。1番目は5色それぞれにこの12通りがあるので 5×12＝60。最初の計算と一致。
「並べる数え方＝1番目×2番目×3番目、使うたびにかける数が1ずつ減る」は中2の確率の土台。何色を並べるかで「かけ算を何回でやめるか」が決まる。樹形図で枝が減っていくのを目で見ておくと、公式に頼らず数えられるよ。
5×5×5＝125（5³）としない。使った色はもう使えないので、かける数は 5→4→3 と 1ずつ減る。また 5×4×3×2＝120 としない（並べるのは3色だけ＝かけ算は 3回でストップ、4番目・5番目はかけない）。「72」なども出やすいが、正しくは 5×4×3＝60。
こたえ60通り

ミスの型は？
きょうのカギは2つ。文字式の計算は「同類項は係数だけ計算」、そしてひき算は「−( )は中の全部の符号を変える」。とくに −(3x−7)＝−3x＋7 がきれいにできたら大きな前進だよ。英語は Let's（Let's play tennis.）。そして前にやった「並べる（順列）」を1問。新しい計算と忘れたころの復習を、30分でちゃんと両どりできた。よくがんばったね。あしたも5分からでOK。
