「私は犬が好きですが、猫は好きではありません。」
✏️ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
📖 文法のせつめい（読んでから下の答えへ。わかる子は読みとばしてOK）

いまの文法＝接続詞 but でつなぐ。2つの文を「でも・けれど」で結ぶときに使うのが but。前と後ろで反対の内容になるのがポイント。

つなぎ語はたらき例
and「そして」＝同じ向きにつなぐI like dogs and cats.
but「でも・けれど」＝反対をつなぐI like dogs, but I don't like cats.

なぜこの形？「犬は好き」＝I like dogs。「猫は好きではない」＝I don't like cats（一般動詞の否定は don't＋動詞）。前と後ろが反対なので、間を but でつなぐ。but の前にはコンマ（,）をつけることが多い。

つながり：and（順）と but（逆）を使い分けられると、2文を1文にまとめて書けるようになる。これから日記や英作文で文を長く書くときの土台。

よくある間違い
✕ I like dogs, but I don't like cats も and にする → ○ 反対の内容は but でつなぐ（「好き」と「好きではない」は反対）
✕ I don't like cats を I not like cats → ○ 一般動詞の否定は don't ＋動詞（文頭は大文字・文末はピリオド）

ゴール🌸：反対の内容の2文を but でつなぎ、否定は don't ＋動詞で書ける。

✍ よみとり確認：「でも・けれど（反対）」は（ ① ）／ 「そして（同じ向き）」は（ ② ）／ 一般動詞の否定は（ ③ ）＋動詞
① but ／ ② and ／ ③ don't
こたえI like dogs, but I don't like cats.

⓪ 正負の数と文字式、ここを思い出そう（夏前総まとめ）

今日は「正負の数」と「文字式」をミックスした総まとめ。まず両方の土台をひとことで確認する。

① 符号のルール（×・÷）

同じ符号どうし＝＋、ちがう符号どうし＝−。とくに マイナス×マイナス＝プラス が最重要。

  • (−3)×(−2)＝＋6（マイナス×マイナス＝プラス）
  • (−3)×(＋2)＝−6（ちがう符号＝マイナス）
② 累乗とかっこ（−4² と (−4)² は別物）

−4²＝−(4×4)＝−16（マイナスは2乗の外）。(−4)²＝(−4)×(−4)＝＋16（かっこごと2乗）。かっこが2乗の中か外かで符号が変わる。

③ 計算の順番

累乗 → ×÷ → ＋−（かっこがあればいちばん先）。この順をくずさない。

④ 文字式の約束

×は書かない・数は文字の前（a×3→3a）。負の数を代入するときはかならずかっこ（x＝−2 → (−2)）。

(−3)²×2 を計算 → まず (−3)²＝＋9 → 9×2＝18（累乗が先）。
−4² と (−4)² を同じだと思わない。−4²＝−16、(−4)²＝＋16。かっこの位置で符号が決まる。
−16 ＋ 18 ＝ 2
ステップ1（−4²を出す）−4² は 4だけを2乗して、最後にマイナス＝−(4×4)＝−16。−4² は「−(4²)」の意味。(−4)² ではないので注意。
ステップ2（(−3)²を出す）(−3)² は かっこごと2乗＝(−3)×(−3)＝＋9。マイナス×マイナス＝プラス。かっこがあるので符号まで2乗される。
ステップ3（かけ算）累乗の次はかけ算 → 9×2＝18。
ステップ4（たし算）さいごにたし算 → −16 ＋ 18。大きいほうが＋18で、差は2 → ＋2。
けんざんもう一度ゆっくり：−4²＝−16、(−3)²×2＝9×2＝18。−16＋18＝＋2。符号は「＋のほうが大きい」のでプラス。答え 2。
「−4²」と「(−4)²」のちがいは、この先の文字式（−x² と (−x)²）・方程式・関数でずっと効いてくる。かっこが2乗の中か外かで符号が変わる、という見方を今ここで固めておくと、ミスがぐっと減る。塾の計算テストでも超定番のひっかけ。
−4² と (−4)² は別物！ −4²＝−(4×4)＝−16（マイナスは2乗の外）。(−4)²＝(−4)×(−4)＝＋16（かっこごと2乗）。今回は −4² なので −16。(−3)² は かっこつきだから ＋9 になる。計算の順番は「累乗 → ×÷ → ＋−」をくずさない。

数②｜L3 応用　文字式の計算（分配法則・同類項）
次の式を計算しよう。
3(2x − 1) − 2(x − 4)
4x ＋ 5
ステップ1（前のかっこを外す）3(2x−1)＝3を中の両方にかける＝3×2x − 3×1＝6x − 3。1つだけにかけない。
ステップ2（うしろのかっこを外す）−2(x−4)＝−2を中の両方にかける＝(−2)×x ＋ (−2)×(−4)＝−2x ＋ 8。うしろの −4 に −2 をかけると ＋8（マイナス×マイナス＝プラス）。
ステップ3（全部ならべる）6x − 3 − 2x ＋ 8。ここまでで符号がそろっていればOK。
ステップ4（同類項をまとめる）文字どうし：6x − 2x＝4x。数どうし：−3 ＋ 8＝＋5。合わせて 4x ＋ 5。
けんざんx＝1 を元の式と答えに入れて確かめる。元の式：3(2−1)−2(1−4)＝3×1 − 2×(−3)＝3 ＋ 6＝9。答え：4×1＋5＝9。両方9で一致 → 正しい。
「分配法則でかっこを外す → 同類項をまとめる」は、このあとの一次方程式を解くときに毎回使う土台。とくに −( ) の前のマイナスを中の全部にかける感覚は、方程式・連立方程式でずっと効く。塾でもここの符号ミスを徹底的に直す。
−2(x−4) の符号がいちばんの注意。前が「−2」なので、中の −4 にかけると ＋8（−2×−4＝＋8）。ここを −8 にするのが最頻ミス。また 3を 2x だけにかけて −1 を忘れない（中の全部にかける）。同類項は「文字は文字、数は数」でまとめる（4x と 5 はちがう種類だから 9x などにしない）。
こたえ4x ＋ 5

数③｜L3 応用　式の値（代入）
x ＝ −2 のとき、次の式の値を求めよう。
2x² − 3x
14
ステップ1（かっこつきで代入）x のところに (−2) を入れる：2×(−2)² − 3×(−2)。負の数はかならずかっこをつけて入れる。
ステップ2（累乗を先に）(−2)²＝(−2)×(−2)＝＋4。だから 2×4＝8（前半）。
ステップ3（うしろの項）−3×(−2)＝＋6（マイナス×マイナス＝プラス）。
ステップ4（合わせる）8 ＋ 6＝14。
けんざんもう一度：(−2)²＝4 → 2×4＝8。−3×(−2)＝＋6。8＋6＝14。引き算（8−6＝2）にしない。答え 14。
「文字に数を代入して値を出す」は、関数（y＝の式に x を入れる）・方程式の確かめでずっと使う。とくに 負の数はかっこをつけて入れるは、符号ミスを防ぐ一生もののクセ。今これを身につけると、この先の代入計算がぐっと安定する。
かっこを必ずつける！ 正しくは (−2)²＝(−2)×(−2)＝＋4。でもかっこを書き忘れて −2² にすると −(2×2)＝−4 になってしまい、答えが大きくズレる。x に負の数を入れるときは かならず (−2) とかっこつきで入れる。また −3x に x＝−2 を入れると −3×(−2)＝＋6（プラスになる）。ここを −6 にするのが最頻ミス。「マイナス×マイナス＝プラス」をかっこで見える化する。

数④｜L3 応用　文章を文字式で表す（割引きの代金）
次の数量を、文字を使った式で表そう。
a円 の品物を 3割引き で買ったときの代金。
7a/10（円）　〔＝0.7a 円〕
ステップ1（割引きを言いかえる）3割引き＝もとの値段から 3割（＝310）をひく。残るのは 10割 − 3割 ＝ 7割。つまり「もとの値段の7割を払う」。
ステップ2（7割を式にする）7割＝710（または0.7）。代金＝もとの値段×710＝a × 710＝7a/10（円）。×を消して、数は文字の前へ。
別の書き方「ひく」で考えてもよい：代金＝a − a×310＝a − 3a/10＝7a/10。同じ答えになる。
けんざんa＝1000円で確かめる。3割引き＝300円引き → 払うのは700円。式に入れると 7×100010＝700010＝700円。ぴったり合う。
「割合のことばを文字式にする」は、中1の文字式・方程式の文章題でくり返し出る山場。「○割引き＝(10−○)割を払う」の言いかえができると、定価・割引き・税こみの問題がいっきに解けるようになる。塾の文章題でも割合×文字式は定番。
「3割引き」を 3a/10 と書かない（それは「ひく金額」であって、払う代金ではない）。払うのは残りの7割だから 7a/10。割合は「割→分数」で、3割＝310、7割＝710。代金は「もとの値段 − 割引き分」＝もとの7割、と最後にことばで確かめる。
こたえ7a/10（円）　〔＝0.7a 円〕

ステップ1（素因数分解する）48を素数のかけ算に分ける：48＝2×24＝2×2×12＝2×2×2×6＝2×2×2×2×3。つまり 48 ＝ 2⁴ × 3。
ステップ2（平方数の条件を思い出す）平方数（ある整数の2乗）になるには、素因数の指数がぜんぶ偶数であればよい（2乗＝同じ数のペアがそろう、ということ）。
ステップ3（足りない素数をさがす）2⁴ の指数4は偶数でOK。でも 3 の指数は1（奇数）でペアが足りない。だから 3を1つかけて 3² にすると、全部の指数が偶数になる。
ステップ4（できる平方数）48×3＝144。たしかに 144 ＝ 2⁴ × 3² ＝ (2² × 3)² ＝ 12²。だから かける数は3、できる平方数は144＝12²。
けんざん本当に48×3＝144？ → 48×3＝144。○。144は何かの2乗？ → 12×12＝144。○。指数で確かめ：2⁴×3²、指数4と2はどちらも偶数 → 平方数で正しい。
「素因数分解して指数を偶数にそろえる」は、平方根（√）や約数・倍数の問題でずっと使う考え方。√48 を簡単にする、最小公倍数を求める、なども全部この素因数分解が土台。塾のテストでも平方数づくりは定番で、ここが速いと整数問題が一気に強くなる。
「指数が奇数の素数」だけを補うのがコツ。48＝2⁴×3 で、2はもう指数4（偶数）だから何もしなくていい。足りないのは3（指数1）だけなので、3を1つかけて 3²。ここで「2もかけなきゃ」と思うと、よけいな数をかけて"できるだけ小さい"にならない。まず素因数分解 → 指数が奇数のものを偶数にする、の順で考える。
