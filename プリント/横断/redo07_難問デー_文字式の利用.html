英語 ③ 英作文1問（いまの文法を確認） 目安 3分
私の父は東京で働いています。
英語で書いてみよう（ノートに）
My father works in Tokyo.
読み：マイ ファーザー ワークス イン トウキョウ
① 日本語を分ける：「私の父は／東京で／働いています」。主語は my father＝三人称単数。「〜で」は場所。
② 使う文法：主語が三人称単数なので work→works。「〜で（都市など広い場所）」は in。
③ 語順（左から組む）：My father ＋ works ＋ in Tokyo.
④ 形のチェック：三単現の s を忘れず works。都市は in Tokyo。
✕ My father work in Tokyo（s がない）→ ○ works。
✕ at Tokyo（都市に at）→ ○ in Tokyo。都市・国など広い場所は in。
⑤ 確かめ算：「三単現の s」「場所の前置詞 in」の2つができていれば正解。
🌸 ゴール：三単現の s を付け、都市など広い場所に in を選べる。

数①｜L4 難問　文章を2段階で文字式にする（割引きの売値）
ある品物の定価は a 円 です。
このお店で、定価の x 割引き で売ることにしました。
(1) x 割引き のとき、売値（実際にはらうねだん）を、a と x を使った式で表しなさい。
(2) 定価が 2000円、3割引き のとき、売値は何円ですか。(1)の式を使って求めなさい。
⓪ 「割引き」って文字式でどう書くの？

やさしく言うと：「1割」は全体を10こに分けた 1こぶん。だから「1割＝110＝0.1」。「x 割」は x10 のこと。「x 割引き」は、定価から x 割ぶんを ひいた残りをはらう、という意味だよ。

① わかっていること／求めること
  • わかっている：定価は a 円。x 割引きで売る。
  • 求める(1)：x 割引きの売値を a と x の式で。
  • 求める(2)：a＝2000、x＝3 のときの売値（円）。
定価1000円を「2割引き」なら？ → ひく分＝1000の2割＝1000×210＝200円。売値＝1000−200＝800円。「定価−ひく分」が売値。
「x 割引き」を「定価の x 割」とまちがえるのが最頻。x 割は「ひく分」。はらうのは 残り。だから (10−x) 割ぶんをはらう、と考えるとはやい。
ステップ1（1割を式に）1割＝110。だから定価 a 円の「x 割」＝a×x10＝ax10（円）。これが ひく分。
ステップ2（売値＝定価−ひく分）売値＝a−ax10（円）。これでも正解。
ステップ3(1)（まとめてきれいに）はらうのは「残りの (10−x) 割」とも言える。残りの割合＝10−x10。だから売値＝a×10−x10＝a(10−x)10（円）。a−ax10 と a(10−x)10 は同じ式（カッコを開くと a−ax10）。
ステップ4(2)（数を入れる）a＝2000、x＝3 を入れる。売値＝2000×(10−3)10＝2000×710＝1400010＝1400円。
確かめべつのやり方でも出す。3割引きのひく分＝2000×310＝600円。売値＝2000−600＝1400円。式の答えと一致。
「x 割引き」を「x 割をはらう」と読まない。x 割は ひく分で、はらうのは残りの (10−x) 割。1割を 1100 や x×10 などにしない（1割＝110、x割＝x10）。(2)で式を使わず暗算だけで答えない（(1)の式に a＝2000、x＝3 を入れて計算する練習がねらい）。
「全体を文字 a で、割合を x10 や (10−x) で表す」は、このあとの 方程式の文章題（割合・売買・速さ） で必ず使う。「文章の中の割引き・増し・○%を文字の式に直す」力は、中1〜中3を通じていちばん問われる場面のひとつ。塾のテストでも割引き・割合の文字式はL4の定番。2つの形（a−ax10 と a(10−x)10）が同じだと気づけたら大きな前進。
こたえ(1) a(10−x)10 円（＝ a−ax10 円）　(2) 1400円

「定価−ひく分」または「残り(10−x)割」の形で、もう3回書いてみよう。

数②｜L4 難問　図のようすを文字式で表す（面積と周の長さ）
下の図のように、たて a cm、よこ b cm の長方形があります。
この長方形の 右下の角 から、1辺が x cm の正方形を切り取りました。
(1) 切り取ったあとの図形（色のついた部分）の 面積 を、a・b・x を使った式で表しなさい。
(2) 切り取ったあとの図形の まわりの長さ（周） を、a・b を使った式で表しなさい。
色のついたL字形が「切り取ったあと」。点線の正方形（1辺 x cm）を切り取った。
⓪ 面積と周は、文字でどう表す？

やさしく言うと：長方形の 面積＝たて×よこ、正方形の 面積＝1辺×1辺。文字でも同じ。たて a・よこ b なら面積は a×b＝ab。1辺 x の正方形は x×x＝x²。周（まわりの長さ）は、外側をぐるっと1周した長さの合計だよ。

① わかっていること／求めること
  • わかっている：もとは たて a・よこ b の長方形。右下から 1辺 x の正方形を切った。
  • 求める(1)：残った色つき部分の面積（a・b・x の式）。
  • 求める(2)：残った図形の周（a・b の式。x は消える）。
たて5・よこ8の長方形から1辺2の正方形を切ると面積は？ → 全体5×8＝40、切った2×2＝4、残り＝40−4＝36。「全体−切った分」でOK。
周も「もとから減る」と思って x をひいてしまうのが最頻。角を切ると、へこんだぶん辺が増えるので 周は変わらない。図でへこみの2辺をずらすと、もとの長方形の周とぴったり同じ長さになるよ。
ステップ1(1)（全体の面積）もとの長方形の面積＝たて×よこ＝a×b＝ab（cm²）。
ステップ2(1)（切った分・残り）切り取った正方形の面積＝x×x＝x²（cm²）。残り＝全体−切った分＝ab−x²（cm²）。これが(1)の答え。
ステップ3(2)（周を辺ごとにたす）L字形の周を上から順に：上 b ＋ 右(上) (a−x) ＋ 切り口・横 x ＋ 切り口・たて x ＋ 下(左) (b−x) ＋ 左 a。全部たすと b＋(a−x)＋x＋x＋(b−x)＋a。
ステップ4(2)（x が消える）x の符号をまとめると −x＋x＋x−x＝0。残るのは a＋a＋b＋b＝2a＋2b（cm）。周はもとの長方形と同じ（x で変わらない）。
確かめa＝8、b＝10、x＝3 で計算。(1)面積＝8×10−3²＝80−9＝71cm²。(2)周＝2×8＋2×10＝16＋20＝36cm。図でへこみ部分の2辺を外へずらすと、たて8・よこ10の長方形の周になるので 36cm で一致。
面積を ab−x としない（正方形の面積は x×x＝x²。x ではない）。周から x をひいて 2a＋2b−2x などとしない（角を切ると辺が増え、へこみの2辺がへった2辺をちょうど埋めるので、周は変わらず 2a＋2b）。(2)に x を残さない（計算すると x は消える。残ったら計算ミス）。
「全体−部分」で面積を文字式にする、辺を1本ずつ文字でたして周を出す——この2つは、図形を数で考える 図形と式 の基本。「角を切っても周は変わらない」は、中2・中3の図形でも出てくる有名な性質。図のようすを ab や x² の文字式に直せると、長方形・正方形の面積の公式が「文字でも同じ」と腑に落ちる。
こたえ(1) ab−x²（cm²）　(2) 2a＋2b（cm）

数③｜★さらに挑戦（できたらでOK）　L4　規則を見つけて n の式にする（碁石ならべ）
碁石（ごいし）を、正方形の 外周（まわり） にすき間なく1列に並べます。
1辺に n 個 ずつ並べるとき、
(1) 使う碁石の 全部の個数 を、n を使った式で表しなさい。
(2) 全部で 96 個 使ったとき、1辺は何個ですか。(1)の式を使って求めなさい。
図は 1辺 n＝5 の例。角の碁石は2つの辺で共通だから「数えすぎ」に注意。
⓪ 「外周だけ・角は1回」で数える

やさしく言うと：正方形には辺が4つ。1辺 n 個だから単純には 4×n 個…と思うけど、4つの角の碁石は、となりあう2辺で1個を2回数えている。だから数えすぎた分（角4個）をひく必要があるよ。

① わかっていること／求めること
  • わかっている：正方形の外周に、1辺 n 個ずつ並べる。
  • 求める(1)：全部の個数を n の式で。
  • 求める(2)：全部で96個のときの n（1辺の個数）。
1辺 n＝5 で数えてみる。4辺ぶん＝4×5＝20。でも角を2回ずつ数えたので 4個ひく → 20−4＝16個。図と一致。
4×n でそのまま答えてしまう（角を引き忘れる）のが最頻。角の4個は重なって2回数えているので、必ず −4 する。
ステップ1(1)（まず4辺ぶん）1辺 n 個が4辺ぶんで n×4＝4n（個）。ただし角を数えすぎている。
ステップ2(1)（数えすぎをひく）正方形の角は4つ。各角の碁石は、となりあう2辺で1回ずつ＝2回数えている。だから余分は4個。全部＝4n−4＝4n−4（個）。これが(1)の答え。
ステップ3(2)（式に96を入れる）(1)より全部の個数は 4n−4。これが96個だから、文章を読みかえると「4n−4＝96 になる n を求める」。4n−4＝96 → 4n＝100 → n＝25（個）。
確かめn＝25 を(1)の式に入れる：4×25−4＝100−4＝96個。たしかに96個で、(2)と一致。
4×n でそのまま答えない（角の4個を2回数えているので −4）。「角を4回ぜんぶ余分」と思って −8 などにしない（角は2回数えたうちの1回ぶん＝1個ずつ余分。4角で −4）。(2)を「96÷4＝24」で答えない（−4を戻し忘れ。正しくは 4n−4＝96 → 4n＝100 → n＝25）。
「図や並びのきまりを見つけて n の式にする」のが 規則性（数の並び・図形の数え方）。高校入試でも超頻出で、中3まで形を変えて出続ける。「単純に数える → 重なりをひく」という考え方は、組み合わせや表の数え方にもつながる強力な道具。1辺 n の例で先に試して式を確かめるクセは、規則性をとくときの王道だよ。
こたえ(1) 4n−4（個）　(2) 25個
