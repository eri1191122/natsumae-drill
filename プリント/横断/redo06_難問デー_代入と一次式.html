数①｜L4 難問　文字に負の数を代入して式の値を求める（累乗・複数項・符号）
x ＝ −3、y ＝ 2 のとき、次の式の値を求めなさい。
(1) x2 − 2xy
(2) −x2 ＋ 3y2 − xy
⓪ 「代入」ってなに？

やさしく言うと：文字（x や y）のところに、決められた数を入れて計算すること。今回は x のところに −3、y のところに 2 を入れる。負の数を入れるときは、まわりに（　）をつけて入れるのがいちばん大事なコツ。

① わかっていること／求めること
  • わかっている：x ＝ −3、y ＝ 2。
  • 求める(1)：x2 − 2xy の値。
  • 求める(2)：−x2 ＋ 3y2 − xy の値。
x ＝ −3 のとき x2 は？ → x2 は「x を2回かける」だから (−3)2＝(−3)×(−3)＝＋9。マイナス×マイナスは＋。かならず（　）をつけて (−3)2 と書く。
x2 に −3 を入れて −32＝−9 としてしまうのが最頻ミス。（　）をつけて (−3)2＝＋9。（　）がないと「3を2乗してマイナス」になってしまう。
ステップ1(1)（（　）つきで代入）x2 − 2xy に x＝−3、y＝2 を（　）つきで入れる。(−3)2 − 2×(−3)×(2)。
ステップ2(1)（累乗を先に）(−3)2＝(−3)×(−3)＝＋9。つぎに 2×(−3)×(2)＝−12。
ステップ3(1)（ひき算）＋9 −(−12)＝9＋12＝21。「− 2xy」のところは −(−12) になり、二重マイナスで ＋12 にひっくり返る。(1)の答え＝21。
ステップ4(2)（（　）つきで代入）−x2 ＋ 3y2 − xy に入れる。−(−3)2 ＋ 3×(2)2 − (−3)×(2)。先頭の −x2 は「x2 にマイナス」だから −(−3)2。
ステップ5(2)（1項ずつ計算）−(−3)2＝−(＋9)＝−9。　3×(2)2＝3×4＝＋12。　−(−3)×(2)＝−(−6)＝＋6。
ステップ6(2)（合計）−9 ＋12 ＋6＝9。(2)の答え＝9。
確かめ(1)：(−3)2＝9、2xy＝2×(−3)×2＝−12、9−(−12)＝21。OK。(2)：−x2＝−9、3y2＝12、xy＝−6 で −xy＝＋6、−9＋12＋6＝9。OK。
(−3)2 を −9 としない（（　）つきなら ＋9）。−x2 に代入するとき −(−3)2 の「先頭のマイナス」を忘れない（答えは −9）。−2xy が −(−12)＝＋12 になるのを −12 のままにしない（ひく数がマイナスだから ＋にひっくり返る）。代入はとにかく 負の数は（　）でくるむのが事故を防ぐ一番のコツ。
「文字に値を代入して式の値を出す」は、このあとの 関数（y＝ax＋b に x を入れて y を出す）や 図形の面積・体積の公式に数を入れる場面で毎回使う。とくに「負の数は（　）つきで、累乗を先に」は中学・高校でずっと効く土台。塾のテストでもこの複合代入はL4の定番。

数②｜L4 難問　一次式の加減・分配（分数係数・−（　）の符号・3項以上）
次の式を計算して、できるだけかんたんにしなさい。
(1) 2x − 12 − x − 33
(2) 2(3x − 1) − (x − 4) − 5x
⓪ 一次式の計算でいちばん事故るところ

やさしく言うと：一次式の計算は「分数は通分」「−（　）はカッコの中ぜんぶ符号がかわる」「同じ文字どうし・数どうしだけまとめる」の3つ。とくに −（　）の符号ミスが中1でいちばん多い事故。

① わかっていること／求めること
  • (1)：分母が 2 と 3 の分数のひき算 → まず通分（分母を6にそろえる）。
  • (2)：分配（カッコを開く）→ −（　）の符号に注意 → 同類項をまとめる。
x2 − x3 は？ → 通分して 3x6 − 2x6＝3x − 2x6＝x6。分子は（　）でまとめて引く。
−（x − 3）を −x − 3 にしてしまうのが最頻ミス。−（x − 3）は −x ＋3（中の −3 も ＋3 に変わる）。引く数が分数のときは、分子全体に（　）をつけて引く。
ステップ1(1)（通分）分母2と3の最小公倍数は6。2x − 12＝3(2x − 1)6、　x − 33＝2(x − 3)6。分母を6にそろえる。
ステップ2(1)（分子を1つにまとめる）3(2x − 1) − 2(x − 3)6。引く側の分子 2(x − 3) には（　）をつけたまま引く。
ステップ3(1)（分子を展開）分子＝3(2x − 1) − 2(x − 3)＝(6x − 3) − (2x − 6)＝6x − 3 − 2x ＋6。−2(x − 3) の −6 が ＋6 に変わるのが山。
ステップ4(1)（同類項）分子＝(6x − 2x) ＋(−3 ＋6)＝4x ＋3。よって 4x ＋ 36。(1)の答え＝4x ＋ 36（これ以上約分できない）。
ステップ5(2)（分配する）2(3x − 1)＝6x − 2。　−(x − 4)＝−x ＋4（−（　）で中ぜんぶ符号がかわる）。よって 6x − 2 − x ＋4 − 5x。
ステップ6(2)（同類項をまとめる）文字：6x − x − 5x＝0x（x が消える）。数：−2 ＋4＝＋2。よって 2。(2)の答え＝2（x が消えて数だけ残る）。
確かめ(1)：x＝3 を元の式に入れる→52 − 03＝2.5。答え 4×3＋36＝156＝2.5。一致。(2)：x＝10 を元の式に入れる→2×29 − 6 − 50＝58 − 56＝2。答え2と一致。
−(x − 4) を −x − 4 としない（正しくは −x ＋4。−（　）は中ぜんぶ符号反転）。(1)で引く側の分子に（　）をつけ忘れて −2x − 6 のまま引かない（−(2x − 6)＝−2x ＋6）。分数の答えを 4x ＋ 36 を 2x ＋ 33 のように勝手に約分しない（4x と3の片方だけ割るのは禁止。分子の全部を同じ数で割れる時だけ約分できる）。(2)で x が消えるのを「まちがい」と思わない（一次式の計算では文字が消えて数だけ残ることがある）。
「分数係数の一次式を通分してまとめる」「−（　）の符号をていねいに開く」は、このあとの 方程式（とくに分数をふくむ方程式）でそのまま使う最重要スキル。ここで分母をそろえる・符号を反転する手つきが正確になると、方程式の文章題でミスがぐっと減る。塾の計算問題でもこの分数係数＋符号の複合はL4の定番。
こたえ(1) 4x ＋ 36　(2) 2

数③｜★さらに挑戦（できたらでOK）　L4　文字式を使って「いつでも成り立つこと」を説明する
となりあう 連続する3つの整数（たとえば 4, 5, 6 のように1ずつ増える3つ）について、
その 3つの数の和は、いつでも3の倍数になる ことを、文字式を使って説明しなさい。
⓪ 「いつでも成り立つ」を文字で示すってどういうこと？

やさしく言うと：「4,5,6 でも 10,11,12 でも…」と全部ためすのは無理。だから どんな整数でも表せる文字（n）を使って、たった1回の式で「ぜんぶの場合」をまとめて説明する。これが文字式のいちばんかっこいい使い方。

① 連続する3整数を文字でどう書く？
  • まんなかの数を n とすると、その前は n − 1、その後は n ＋1。
  • つまり連続する3整数は n − 1、n、n ＋1 と書ける。
和を計算すると？ → (n − 1) ＋ n ＋ (n ＋1)。−1 と ＋1 が消えて 3n。3n は「3×（整数）」だから3の倍数。
3, 4, 5 のような具体例を1つ出して「ほら3の倍数」で終わらせるのは「説明」になっていない（それは1例だけ）。文字 n を使って「どんな場合でも」を示すのが説明。
ステップ1（文字でおく）連続する3つの整数のうち、まんなかの整数を n とおく。すると3つは n − 1、n、n ＋1 と表せる（n は整数）。
ステップ2（和の式を立てる）3つの数の和＝(n − 1) ＋ n ＋ (n ＋ 1)。
ステップ3（計算する）(n − 1) ＋ n ＋ (n ＋ 1)＝n ＋ n ＋ n ＋(−1 ＋1)＝3n。−1 と ＋1 が打ち消し合う。
ステップ4（結論を書く）n は整数だから 3n は 3 ×（整数）。よって連続する3つの整数の和はいつでも3の倍数になる。（説明おわり）
確かめ4,5,6 → 和15＝3×5。10,11,12 → 和33＝3×11。−2,−1,0 → 和 −3＝3×(−1)。どれも 3n の形（まんなか×3）になっていて、3の倍数。
具体例だけで終わらせない（1例では「いつでも」を示せない。文字 n が必要）。最後に「n は整数だから 3n は3の倍数」と一言そえるのを忘れない（3n が3の倍数だと言い切るには「n が整数」が必要）。3つを n、n＋1、n＋2 とおいてもよい（和＝3n＋3＝3(n＋1) でやはり3の倍数。どちらのおき方でも正解）。
「文字でおいて → 式を立てて → 計算して → 結論を言葉で書く」という4ステップは、中2・中3でくり返し出る 「文字式による説明（証明の入り口）」そのもの。偶数・奇数、連続する数、2けたの数の入れかえ…と発展していく。中1のいまこの形に慣れておくと、高校入試の証明問題で大きな武器になる。
こたえまんなかを n とすると3つは n−1, n, n＋1。和＝(n−1)＋n＋(n＋1)＝3n。n は整数だから 3n は3の倍数。よって連続する3整数の和はいつでも3の倍数。
