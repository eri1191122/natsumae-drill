英語きそ U9形容詞・副詞30分

形容詞（ようす）と 副詞（どんなふうに）

形容詞は「ようす」を表す言葉（tall・new・kind）。置き場は2つ＝be動詞の後ろ、または名詞の前。副詞は「どんなふうに」（fast・well）。

特に 頻度副詞（always/usually…）の置き場 が本番でねらわれる。使う単語はかんたんに。

① 核心ルール形容詞には置き場が2つ6分

形容詞（ようすの言葉）には、2つの置き場がある。

be動詞の後ろ：He is kind.（彼は親切だ）　／　名詞の前：a kind boy（親切な少年）
どちらも「ようす」を付けたいだけ。be動詞のあとに置けば「主語＝ようす」、名詞の前に置けば「その名詞のようす」になる。意味はほぼ同じで、置き場がちがうだけ。
② 形容詞の2用法「〜だ」と「〜な○○」6分
置き場形れい
be動詞の後ろ主語＋be＋形容詞This bag is new.
名詞の前a/an＋形容詞＋名詞This is a new bag.
a new bags ✕（形容詞は名詞の前に1回・bag が1つなら s なし）／語順は a boy tall ✕ → a tall boy ○（形容詞は名詞の前）。
③ 副詞と頻度副詞動作のようす・どれくらいの回数6分

副詞は「どんなふうに」動くか（動作のようす）。ふつう文の後ろ。

形容詞副詞れい
good（良い）well（じょうずに）She plays the piano well.
fast（速い）fast（速く・同じ形）He runs fast.
early（早い）early（早く・同じ形）I get up early.

頻度副詞（いつも・たいてい…）の置き場は特別。一般動詞の前／be動詞の後ろ。

頻度副詞意味置き場
alwaysいつも一般動詞の前
be動詞の後ろ
usually / oftenたいてい／よく
sometimes / neverときどき／一度もない
I get up always early ✕ → I always get up early ○（一般動詞 get up の前）。He always is busy ✕ → He is always busy ○（be動詞 is の後ろ）。
④ very で強めるとても・〜すぎる5分
言葉意味れい
veryとてもThis book is very interesting.
soとても（気持ちこめて）I am so happy.
too〜すぎるIt is too hot.
very は形容詞・副詞の前に置く（very kind / very fast）。特訓の大問1では「ようすの言葉（形容詞）の適語」が、大問4の並べ替えでは「a＋形容詞＋名詞」の語順がねらわれる。
⑤ 書いて定着日本語を見て、自分で書こう7分
問題1｜be＋形容詞
彼は親切です。
✏️ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
主語 He ＋ be動詞 is ＋ 形容詞 kind。形容詞は be のうしろ。
こたえHe is kind.
書けた？
📝 3回　He is kind.
問題2｜名詞の前
彼は背の高い少年です。
✏️ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
a ＋ 形容詞 tall ＋ 名詞 boy。形容詞は名詞の前。
こたえHe is a tall boy.
書けた？
📝 3回　He is a tall boy.
問題3｜名詞の前
これは新しい自転車です。
✏️ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
This is ＋ a ＋ new ＋ bike。形容詞 new は名詞 bike の前。
こたえThis is a new bike.
書けた？
📝 3回　This is a new bike.
問題4｜副詞
彼は速く走ります。
✏️ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
He runs ＋ fast。3単現なので run に s。副詞 fast は後ろ。
こたえHe runs fast.
書けた？
📝 3回　He runs fast.
問題5｜副詞 well
彼女はじょうずにピアノをひきます。
✏️ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
She plays the piano ＋ well。「じょうずに」は副詞 well（good ではない）。
こたえShe plays the piano well.
書けた？
📝 3回　She plays the piano well.
問題6｜頻度副詞（一般動詞の前）
私はいつも早く起きます。
✏️ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
I ＋ always ＋ get up ＋ early。always は一般動詞 get up の前。
こたえI always get up early.
書けた？
📝 3回　I always get up early.
問題7｜頻度副詞（be動詞の後ろ）
彼はいつも忙しい。
✏️ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
He ＋ is ＋ always ＋ busy。always は be動詞 is の後ろ。
こたえHe is always busy.
書けた？
📝 3回　He is always busy.
問題8｜very で強める
この本はとてもおもしろい。
✏️ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
This book is ＋ very ＋ interesting。very は形容詞の前。
こたえThis book is very interesting.
書けた？
📝 3回　This book is very interesting.
「形容詞は be のうしろ／名詞の前」「頻度副詞は 一般動詞の前・be の後ろ」。つぎは U10 前置詞へ。
