特訓P4e数学：方程式の利用★文章→式 L3-L4（過不足・二段）

文章を式にする⑤｜「最初の□脚は○人ずつ・残りは△人ずつ」二段の過不足を型で固める

夏・特訓対策　目安：約30分（問1〜問4・通し番号）　対象: 中1　全問に徹底解説つき

今日は「最初の何脚かだけ人数が違う」二段タイプ。これは9月の特訓選抜 大問3(3)とそっくり同じ形だよ。コツは1つだけ＝「前の区間」と「後ろの区間」を分けて、人数を足し算で表すこと。型カードにそって、4問だけじっくり。式が作れたら、もうこの型は体に入ったということ🌸

  1. ① 何を求める？ → それを x とおく（「□を x とする」と書く）
  2. ② 図か表にする（線分図・表）＝目で見える形に
  3. ③ 等しい関係を式に（＝で結べる2つの言い方を探す）
  4. ④ 解く（移項して x ＝ の形へ）
  5. ⑤ 検算（出た数を文章に戻して合うか確かめる）
★過不足・二段のコツ：「最初の□脚は○人ずつ・残りは△人ずつ」は、人数を〔○×□＋△×(脚数−□)〕のように区間ごとに足し算で表す。前の区間と後ろの区間を分けて数え、最後に足すのがポイント。
問1★本番同型 二段の過不足（ぴったり埋まる）L4 応用目安 8分
問1生徒が長椅子に座ります。最初の9脚に7人ずつ、残りの長椅子に5人ずつ座ったら、生徒は全部で83人で、長椅子はちょうど全部うまりました。長椅子は何脚ありますか。
長椅子の数を x にする。前の9脚（7人ずつ）と、残りの脚（5人ずつ）に分けて、座った人数を足し算で表そう。残りの脚数は「全部−9＝(x−9)」。
📐 図：前ブロック（9脚×7人）／後ブロック（残り(x−9)脚×5人）に区切る
二段の過不足は「前の区間」と「後ろの区間」を分けて人数を足す。前9脚は7人ずつ＝9×7＝63人、残りは(x−9)脚に5人ずつ＝5(x−9)人。合計が83人。
①x長椅子の数を x 脚とする（求めるもの）。残りの脚数は (x−9) 脚。
③式前9脚＝9×7＝63人、残り＝5(x−9)人。合計83人だから 63＋5(x−9)＝83。
④解63＋5x−45＝83 → 5x＋18＝83 → 5x＝65 → x＝13。
⑤検算前9脚×7＝63人、残り(13−9)＝4脚×5＝20人。63＋20＝83人○。長椅子は13脚でちょうど埋まる。
残りの脚数は (x−9)。全部の x 脚に5人ずつ座ると考えて 5x としない。前の9脚はすでに7人ずつで埋まっているから、5人ずつなのは「残り(x−9)脚」だけ。
これが9月の特訓選抜 大問3(3)そのもの。「最初の□脚だけ人数が違う」二段の長椅子問題は出題定番。区間で分けて足す型をそのまま使える。

問2二段の過不足（テント・ぴったり）L3-L4目安 7分
問2生徒がテントに泊まります。最初の4張りに6人ずつ、残りのテントに4人ずつ泊まったら、生徒は52人で、テントはちょうど全部使いました。テントは何張りありますか。
問1と同じ二段の型。テントの数を x にして、前4張り（6人ずつ）と残り(x−4)張り（4人ずつ）に分けて足そう。
📐 図：前ブロック（4張り×6人）／後ブロック（残り(x−4)張り×4人）に区切る
前4張りは6人ずつ＝4×6＝24人、残りは(x−4)張りに4人ずつ＝4(x−4)人。合計52人で式を作る。
①xテントの数を x 張りとする。残りは (x−4) 張り。
③式前4張り＝4×6＝24人、残り＝4(x−4)人。合計52人だから 24＋4(x−4)＝52。
④解24＋4x−16＝52 → 4x＋8＝52 → 4x＝44 → x＝11。
⑤検算前4張り×6＝24人、残り(11−4)＝7張り×4＝28人。24＋28＝52人○。
残りの張り数は (x−4)。前の4張りはもう6人ずつで埋まっているから、4人ずつなのは残り(x−4)張りだけ。全部に4人ずつとして 4x としない。

問3★二段の過不足（長椅子・ぴったり）L4目安 8分
問3長椅子に、最初の5脚に8人ずつ、残りの長椅子に6人ずつ座ると、ちょうど全員座れました。生徒は82人です。長椅子は何脚ありますか。
同じ二段の型。長椅子の数を x にして、前5脚（8人ずつ）と残り(x−5)脚（6人ずつ）に分けて足そう。
📐 図：前ブロック（5脚×8人）／後ブロック（残り(x−5)脚×6人）に区切る
前5脚は8人ずつ＝5×8＝40人、残りは(x−5)脚に6人ずつ＝6(x−5)人。合計82人で式を作る。
①x長椅子の数を x 脚とする。残りは (x−5) 脚。
③式前5脚＝5×8＝40人、残り＝6(x−5)人。合計82人だから 40＋6(x−5)＝82。
④解40＋6x−30＝82 → 6x＋10＝82 → 6x＝72 → x＝12。
⑤検算前5脚×8＝40人、残り(12−5)＝7脚×6＝42人。40＋42＝82人○。
6(x−5) の展開に注意＝6x−30。x にも5にも6をかける。6x−5 としない。残りの脚数も (x−5) で、6x としない。
前の人数（8人）の方が後ろ（6人）より多い形でも、考え方は同じ。「前の区間＋後ろの区間＝全部」を崩さないことが特訓選抜の得点源。

問4★最難 二段＋あまり（座れない人がいる）L4目安 9分
問4長椅子に、最初の6脚に9人ずつ、残りの長椅子に7人ずつ座らせたら、4人が座れずにあまりました。長椅子は10脚あります。生徒は何人ですか。
今度は長椅子の数（10脚）が分かっていて、生徒の人数を求める。前6脚（9人ずつ）＋残り(10−6)脚（7人ずつ）＝座れた人数。座れない4人は人が多いから足す。
📐 図：前ブロック（6脚×9人）／後ブロック（残り4脚×7人）／座れないあまり4人
二段＋あまり＝区間ごとの「着席人数」を足して、座れない分を＋する。前6脚＝6×9＝54人、後ろ＝(10−6)＝4脚×7＝28人、座れない4人を足す。生徒＝54＋28＋4。
①x求めるのは生徒の人数。長椅子の数は10脚と分かっているので、人数を順に足していく。
③式前6脚＝9×6＝54人、残り(10−6)＝4脚＝7×4＝28人、座れず4人あまる。生徒＝9×6＋7×(10−6)＋4。
④解9×6＋7×(10−6)＋4＝54＋28＋4＝86。
⑤検算前6脚に54人、後4脚に28人で着席は82人。これに座れない4人を足して82＋4＝86人○。
座れない4人を引かない。座れない＝椅子より人が多いということ＝＋。着席82人「だけ」を答えにせず、あまった4人を足す。
「ぴったり」「たりない」「あまる（座れない）」の3パターンを区間ごとの足し算でそろえると、特訓選抜のどの形が来ても同じ手順で解ける。
