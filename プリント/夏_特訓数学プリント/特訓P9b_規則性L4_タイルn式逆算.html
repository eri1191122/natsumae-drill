特訓P9b数学：規則性★タイル→n式→逆算 L3-L4（9月大問4直撃）

規則性②｜タイルの数え方を「nの式」にして、大きい番目を逆算する

夏・特訓対策　目安：約30分（問1〜問4・通し番号）　対象: 中1　全問に徹底解説つき

9月の特訓選抜・大問4はこの流れがそのまま出る：①図を数える → ②nの式にする → ③大きい番目を逆算する。きみのいちばんの伸びしろ＝規則性。今日は1テーマ「タイル」を①②③で最後まで通して、最後に三角数も1問。式さえ作れれば1000番目でも一瞬だよ。

  1. ① 1・2・3番目を表に書く（図を数えて具体を並べる）
  2. ② 差を見る（毎回いくつ増える？＝この数が n の係数）
  3. ③ n式を作る：差×n＋（1番目−差）。図形は「変わる部分×n＋変わらない部分」
  4. ④ 逆算：「○個になるのは何番目？」は n式＝○ の方程式を解く
  5. ⑤ 確かめ：n＝1,2 や出た番目を式に入れて表と合うか
問1①具体化：タイルを数えて差を見つける L3目安 7分
問1白いタイルを正方形に並べ、そのまわりを黒いタイルで1周ぐるりと囲みます。下の図は1番目・2番目・3番目です。黒いタイルの数を数えて、1番目・2番目・3番目を表にうめなさい。また、黒タイルは1番進むごとに何個ずつ増えますか。
■＝黒タイル　□＝白タイル（白は n×n の正方形、そのまわりを黒が1周）
図の黒タイルを1個ずつ数えて表をうめよう。次に「2番目−1番目」「3番目−2番目」を計算して、増え方（差）を見つける。
番目123…
黒タイル(個)81216…
①数える図を数えると 1番目＝8個、2番目＝12個、3番目＝16個。
②差を見る12−8＝4、16−12＝4。1番進むごとに4個ずつ増える（差は4）。
差は「白の正方形が1辺ずつ大きくなると、囲む黒タイルが上下左右の4方向に1個ずつ＝4個増える」から。図と数がぴったり合うのが大事。
「黒タイルが何個か」と「何番目か」をごちゃ混ぜにしない。表の上の行が「番目」、下の行が「黒タイルの個数」。役割をはっきり分けて書こう。

問2②一般化：n番目をnの式にする L3-L4目安 8分
問2問1と同じタイルの並びで、n番目の黒タイルの数を n の式で表しなさい。
型カードの③を使う。差×n＋（1番目−差）。問1で差＝4、1番目＝8だから…。式ができたら n＝1, 2, 3 を入れて表（8, 12, 16）に合うか確かめよう。
等差の並びの式：n番目＝（差）×n＋（1番目−差）。差は「毎回いくつ増えるか」。最後に n＝1 で必ず確かめる。
③式を作る差は4、1番目は8。式＝4×n＋(8−4)＝4n＋4。
⑤確かめn＝1：4×1＋4＝8○、n＝2：4×2＋4＝12○、n＝3：4×3＋4＝16○。表とぴったり合う。
差だけ見て「4n」で終わらせない。それだと n＝1 で4個になり、図の8個と合わない。1番目に合わせる調整（＋4）が必ず要る。式を作ったら n＝1 で検算するクセを。
「具体（数えた表）→ 差 → 4n＋4 という n の式」へ翻訳するこの流れが、9月大問4(2)でそのまま問われる。式づくりが規則性の心臓。
こたえn番目＝4n＋4（個）

問3③逆算：大きい番目を方程式で出す L4目安 8分
問3問2の式（n番目＝4n＋4）を使って答えなさい。(1) 30番目の黒タイルは何個ですか。(2) 黒タイルがちょうど100個になるのは何番目ですか。
(1)は n に数を「入れる（代入）」。(2)は逆に「式＝100」という n の方程式を解く。型カードの④。ノートに式を書いてから解こう。
n式ができれば 「番目→個数」は代入、「個数→何番目」は方程式（逆算）。これが規則性の最後のゴール。
(1) 代入n＝30 を 4n＋4 へ：4×30＋4＝120＋4＝124。124個。
(2) 逆算（方程式）黒タイル＝100 だから 4n＋4＝100。両辺−4 → 4n＝96 → 両辺÷4 → n＝24。24番目。
⑤確かめn＝24 を式へ：4×24＋4＝96＋4＝100○。ちゃんと100個になる。
100÷4＝25 と即答しない。それは「＋4」を忘れた答え。逆算は先に＋4を移項して引いてから割る（4n＝96 → n＝24）。式を立てて順番に解くのが安全。
この(2)が9月大問4(3)の「○個になるのは何番目（n＝16型）」とまったく同じ。代入と逆算をセットで出せれば大問4は取れる。
こたえ(1) 124個／(2) 24番目

問4★本番型：黒タイル→白タイル（全体−黒）L4目安 8分
問41辺が n cm の正方形を、1cm四方の白タイルでしきつめ、そのまわりの3つの辺（上・左・右）だけを黒タイルに取りかえます。下の表のように、黒タイルは n が1増えるごとに3枚ずつ増えます。
n234…
黒タイル4710…
(1) 黒タイルの枚数を n の式で表しなさい。
(2) 1辺が16cmのとき、白タイルは何枚ありますか。
本番は「黒を出して終わり」ではない。全体は n² 枚。白タイルは「全体−黒」で出す。n=16 を最後に入れよう。
(1) 黒表を見ると、n=2で4枚、n=3で7枚、n=4で10枚。差は3。式は 3n＋調整。n=2で4になるように、3×2−2＝4。だから黒タイルは 3n−2 枚。
確かめn=3なら3×3−2＝7、n=4なら3×4−2＝10。表と合う○。
(2) 全体1辺16cmの正方形なので、タイル全体は 16²＝256 枚。
(2) 黒黒タイルは 3×16−2＝48−2＝46 枚。
(2) 白白タイル＝全体−黒＝256−46＝210 枚。
白タイルを3n−2で答えない。3n−2は黒。聞かれている白は、全体 n² から黒を引く。n=16の全体を16×4などにしない。正方形全体は16×16。
9月特訓選抜 大問4の本番型。黒をn式化→全体n²→白=全体−黒の三段処理ができれば、規則性の後半を拾える。
こたえ(1) 3n−2 枚／(2) 白タイル 210枚
