問51個80円のみかんと1個120円のりんごを合わせて16個買ったら、代金は1600円でした。みかんは何個買いましたか。
問6ある道のりを、行きは分速60m、帰りは分速40mで往復したら、合計25分かかりました。片道の道のりは何mですか。
問7あめを子どもに配ります。1人6個ずつ配ると13個あまり、1人8個ずつ配ると7個たりません。子どもは何人いますか。
問8ある品物に、原価の3割の利益を見込んで定価をつけたら2730円になりました。原価はいくらですか。
問5は「みかん x 個・りんご(16−x)個」、問6は「片道 x、x/60＋x/40＝25」、問7は「あめの総数を2通りで表す」、問8は「定価＝原価×1.3」。
問5みかん x 個、りんご(16−x)個。80x＋120(16−x)＝1600 → 80x＋1920−120x＝1600 → −40x＝−320 → x＝8。みかん8個。検算 80×8＋120×8＝640＋960＝1600○。
問6片道を x とする。x60＋x40＝25。両辺×120：2x＋3x＝3000 → 5x＝3000 → x＝600。片道600m。検算 行き10分＋帰り15分＝25分○。
問7子ども x 人。あめの総数を2通りで表す：6x＋13＝8x−7。13＋7＝8x−6x → 20＝2x → x＝10。10人。検算 6×10＋13＝73、8×10−7＝73○。
問8原価を x。定価＝x×1.3＝2730 → x＝2730÷1.3＝2100。原価2100円。検算 2100×1.3＝2730○。
問5 最後に「みかん」を答える（りんごと取りちがえない）。問6 往復の式は時間＝道のり÷速さ。問7 たりない側は 8x−7（7個ひく）。問8 2730−3割＝1911 としない（割るのは1.3）。

問9(−24)÷(−3)−(−5)×2 問10−3²＋(−4)²−(−2)³

大問4平面図形（L字面積・折り返し角）L3各10点・計20点
問11下の図のL字形（たて12cm・横10cmの長方形から、右上のかど 横3cm・たて5cm を切りとった形）の面積を求めなさい。 問12長方形の紙テープを下の図のように1回折り返したとき、折り目と辺がつくる角が58°でした。重なってできる角 x を求めなさい。
問11は「大きい長方形−切りとった小長方形」。問12は「折ると左右が等しい＋三角形の内角和」。
問11大きい長方形＝12×10＝120。切りとった小長方形＝3×5＝15（横3cm・たて5cm）。L字＝120−15＝105 cm²。
問12折り返しで両側の角が等しく58°。三角形の内角和より x＝180−58−58＝64°。
問11 切りとるのは「横3cm・たて5cm」の小さい長方形。たてと横を取りちがえない。問12 x＝58としない（折り目の左右は等しいが、xは残りの角）。
こたえ問11 105 cm²／問12 64°
