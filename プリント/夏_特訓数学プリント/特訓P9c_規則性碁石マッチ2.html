規則性③｜マッチ棒と碁石の数を「nの式」にして、大きい番目を逆算する

夏・特訓対策　目安：約30分（問1〜問4・通し番号）　対象: 中1　全問に「あほでもわかる7層解説」つき

9月の特訓選抜・大問4はこの流れがそのまま出る：①図を数える → ②nの式にする → ③大きい番目を逆算する。規則性はきみのいちばんの伸びしろ。今日はマッチ棒と碁石で①②③を最後まで通すよ。式さえ作れれば100本でも1000番目でも一瞬。あせらなくていい、1層ずつ読めば必ず分かる🌸

  1. ① 1・2・3番目を表に書く（図を数えて具体を並べる）
  2. ② 差を見る（毎回いくつ増える？＝この数が n の係数）
  3. ③ n式を作る：差×n＋（1番目−差）。図形は「ふえる部分×n＋ふえない部分」
  4. ④ 逆算：「○本（○個）になるのは何番目？」は n式＝○ の方程式を解く
  5. ⑤ 確かめ：n＝1,2,3 や出た番目を式に入れて表と合うか
問1①具体化：マッチ棒を数えて差を見つける L3目安 7分
問1マッチ棒で正方形を横に1列に並べます。下の図は1番目・2番目・3番目です。マッチ棒の本数を数えて、1番目・2番目・3番目を表にうめなさい。また、1番進むごとに何本ずつ増えますか。
━ ＝マッチ棒1本。正方形を右へ1個ずつつなげて並べる（となりと1辺を共有する）
図のマッチ棒を1本ずつ数えて表をうめよう。次に「2番目−1番目」「3番目−2番目」を計算して、増え方（差）を見つける。
① 何を聞かれてる？（やさしい言いかえ）
「マッチ棒の正方形を1個・2個・3個ならべたとき、ぼうは何本ずつ？　そして次に進むと何本ふえる？」を聞かれているだけ。むずかしく考えない。
② わかってること／求めるもの
  • わかってること：図の1〜3番目（正方形が1個・2個・3個）
  • 求めるもの：①各番目の本数　②1番進むごとの「差（ふえる本数）」
③ 図でイメージ
最初の正方形は、4本そろえないと作れない（4本）。次からは、となりに新しい正方形をくっつけるけど、左の1辺はもう前の正方形のぼうを使いまわす。だから新しく足すのは 上・下・右の3本だけ。図の「ふえた所」だけ見ると、いつも3本ずつ増えている。
④ 作戦＋たとえ話
これは「階段を1段ずつ上る」のと同じ。1段目だけは特別（4本）、そのあとは毎回おなじ段差（＋3本）。だから「最初の1個ぶん」と「あとは何回足すか」を分けて考えるのがコツ。今日はまず表に書いて、差を目で見つける。
⑤ 1ステップずつ（なぜ付き）
番目（n）123…
マッチ棒(本)4710…
数える1番目＝4本、2番目＝7本、3番目＝10本。（なぜ：正方形1個＝4本、2個目で＋3、3個目で＋3）
差を見る7−4＝3、10−7＝3。1番進むごとに3本ずつ増える（差は3）。（なぜ：足すのは上下右の3本だけだから）
⑥ よくある間違い
「正方形が何個か」と「ぼうが何本か」をごちゃ混ぜにしない。表の上の行が「番目（正方形の個数）」、下の行が「マッチ棒の本数」。差を「4」と書かない＝最初だけ4本だが、ふえる差は3本。最初の1本と、増える差はべつもの。
⑦ 検算＋次に見抜くコツ
表の差を全部確かめる：7−4＝3、10−7＝3でそろっている○。
規則性のコツは ①差を見る ②1番目に合わせる ③確かめる。まず表を作って差を出す、これが全部のスタート。
こたえ1番目 4本／2番目 7本／3番目 10本　差は3（3本ずつ増える）

問2②一般化：n番目をnの式にする L3-L4目安 8分
問2問1と同じマッチ棒の正方形の並びで、n番目のマッチ棒の本数を n の式で表しなさい。
型カードの③を使う。差×n＋（1番目−差）。問1で差＝3、1番目＝4だから…。式ができたら n＝1, 2, 3 を入れて表（4, 7, 10）に合うか確かめよう。
① 何を聞かれてる？（やさしい言いかえ）
「100番目だろうが1000番目だろうが、本数が一発で出る『計算式』を作って」ということ。番目を n と書いて、その式を作る。
② わかってること／求めるもの
  • わかってること：差は3、1番目は4（問1より）
  • 求めるもの：n番目の本数を表す式
③ 図でイメージ
図でいうと「最初の1本ぶん（左はしの4本）」はずっと変わらない。変わるのは『3本セット』を何回くっつけるか。n番目なら、3本セットを n 回足したいけど、最初の正方形はもう4本そろっている…ここをどう数えるかが式作りのカギ。
④ 作戦＋たとえ話
「最初に4本もらってスタート、そのあと1個進むごとに3本ずつもらう」貯金みたいなもの。1番目＝4、2番目＝4＋3、3番目＝4＋3＋3…。これを式にすると「4＋3×(進んだ回数)」。型カードでは 差×n＋（1番目−差） で一発に出せる。
⑤ 1ステップずつ（なぜ付き）
等差の並びの式：n番目＝（差）×n＋（1番目−差）。差は「毎回いくつ増えるか」。最後に n＝1 で必ず確かめる。
式を作る差は3、1番目は4。式＝3×n＋(4−3)＝3n＋1。（なぜ：3n だけだと n＝1 で3になり、図の4本と1本ずれる。そのずれ＋1をたす）
代入と方程式のちがいここで作るのは「式そのもの」。代入＝この式の n に数を入れて本数を出すこと（問3で使う）。方程式＝この式＝○ とおいて n を逆に求めること（問3(2)で使う）。今は式を作るだけ。
確かめn＝1：3×1＋1＝4○、n＝2：3×2＋1＝7○、n＝3：3×3＋1＝10○。表とぴったり合う。
⑥ よくある間違い
差だけ見て「3n」で終わらせない。それだと n＝1 で3本になり、図の4本と合わない。1番目に合わせる調整（＋1）が必ず要る。式を作ったら n＝1 で検算するクセを。「番目 n」と「正方形の個数」を別の文字にしない＝この問題では番目＝正方形の個数なので同じ n でよい。
⑦ 検算＋次に見抜くコツ
n＝1,2,3 を式に入れて表（4,7,10）と全部一致○。
コツは ①差を見る（3）→ 係数 ②1番目に合わせる（＋1）→ 定数 ③確かめる（n＝1で必ず）。この3手で必ず正しい式になる。
「具体（数えた表）→ 差 → 3n＋1 という n の式」へ翻訳するこの流れが、9月大問4(2)でそのまま問われる。式づくりが規則性の心臓。
こたえn番目＝3n＋1（本）

問3③逆算：大きい番目を方程式で出す L4目安 8分
問3問2の式（n番目＝3n＋1）を使って答えなさい。(1) 20番目のマッチ棒は何本ですか。(2) マッチ棒をちょうど100本使うのは何番目ですか。
(1)は n に数を「入れる（代入）」。(2)は逆に「式＝100」という n の方程式を解く。型カードの④。ノートに式を書いてから解こう。
① 何を聞かれてる？（やさしい言いかえ）
(1)「20番目って何本？」＝番目→本数を出す。(2)「100本になるのは何番目？」＝本数→番目を逆に出す。向きが反対なのがポイント。
② わかってること／求めるもの
  • わかってること：式は 3n＋1（問2）
  • (1) 求めるもの：n＝20 のときの本数
  • (2) 求めるもの：本数が100になる番目 n
③ 図でイメージ
(1)は「20個ならべた正方形の列」を想像。全部数えるのは大変だから式で一発。(2)は逆に「100本のマッチ棒をぜんぶ使い切ると、正方形は何個ならぶ？」を聞かれている。
④ 作戦＋たとえ話
式は「両方向に使える道具」。代入＝行き（番目→本数）、逆算＝帰り（本数→番目）。帰り道は「箱を開ける順番」が大事：3n＋1 は「n を3倍して1を足した」順だから、もどすときは逆順で、先に1を引いて、あとで3で割る。くつをはくとき（くつ下→くつ）、ぬぐときは逆（くつ→くつ下）と同じ。
⑤ 1ステップずつ（なぜ付き）
n式ができれば 「番目→本数」は代入、「本数→何番目」は方程式（逆算）。これが規則性の最後のゴール。
(1) 代入n＝20 を 3n＋1 へ：3×20＋1＝60＋1＝61。61本。（なぜ：式の n に20を入れるだけ）
(2) 逆算＝方程式を立てる本数＝100 だから 3n＋1＝100。（なぜ：式が表す本数を100に等しくおく）
(2) 解く（順番が命）まず両辺−1 → 3n＝99。次に両辺÷3 → n＝33。33番目。（なぜ：先に＋1を消してから、3で割る。逆順）
⑥ よくある間違い
100÷3＝33.3… と割るだけにしない。それは「＋1」を忘れた答え。逆算は先に＋1を移項して引いてから割る（3n＝99 → n＝33）。(2)で61本と書かない＝それは(1)の答え。聞かれているのは「番目」。代入と逆算がごちゃ混ぜにならないよう、(1)は本数・(2)は番目と確認。
⑦ 検算＋次に見抜くコツ
(2)の答え n＝33 を式へ戻す：3×33＋1＝99＋1＝100○。ちゃんと100本になる。
コツ：「○番目は？」と聞かれたら代入、「○本になるのは何番目？」と聞かれたら方程式（逆算）。問題文の最後をよく読んで、行きか帰りかを見ぬく。
この(2)が9月大問4(3)の「○本になるのは何番目」型とまったく同じ。代入と逆算をセットで出せれば大問4は取れる。
こたえ(1) 61本／(2) 33番目

問4★別パターン：碁石を正方形の周にならべる L3-L4目安 8分
問4碁石を、正方形のまわり（周）にそってならべます。1辺に2個・3個・4個…とならべると、下の図のようになります。
●＝碁石。正方形の周だけにならべる（中はからっぽ）。1辺の個数を n とする。
1辺の個数（n）234…
碁石(個)4812…
(1) 碁石の個数を n の式で表しなさい。
(2) 碁石をちょうど 60個 つかうのは、1辺が何個のときですか。
1辺が n 個のとき、4つの角（かど）を二重に数えないのがコツ。表の差を見て式を作り、(2)は式＝60の方程式を解こう。
① 何を聞かれてる？（やさしい言いかえ）
(1)「1辺 n 個のとき、周の碁石は全部で何個？」を式に。(2)「60個ちょうどになるのは、1辺いくつ？」を逆算で。問1〜3と同じ流れの別パターン。
② わかってること／求めるもの
  • わかってること：表（n＝2で4個、3で8個、4で12個）
  • (1) 求めるもの：碁石の個数の n 式
  • (2) 求めるもの：60個になる1辺の個数 n
③ 図でイメージ
1辺を大きくすると、図の上の辺・下の辺・左の辺・右の辺、それぞれに1個ずつ＝4個増えていく（n＝3→4で 8個→12個、＋4）。角の碁石は2つの辺が共有しているので、数えるとき角を二重に数えないのがポイント。
④ 作戦＋たとえ話
考え方は問1とまったく同じ「差×n＋調整」。1辺がのびると4方向に1個ずつ増えるから差は4。あとは「n＝2で4個になるように調整」するだけ。マッチ棒も碁石も、表→差→式→検算の同じレールに乗せれば解ける。
⑤ 1ステップずつ（なぜ付き）
差を見る8−4＝4、12−8＝4。差は4。（なぜ：1辺がのびると4方向に1個ずつ）
(1) 式を作る差4、n＝2のとき4個。4×n＋調整。4×2＝8 だが本当は4個なので、調整は −4。式は 4n−4。（なぜ：角の4個を二重に数えたぶんを引くイメージ）
確かめn＝3：4×3−4＝8○、n＝4：4×4−4＝12○。表と合う。
(2) 逆算碁石＝60 だから 4n−4＝60。両辺＋4 → 4n＝64。両辺÷4 → n＝16。1辺16個。（なぜ：−4をもどして＋4、それから÷4）
⑥ よくある間違い
1辺を4倍して「4n」で終わらせない＝角の碁石を二重に数えてしまう。本当は 4n−4（角の4個ぶん引く）。(2)で 60÷4＝15 と即答しない＝「−4」を忘れた答え。先に＋4してから÷4（4n＝64 → n＝16）。
⑦ 検算＋次に見抜くコツ
(2)の n＝16 を式へ：4×16−4＝64−4＝60○。ちゃんと60個。
コツは問1〜3と同じ ①差を見る（4）②1番目に合わせる（−4）③確かめる。パターンが変わっても、レールは同じ。表を作って差から式、そして逆算は「足し引きを先・かけ割りをあと」で戻す。
9月特訓選抜・大問4は「マッチ棒・碁石・タイル」など見た目を変えて出る。どんな図でも 表→差→n式→逆算 の同じレールに乗せられれば、初見の図でも取れる。これが大問4攻略の本体。
こたえ(1) 4n−4（個）／(2) 1辺16個
