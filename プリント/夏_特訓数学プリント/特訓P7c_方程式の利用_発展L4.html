方程式の利用・発展｜9月特訓選抜の「大問6（速さの最難問）」を直撃する4問

夏・特訓対策　目安：約35分（問1〜問4・通し番号）　対象: 中1　全問に徹底解説つき

ここは特訓選抜のいちばんの山＝大問6。出会い・追い越し・時刻の逆算・動点と、ひとつ上のレベルがそろっている。むずかしいけれど、やることは今までと同じ＝①図をかく ②等しいものを＝で結ぶ ③解いて文章にもどす。今日は4問だけ。1問ずつ図をていねいにかいて、「型」を体に入れよう。とけなくても、図がかければ大きな一歩だ。

  1. ① 何を求める？ → それを x とおく（時刻を聞かれても、まず「○分後」を x にすると楽）
  2. ② 図にする（線分図・ダイヤ図・動点図）＝だれがどこをどれだけ進むかを見える形に
  3. ③ 等しい関係を式に（道のり・面積などで「＝で結べる2つの言い方」をさがす）
  4. ④ 解く（移項して x ＝ の形へ。分数は両辺×分母で払う）
  5. ⑤ 検算＆もどす（出た x を文章に戻す。時刻は「出発時刻＋x分」で何時何分かに直す）
問1★出会い＋出会う時刻（同時出発・向かい合い）L3 発展目安 8分
問11800mはなれたP地点とQ地点がある。9時ちょうどに、きみはPを分速70mで、もう1人はQを分速80mで、同時に向かい合って歩き出した。(1) 2人は出発して何分後に出会うか。(2) 出会う時刻は何時何分か。
向かい合って進むと、2人の進んだ道のりを合わせると、はじめの距離（1800m）になる。出会うまでの時間を x にして、まず(1)。時刻(2)は「9時＋x分」で出す。
⓪ 速さの3つの関係
  • 道のり＝速さ×時間　時間＝道のり÷速さ　速さ＝道のり÷時間
  • 「出会い」＝向かい合って近づく → 2人の道のりを足すと、はじめの間の距離になる。
出会い算＝2人の進んだ道のりの合計＝はじめにはなれていた距離。同時出発なので、2人とも進んだ時間は同じ x。時刻は最後に「出発時刻＋x分」で直す。
①x出発してから x 分後に出会うとする（2人とも x 分歩く）。
③式道のりの合計＝1800：きみは 70x m、もう1人は 80x m 進む。70x＋80x＝1800。
④解150x＝1800 → x＝12。（1）の答え＝12分後。
⑤検算＆もどすきみ 70×12＝840m、相手 80×12＝960m。840＋960＝1800m○。時刻は 9時00分＋12分＝9時12分。
「合計」を「差」にしない。向かい合うと近づくので、2人の道のりは足す。差にすると追い越し算とまざる。(2)で「12」とだけ書かない＝聞かれたのは時刻。9時に12分を足して「9時12分」と答える。
出会い→出会う時刻は特訓選抜 大問6(1)の定番。「○分後」を出してから「出発時刻＋○分」で時刻に直す流れは、次の問2（追い越し）でもそのまま使う。

問2★追い越し＋時刻の逆算（後から出た人が追いつく時刻）L4 発展・最難目安 10分
問2妹が8時50分に家を出て、分速60mで駅へ歩き出した。姉は8時59分に家を出て、同じ道を分速150mで自転車で追いかけた。姉が妹に追いつくのは何時何分か。
追いつく＝「2人の進んだ道のりが同じ」になったとき。妹は姉より早く出ている（差は何分？）。姉が出発してからの時間を x にして、まず「何分後に追いつくか」を出し、最後に時刻へ逆算しよう。
追い越し算＝2人の進んだ道のりが等しくなる。同じ家から同じ道を進むので、追いついた瞬間は「家からの道のり」が一致＝＝で結ぶ。出会い（合計）とちがう点に注意。時刻は最後に「姉の出発時刻＋x分」で直す（時をまたぐので+−に注意）。
①x姉が出発してから x 分後に追いつくとする。先に出た差は 8時59分−8時50分＝9分。だから、そのとき妹は (x＋9) 分歩いている。
③式道のりが等しい：姉 150x ＝ 妹 60(x＋9)。150x＝60(x＋9)。
④解150x＝60x＋540 → 150x−60x＝540 → 90x＝540 → x＝6。姉が出発してから6分後。
⑤検算＆もどす姉 150×6＝900m、妹 60×(6＋9)＝60×15＝900m。どちらも900mで一致○。時刻は 8時59分＋6分。59＋6＝65分＝1時間5分くりあがるので 9時05分。
妹の時間を x にしない（妹は x＋9 分歩いている）。60(x＋9) を 60x＋9 としない＝かっこの中の全部に60をかける（＝60x＋540）。時刻のくりあがりに注意＝8時59分＋6分は「8時65分」ではなく9時05分。
これが特訓選抜 大問6(2)の最難パターンそのもの。「先に出た人は時間が多い → ＋で式に入れる」「答えを時刻に逆算する（くりあがり）」の2つが山。問1の出会い（足す）と見くらべ、足すか＝かを図で判断しよう。
こたえ9時05分（姉が出発して6分後）

問3★バス2台のすれ違い（速さを先に求める）L4 本番級目安 9分
問3A駅とB駅は24kmはなれています。バスPは9時00分にA駅を出発し、バスQは9時06分にB駅を出発して、たがいに向かい合って同じ速さで走りました。2台は9時18分にすれ違いました。バスの速さは時速何kmですか。
2台の速さを v km/h とする。Pは18分、Qは12分走っている。時間を「時間」に直して、2台の進んだ道のりの和＝24km にする。
本番の速さ発展は、いきなり自転車を考えず、まずバスの速さを条件から推定する。すれ違いは、2台の進んだ道のりの和がA駅〜B駅の距離になる。
①xバスP・Qの速さを v km/h とする。
P：9:00→9:18 ＝ 18分 ＝ 18/60時間 Q：9:06→9:18 ＝ 12分 ＝ 12/60時間 Pの道のり + Qの道のり = 24km
③式v×18/60＋v×12/60＝24。
④解18/60＋12/60＝30/60＝1/2。だから v×1/2＝24 → v＝48。
⑤検算Pは48×18/60＝14.4km、Qは48×12/60＝9.6km。14.4＋9.6＝24km○。
18分と12分をそのまま18時間・12時間として使わない。時速なので、分は必ず60で割って時間に直す。PとQの時間は同じではない。Qは9時06分出発なので12分だけ。

問4★バス2台＋自転車（追い越し・すれ違い時刻逆算）L4-L5目安 12分
問4問3と同じA駅・B駅間で、バスPは9時00分にA駅を出発し、バスQは9時06分にB駅を出発します。ある人が自転車でA駅を出発したところ、9時10分にバスPに追い越され、その12分後にバスQとすれ違いました。自転車がA駅を出発した時刻を求めなさい。
問3のバスの速さ48km/hを使う。9:10にバスPがいる位置、9:22にバスQがいる位置をそれぞれA駅からの距離で出す。自転車はその2点を12分で進んだ。
これは本番最難型。バスの位置を2回出す → その間の自転車の速さを出す → 追い越された時刻から出発時刻を逆算する。1本の式で終わらない、2段処理の問題。
①Pの位置バスPは9:00から9:10まで10分走る。48km/h×10/60＝8km。9:10に自転車はA駅から8kmの地点で追い越された。
②Qの位置「その12分後」は9:22。バスQは9:06から9:22まで16分走る。B駅から48×16/60＝12.8km進む。A駅から見ると 24−12.8＝11.2km。
③自転車の速さ自転車は9:10から9:22の12分で、8km地点から11.2km地点まで進む。進んだ道のりは3.2km。速さ＝3.2÷(12/60)＝16km/h。
④出発時刻自転車がA駅から8km地点まで行く時間は、8÷16＝0.5時間＝30分。9:10の30分前に出発したので、8時40分。
⑤検算8:40→9:10で30分、16km/hなら8km進む。9:10→9:22で12分、さらに3.2km進んで11.2km地点。Qも9:22にAから11.2km地点なので一致○。
Qの位置をB駅からの距離のまま使わない。自転車はA駅から進むので、Qの位置もA駅からの距離に直す（24−12.8）。最後に「30分」と答えない。聞かれているのは出発時刻。

大問6対策の発展4問、本当におつかれさま！今日の収穫は「速さの問題は、時刻ごとの位置をA駅からの距離にそろえる」こと。出会い＝道のりの合計、追い越し＝道のりが等しい、そして本番級は「バスの速さを先に求める→別の時刻の位置を出す→出発時刻へ戻す」。とけなかった問題は、時刻・位置・速さを表にしてノートに書き直そう。この山を登れたら、特訓選抜の大問6はきみの得点源になる🌸
