文章を式にする②｜速さ（往復・追いつき・出会い）と 割合・利益

夏・特訓対策　目安：約30分（問1〜問8・通し番号）　対象: 中1　全問に徹底解説つき

P4の型カードはそのまま。今日は「速さ」と「割合」を式にする回。コツは①速さは道のり＝速さ×時間と単位をそろえる②割合は「○割引き＝×(1−○)」「○%増し＝×(1＋○/100)」。式さえ作れれば、あとはP3の解法で解くだけ。

  1. ① 何を求める？ → x とおく　② 単位をそろえる（時速↔分速、km↔m）
  2. ③ 図にする（線分図）　④ 等しい関係を式に（道のり・代金・割合で＝）
  3. ⑤ 解いて検算
問1〜2速さ（道のり＝速さ×時間・往復）L2→L3目安 8分
問1時速4kmで x 時間歩いたら、進んだ道のりは12kmでした。x を求めなさい。

問2家から駅まで、行きは分速60m、帰りは分速80mで往復したら、合計で35分かかりました。家から駅までの道のりは何mですか。
「道のり・速さ・時間」のどれを x にするか決め、図と式をノートに書こう。
⓪ 速さの3つの関係
  • 道のり＝速さ×時間　時間＝道のり÷速さ　速さ＝道のり÷時間
  • 単位をそろえる：時速と分、km と m が混ざったら必ず直す。
問1道のり＝速さ×時間：4×x＝12 → x＝3。検算 4×3＝12○。3時間。
問2 ①x片道の道のりを x m とする。時間＝道のり÷速さ。
問2 ④式行き x60分＋帰り x80分＝35。x60＋x80＝35。
問2 ⑤解両辺×240：4x＋3x＝8400 → 7x＝8400 → x＝1200。検算 20＋15＝35分○。1200m。
問2 往復の「合計時間」で式を立てる（道のりは行き帰り同じ x）。分数は両辺×（分母の最小公倍数）で払う。
こたえ問1 x＝3（3時間）／問2 1200m

問3〜4★速さ（追いつき・出会い）L3 応用目安 8分
問3妹が分速60mで家を出発しました。その8分後に姉が分速100mで妹を追いかけました。姉が出発してから何分後に妹に追いつきますか。

問42000mはなれたA地点とB地点から、PさんはAを分速70m、QさんはBを分速130mで、同時に向かい合って出発しました。2人は何分後に出会いますか。
線分図をかいて「だれとだれの道のりが等しいか／合わせて全体か」を見つけてから式にしよう。
追いつき＝2人の進んだ道のりが等しくなる。出会い＝2人の道のりの合計＝はじめの間の距離。図にすると等式が見える。
問3 ①x姉が出発してから x 分後に追いつくとする。妹は姉より8分多く歩いている。
妹：60×(x＋8) ─────────────→ 姉：100×x ───────────→（追いつく＝同じ道のり）
問3 ④式道のりが等しい：100x＝60(x＋8)。
問3 ⑤解100x＝60x＋480 → 40x＝480 → x＝12。検算 姉100×12＝1200、妹60×20＝1200○。12分後。
問4 ①x出発から x 分後に出会うとする。
A→P:70x ──→●←── Q:130x ←B　（合計＝2000m）
問4 ④式道のりの合計＝2000：70x＋130x＝2000。
問4 ⑤解200x＝2000 → x＝10。検算 700＋1300＝2000○。10分後。
問3 妹の時間を x にしない（妹は x＋8分歩いている）。問4 差でなく和（向かい合うので合計が2000）。
追いつき・出会いは特訓選抜 大問6（速さ）の定番。線分図で「等しい関係」を見つける練習が効く。
こたえ問3 12分後／問4 10分後

問5〜6割合（％・割引き）を式に L2→L3目安 7分
問5ある数の15%が90です。ある数を求めなさい。

問6定価 x 円の品物を2割引きで買ったら、代金は1600円でした。定価を求めなさい。
「もとにする量を x」にして、割合のかけ算を式にしてからノートで解こう。
⓪ 割合のことば → 数
  • ○%＝○100（15%＝0.15）　○割＝○10（2割＝0.2）
  • □%引き＝×(1−□/100)（2割引き＝×0.8）　□%増し＝×(1＋□/100)
  • 「の」はかけ算：x の15%＝x×0.15。
問5x の15%＝90：0.15x＝90 → x＝90÷0.15＝600。検算 600×0.15＝90○。600。
問62割引き＝×(1−0.2)＝×0.8。0.8x＝1600 → x＝2000。検算 2000×0.8＝1600○。2000円。
問6 2割引きを ×0.2 としない（引いた残り＝×0.8）。「割引き」は残りをかける。
こたえ問5 600／問6 2000円

問7〜8★利益（原価・定価）と 食塩水の入口 L3 応用目安 7分
問7原価 x 円の品物に、2割の利益を見込んで定価をつけたら、定価は3600円になりました。原価を求めなさい。

問8濃度8%の食塩水が x g あります。この中にとけている食塩が40gのとき、食塩水は何gですか。
問7は「定価＝原価×（1＋利益の割合）」、問8は「食塩＝食塩水×濃度」を式にしてからノートで解こう。
問72割の利益＝×(1＋0.2)＝×1.2。1.2x＝3600 → x＝3000。検算 3000×1.2＝3600○。原価3000円。
問8 ④式食塩＝食塩水×濃度：x×0.08＝40（濃度8%＝0.08）。
問8 ⑤解0.08x＝40 → x＝40÷0.08＝500。検算 500×0.08＝40○。500g。
問7 利益を見込む＝×1.2（足す）、割引きは×0.8（引く）。混同しない。問8 8%は0.08（0.8ではない）。
「食塩＝食塩水×濃度」は中2の食塩水（連立方程式）の土台。今は1種類の式で入口を固める。
