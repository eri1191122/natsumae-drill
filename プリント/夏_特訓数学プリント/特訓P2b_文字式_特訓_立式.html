文字式 特訓｜負の数の代入（かっこが命）と「立式」（ことば・文章を式に）

夏・特訓対策　目安：約30分（問1〜問19・通し番号）　対象: 中1　全問に徹底解説つき

P2aで手順を固めたら特訓レベルへ。①負の数の代入は (　) で囲む（x²＝9 と −x²＝−9）。②立式＝ことば・文章を式に。立式は夏のいちばん大事な「文章→式」と方程式の心臓部。まず自分で書いて、それから解説へ。解説は小問ごとに色分け。

問1〜5★負の数の代入（かっこが命・x²と−x²）L3 応用目安 6分
x＝−3 のとき、次の式の値を求めよう。
問1 2x　問2 x²　問3 −x²　問4 −x　問5 x²−x
途中式をノートに書いてから答え合わせをしよう。
⓪ 負の数を代入するときの鉄則

文字を必ず (　) で囲んで置きかえる。x＝−3 なら x のところに (−3)。こうすると「−の符号」と「2乗」がぐちゃぐちゃにならない。

① x² と −x² は別もの（最重要）
  • x²＝(−3)²＝＋9（かっこごと2乗＝−も消える）。
  • −x²＝−(−3)²＝−9（−は2乗の外。先に9、最後に−）。
かっこを書かず −3² と書くと −9 の意味に。負の数は必ずかっこで囲む。
合言葉：「負の数はかっこで囲んで代入」。これで問2 x²＝＋9 と 問3 −x²＝−9 のちがいが出る。特訓選抜・全統中で必出。
問12x＝2×(−3)＝−6（2−3＝−1 としない）。
問2x²＝(−3)×(−3)＝＋9。
問3−x²＝−(−3)²＝−9。−は2乗の外。
問4−x＝−(−3)＝＋3。マイナスのマイナスはプラス。
問5x²−x＝9−(−3)＝12。−x の部分は問4と同じ＋3。
検算問1 −6÷2＝−3＝x。問2 (−3)×(−3)＝9。問5 9＋3＝12。合う。
x² と −x² の区別・「負の数はかっこ」は中3の平方根・2次関数までずっと問われる超重要点。
(−3)² を −9 としない（＝＋9）。−x² を ＋9 としない（＝−9）。2x を 2−3＝−1 としない。
こたえ問1 −6／問2 9／問3 −9／問4 3／問5 12

問10〜13★数量を文字式で表す（立式）L3 応用目安 6分
次の数量を文字式で表そう（単位もつける）。
問10 1個 x 円のケーキ3個を y 円の箱に入れた代金
問11 分速70mで a 分歩いた道のり
問12 定価 x 円の品物を2割引きで買った代金
問13 お菓子を m 人に n 個ずつ配ったら3個あまった。全部で何個か
ことばを式にして、ノートに書こう。
立式の3ステップ：①ことばで関係を書く②具体的な数で式を作る③数を文字に置きかえる。割合は「2割引き＝0.8倍」、速さは「道のり＝速さ×時間」。
問10代金＝(x×3)＋y＝3x＋y（円）。
問11道のり＝速さ×時間＝70×a＝70a（m）。
問122割引き＝(1−0.2)＝0.8倍。代金＝x×0.8＝0.8x（円）。
問13配った数＝n×m＝mn。あまり3も足して mn＋3（個）。
確かめ問10 x＝400,y＝200→1400円。問12 x＝500→400円。問13 m＝5,n＝4→23個。合う。
これが「立式」＝文章→式。0/25で苦しんだ所＆方程式の利用（特訓大問3）の始まり。今は答えを出さず式で表すだけ。
問12 2割引きを ×0.2 としない（残りは ×0.8）。問13 m＋n＋3 としない（n個ずつm人＝mn）。「ずつ」はかけ算。
こたえ問10 3x＋y 円／問11 70a m／問12 0.8x 円／問13 mn＋3 個

問18〜19文字式の利用（整数を文字で表す）L3 応用目安 4分
問18 連続する3つの整数のうち真ん中が n のとき、3つの和を n の式で表し簡単にしよう。
問19 十の位が a、一の位が b である2けたの自然数を文字式で表そう。
ノートに書いて考えよう。
問18真ん中が n なら前は n−1、後は n＋1。和は (n−1)＋n＋(n＋1)＝3n（−1と＋1が消える）。確かめ：真ん中5（4,5,6）の和15＝3×5。
問192けたの数＝10×(十の位)＋(一の位)。例34＝10×3＋4。だから 10a＋b。
「整数を文字で表す（連続整数・位の数）」は特訓・高校入試の文字式の利用や証明で頻出。10a＋b は中2の証明でそのまま使う。
問18 n＋(n＋1)＋(n＋2) としない（真ん中が n なので前は n−1）。問19 2けたの数を ab と書かない（ab は a×b。位の数は 10a＋b）。
こたえ問18 (n−1)＋n＋(n＋1)＝3n／問19 10a＋b
