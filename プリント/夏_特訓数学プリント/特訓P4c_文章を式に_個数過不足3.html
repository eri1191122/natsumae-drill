特訓P4c数学：方程式の利用★文章→式 L2-L3（最重要）

文章を式にする③｜個数・過不足の型カードを「4問・じっくり」でもう一度固める

夏・特訓対策　目安：約30分（問1〜問4・通し番号）　対象: 中1　全問に徹底解説つき

これがきみのいちばんの伸びしろ＝「文章を式にする」力。全国統一テストで0/25だったのもここ。前回（P4b）と数も話も変えたのは、答えを暗記するのではなく「型」を体に入れるため。今日も4問だけ。1問ずつ型カードにそって途中式をていねいに書こう。

  1. ① 何を求める？ → それを x とおく（「□を x とする」と書く）
  2. ② 図か表にする（線分図・表）＝目で見える形に
  3. ③ 等しい関係を式に（＝で結べる2つの言い方を探す）
  4. ④ 解く（移項して x ＝ の形へ）
  5. ⑤ 検算（出た数を文章に戻して合うか確かめる）
問12種類の個数（合計が決まっている）L2 標準目安 7分
問11本120円の鉛筆と、1冊180円のノートを合わせて15個買ったら、代金は2280円でした。鉛筆とノートをそれぞれ何個買いましたか。
①〜⑤の型カードを使う。片方を x、もう片方を「15−x」と表にしてからノートで式を作ろう。
2種類で合計個数が決まっているとき：片方を x、もう片方は「合計−x」で表す。これで文字は1つで済む。
①x鉛筆を x 本とする → ノートは (15−x) 冊。
┌──────┬───────┬──────────┐ │ │ 鉛筆 │ ノート │ ├──────┼───────┼──────────┤ │ 個数 │ x │ 15−x │ │ 金額 │ 120x │ 180(15−x) │ └──────┴───────┴──────────┘
③式金額の合計＝2280。120x＋180(15−x)＝2280。
④解120x＋2700−180x＝2280 → −60x＝−420 → x＝7。
⑤検算鉛筆7本、ノートは(15−7)＝8冊。120×7＋180×8＝840＋1440＝2280○。
180(15−x) を 180×15−x としない（＝2700−180x。x にも180をかける）。かっこの前の数は、中の全部にかける。
「片方を x、もう片方を 合計−x」は特訓選抜 大問3（方程式の利用）の超定番。個数・代金・速さの混合でも同じ。
こたえ鉛筆 7本／ノート 8冊

問22種類の人数（合計人数と合計料金）L3 標準目安 7分
問2ある美術館に、大人と子どもが合わせて30人で入りました。入館料は大人1人500円、子ども1人200円で、全員の合計は10500円でした。子どもは何人ですか。
問1と同じ型。今回きかれているのは「子ども」。子どもを x 人とすると大人は「30−x」人。表にしてから式を作ろう。
きかれているものを x にすると、最後がラク。今回は子どもを x、大人は「合計−x＝30−x」。
①x子どもを x 人とする（求めるもの）→ 大人は (30−x) 人。
┌──────┬──────────┬────────┐ │ │ 大人 │ 子ども │ ├──────┼──────────┼────────┤ │ 人数 │ 30−x │ x │ │ 料金 │ 500(30−x) │ 200x │ └──────┴──────────┴────────┘
③式料金の合計＝10500。500(30−x)＋200x＝10500。
④解15000−500x＋200x＝10500 → −300x＝−4500 → x＝15。
⑤検算子ども15人、大人(30−15)＝15人。500×15＋200×15＝7500＋3000＝10500○。
きかれたのは「子どもの人数」。x にしたものが子どもだから、x がそのまま答え。大人の数(15人)を答えにしないよう、最後に「何を x にしたか」を読み直す。
こたえ子ども 15人

問3★過不足算（あまる・たりない）L3 応用目安 8分
問3画用紙を子どもに配ります。1人に5枚ずつ配ると12枚あまり、1人に7枚ずつ配ると4枚たりません。子どもの人数を求めなさい。
「配り方が2通り→変わらない『画用紙の全部の数』を2通りで表して＝で結ぶ」。人数を x にしよう。
過不足のコツ：変わらないのは「画用紙の全部の数」。これを2通りの配り方で表して＝で結ぶ。あまる＝＋、たりない＝−（全部＝1人分×人数＋あまり ／ 全部＝1人分×人数−たりない）。
①x子どもの人数を x 人とする（求めるもの）。
③式画用紙の数を2通りで：5枚ずつ＝5x＋12（あまる）、7枚ずつ＝7x−4（たりない）。同じ画用紙だから 5x＋12＝7x−4。
④解12＋4＝7x−5x → 16＝2x → x＝8。
⑤検算5×8＋12＝40＋12＝52、7×8−4＝56−4＝52。どちらも52枚で一致○。
「たりない」を ＋ にしない（たりない＝−）。1人に配る数が多い方ほど、たりなくなる、と図でイメージすると符号をまちがえない。
過不足は特訓選抜 大問3の定番。「変わらない全体を2通りで表して＝」は、椅子・班分け・お金の問題でもそのまま使える。
こたえ子ども 8人（画用紙は52枚）

問4★班分け型（あまる・最後の1班だけ少ない）L3 応用目安 8分
問4生徒を何班かに分けます。1班に6人ずつ入れると10人が入りきれずあまります。1班に8人ずつ入れると、最後の1班だけ2人になります。班はいくつあり、生徒は何人ですか。
変わらないのは「生徒の人数」。これを班の数を使って2通りで表そう。班の数を x にする。
班分けも過不足と同じ：変わらない「生徒の人数」を、班の数 x を使って2通りで表して＝で結ぶ。「最後の1班だけ2人」＝最後の班以外（x−1班）は8人ずつ、最後だけ2人。
①x班の数を x 班とする。
③式6人ずつだと10人あまる＝6x＋10（人）。8人ずつだと最後の1班だけ2人＝(x−1)班に8人ずつ＋最後2人＝8(x−1)＋2（人）。同じ生徒数だから 6x＋10＝8(x−1)＋2。
④解6x＋10＝8x−8＋2 → 6x＋10＝8x−6 → 10＋6＝8x−6x → 16＝2x → x＝8。
⑤検算班の数8班。生徒＝6×8＋10＝58人。8人ずつ：7班に56人、最後の1班に2人で計58人○。
「あまる10人」は ＋（入りきらず人があまる＝人数の方が多い）。「最後だけ2人」を 8x＋2 としない＝最後の班は8人ではなく2人なので、8人の班は (x−1) 班分だけ。
こたえ班 8つ／生徒 58人
