特訓P9a数学：規則性演習 L2-L3（差→n式の基礎）

規則性 演習｜「差を見つけて n の式にする」を反射でできるまで

夏・特訓対策　目安：約30分（問1〜問8・通し番号）　対象: 中1　全問に徹底解説つき

P9（碁石の周・L4）の前の手順固め。コツは①差を見る②差×n＋（1番目−差）③n＝1で確かめ。数の並び・表・かんたんな●の図で、たくさん練習しよう。ここが速くなると応用がラクになる。

  1. ① 差を見る（毎回いくつ増える／減る？）
  2. ② n式＝差×n＋（1番目−差）（減るときは差がマイナス）
  3. ③ n＝1, 2 で確かめる
問1〜4数の並びを n の式に L2目安 8分
次の並びの n番目を n の式で表しなさい。
問12, 5, 8, 11, …　問21, 5, 9, 13, …　問320, 17, 14, 11, …　問45, 11, 17, 23, …
差を書きこんでから、差×n＋（1番目−差）で式を作ろう。
問1差3。3n＋(2−3)＝3n−1。確かめ n＝1→2○。
問2差4。4n＋(1−4)＝4n−3。確かめ n＝1→1○。
問3差は−3（減る）。−3n＋(20−(−3))＝−3n＋23。確かめ n＝1→20○。
問4差6。6n＋(5−6)＝6n−1。確かめ n＝1→5○。
問3 減るときは差がマイナス（−3）。差だけで終わらず「1番目−差」の調整を必ず足す。

問7〜8●の並びを n の式に（図から差を見る）L3目安 6分
問7●を下の図のように並べます。n番目の●の個数を n の式で表しなさい。 問8●を2段に並べます。1番目2個、2番目4個、3番目6個、…。n番目の個数を n の式で。
1・2・3番目の個数を表に書き、差を見てから n の式にしよう。
問71,3,5,…差2。2n＋(1−2)＝2n−1（奇数の式）。確かめ n＝3→5○。
問82,4,6,…差2。2n＋(2−2)＝2n（偶数の式・2段×n列）。確かめ n＝3→6○。
図でも「1・2・3番目の数を並べて差を見る」が基本。●を数え間違えないよう図に数を書く。
