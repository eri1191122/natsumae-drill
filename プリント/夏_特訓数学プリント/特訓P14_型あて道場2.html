特訓P14数学：方程式の利用型あて道場② L3-L4

型あて道場②｜「これは何型？」を自分で見抜く特訓

夏・特訓対策　目安：約30分（第1問〜第6問）　対象: 中1　全問に徹底解説つき

今日も見出しに型名を出さない。きみが自分で「これは何型か」を見抜くのが今日の主役。本番の大問3も、最初に型を当てられるかどうかで決まる。数字はいつも変わるから暗記はきかない。だから「どの言葉でその型と分かるか」を体で覚えよう。今日はP13より一段むずかしいL3-L4。1問だけ今までに無い新しい型がまざってる。あわてず①何型？から。最後の第6問はひっかけ。あわてず言葉をよく読もう🌸

  1. ① これは何型？ をまず書く（下の4つから選ぶか、自分の言葉で）
    　⑴ 個数・代金型　⑵ 速さ・往復型　⑶ 過不足型　⑷ 割合・利益型
  2. ② x を置く（「□を x とする」と書く）
  3. ③ 式を作る（＝で結べる2つの言い方を探す）
  4. ④ 解く（移項して x ＝ の形へ）
  5. ⑤ 検算（出た数を文章に戻して合うか確かめる）
  6. 最後に 型が当たったかも ○✕ をつける
第1問L3 標準目安 5分
第1問1個130円のパンと、1個90円のおにぎりを、合わせて20個買ったら、代金は2200円でした。パンは何個買いましたか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。
個数・代金型。「合わせて20個」と合計の個数が決まっているのが見分け方。片方を x、もう片方は「合計−x」で表すと文字は1つで済む。
①xパンを x 個とする → おにぎりは (20−x) 個。
③式金額の合計＝2200。130x＋90(20−x)＝2200。
④解130x＋1800−90x＝2200 → 40x＝400 → x＝10。
⑤検算パン10個、おにぎり(20−10)＝10個。130×10＋90×10＝1300＋900＝2200○。
90(20−x) を 90×20−x としない（x にも90をかける＝1800−90x）。最後はパンを答える（x がそのまま答え）。
「合計の個数があれば片方を合計−xで表す」は大問3(1)の定番。数字が変わっても見分け方は同じ。
こたえパン 10個（おにぎり 10個）

第2問L3-4 標準〜応用目安 5分
第2問A町とB町の間を、行きは時速4km、帰りは時速6kmで往復しました。往復にかかった時間は合わせて5時間でした。A町とB町の間の道のりは何kmですか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。
速さ・往復型。行きと帰りで速さがちがい、時間の合計が分かっているのが見分け方。等しい関係は「行きの時間＋帰りの時間＝5時間」。時間＝道のり÷速さを使う。
①xA町B町間の道のりを x km とする（行きも帰りも同じ道）。
③式行きの時間＝x÷4、帰りの時間＝x÷6。合計が5時間だから x/4＋x/6＝5。
④解両辺を12倍（4と6の最小公倍数）→ 3x＋2x＝60 → 5x＝60 → x＝12。
⑤検算行き 12÷4＝3時間、帰り 12÷6＝2時間。3＋2＝5時間○。
行きと帰りの速さを足して平均にしない（(4＋6)÷2 は使えない）。時間＝道のり÷速さ。割る向きをまちがえないこと。
「時間の合計＝行き÷速さ＋帰り÷速さ」は大問3(2)で頻出。分数の式は最小公倍数を両辺にかけて整数にすると楽。
こたえ道のり 12km

第3問L4 応用目安 5分
第3問みかんを子どもたちに配ります。1人に4個ずつ配ると6個あまり、1人に6個ずつ配ると、最後の1人だけが2個になりました。子どもの人数とみかんの個数を求めなさい。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。
過不足型。配り方が2通りで「あまる」がある。「最後の1人だけ2個」はその人だけ6個に4個たりない＝全体で4個たりないと読みかえるのがコツ。変わらないみかんの数を2通りで表して＝で結ぶ。
①x子どもの人数を x 人とする。
4個ずつ：4x ＋ あまり6 → みかん＝4x＋6 6個ずつ：本来6x、でも最後の1人は6個でなく2個 ＝6個に4個たりない → みかん＝6x−4
③式みかんの数を2通りで：4個ずつ＝4x＋6、6個ずつは最後の1人が4個少ないので＝6x−4。同じみかんだから 4x＋6＝6x−4。
④解6＋4＝6x−4x → 10＝2x → x＝5。みかん＝4×5＋6＝26個。
⑤検算子ども5人、みかん26個。4個ずつ：4×5＝20、あまり6で計26○。6個ずつ：4人に6個＝24、最後の1人に残り2個、計26○。
「最後の1人だけ2個」を「2個あまる」と読まない。本当は6個ほしいのに2個しかない＝6−2＝4個たりない。だから 6x−4。あまる＝＋、たりない＝−。
「最後の1人だけ○個」型は大問3(3)の応用。『満杯なら何個・実際は何個』の差を「たりない数」に直すのがカギ。
こたえ子ども 5人／みかん 26個

第4問L4 応用目安 5分
第4問ある品物に、原価の4割の利益を見込んで定価をつけました。しかし売れなかったので、定価の1割引きで売ったところ、利益は260円でした。この品物の原価はいくらですか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。
割合・利益型（二段がけ）。「○割の利益」「定価」「○割引き」の言葉が見分け方。4割の利益＝×1.4、1割引き＝×0.9を順にかける。利益＝売値−原価。
①x原価を x 円とする。
原価 x 円 ↓ 4割の利益（×1.4） 定価 1.4x 円 ↓ 1割引き（×0.9） 売値 1.4x×0.9 ＝ 1.26x 円 利益＝売値−原価 ＝ 1.26x−x ＝ 0.26x ＝ 260
③式売値＝1.4x×0.9＝1.26x。利益＝売値−原価だから 1.26x−x＝260 → 0.26x＝260。
④解x＝260÷0.26＝1000。
⑤検算原価1000円。定価1000×1.4＝1400円。売値1400×0.9＝1260円。利益1260−1000＝260円○。
1割引きを「定価から1割の利益」と取りちがえない（引くから×0.9）。利益は売値−原価であって定価−原価ではない。割引き後の値段で考える。
「×(1＋利益)してから×(1−割引)」の二段がけは大問3(4)の上位問題。順番にかけ算で図にすると迷わない。
こたえ原価 1000円

第5問 ★L4 応用目安 5分
第5問定価2000円の商品を、ある割合だけ値引きして売ったところ、売値は1700円になりました。何割引きで売りましたか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。
これが新型＝割引率そのものを x に立てる。今までは金額（原価・道のり）を x にしたが、今回は「何割引きか」が未知数。x 割引き＝割引率は x/10、売値＝定価×(1−x/10)。
①xx 割引きで売ったとする（x が割引きの「割」の数）。割合になおすと x/10。
定価 2000円 ↓ x 割引き（×(1−x/10)） 売値 2000×(1−x/10) ＝ 1700円
③式売値＝定価×(1−割引率)だから 2000×(1−x/10)＝1700 → 2000−200x＝1700。
④解2000−1700＝200x → 300＝200x → x＝1.5。つまり1.5割引き（＝15％引き）。
⑤検算1.5割引き＝0.85倍。2000×0.85＝1700円○。
2000−x＝1700 としない（x は割合であって金額ではない）。値引き「額」を x にしてもいいが、今回は「何割引き」と聞かれているので割合 x/10で立てる。1割＝0.1＝×0.9を忘れずに。
「割引率そのものを文字に置く」は大問3(4)の発展。金額を x にするか割合を x にするか、聞かれているものに合わせて選ぶのが上級の判断。
こたえ1.5割引き（＝15％引き）

第6問 ★L4 応用（ひっかけ注意）目安 5分
第6問鉛筆を子どもたちに配ります。1人に6本ずつ配ると、ちょうど配り切れて余りはありませんでした。鉛筆は2本ずつ束ねた束が27束あります。子どもの人数は何人ですか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。※よく読もう。ひっかけがあるよ。
個数・代金型（素直な1元式）。「配り切れる・余りなし」と過不足っぽい言葉が出るが、配り方は1通りだけ。鉛筆の全体量が「2本×27束」で別の言い方で分かるから、＝で結ぶ式は1つでよい。
①x子どもの人数を x 人とする。
鉛筆の全体 2本×27束 ＝ 54本 配り方 1人6本ずつ ＝ 6x 本（ちょうど） よって 6x ＝ 54
③式鉛筆の全体＝2×27＝54本。1人6本ずつでちょうど配り切れるから 6x＝54。
④解x＝54÷6＝9。
⑤検算子ども9人。6×9＝54本＝2×27束で全部配り切れる○。
過不足ワードで反射的に2式を作らない。「配り切れる・余りなし」でも配り方が1通りなら過不足の2式は不要。全体量が別の言い方で分かれば素直な1元式でよい。
「束・ダース・○本ずつ」で全体量が分かるときは、まず全体を1つの数になおす。過不足型か素直な代金型かを見抜くのが大問3(1)(3)の分かれ道。
こたえ子ども 9人
