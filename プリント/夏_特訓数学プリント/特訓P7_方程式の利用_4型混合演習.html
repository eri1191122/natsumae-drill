方程式の利用｜4つの型を混ぜて演習（個数・速さ・割合・比／動点）

夏・特訓対策　目安：約30分（問1〜問6・通し番号）　対象: 中1　全問に徹底解説つき

P4〜P6で型カードを練習したら、今日は型を混ぜて出す（インターリーブ）回。本番は「これは何型？」を自分で見抜くことが大事。塾の夏期講習で習う4型の総しあげ。まず「何型か」を考えてから、型カードにそって式を作ろう。

  1. 個数・代金型：合計が決まる → 片方 x、もう片方 (合計−x)
  2. 速さ型：道のり＝速さ×時間。追いつき＝道のり等しい／出会い＝合計
  3. 割合・利益型：割引き×(1−)／利益×(1+)／「の」はかけ算
  4. 比・動点型：比は a を1つぶんとして 5a・3a／動点は長さを x で表す
問1ある商品を定価の3割引きで売ったら、売値は1400円でした。定価はいくらですか。
まず何型か考え、x を決めて式をノートに書こう。
割合・利益型（割引き）。「3割引き＝×(1−0.3)＝×0.7」。
式定価を x 円。0.7x＝1400。
解x＝1400÷0.7＝2000。検算 2000×0.7＝1400○。
3割引きを ×0.3 としない（残り×0.7）。
こたえ定価 2000円

問2弟が分速50mで家を出発しました。その10分後に兄が分速70mで追いかけました。兄が出発してから何分後に弟に追いつきますか。
まず何型か考え、線分図と式をノートに書こう。
速さ型（追いつき）。追いつく＝2人の道のりが等しい。弟は兄より10分多く歩いている。
弟：50×(x＋10) ───────────→ 兄：70×x ──────────→（追いつく＝同じ道のり）
式兄が出発してから x 分後。70x＝50(x＋10)。
解70x＝50x＋500 → 20x＝500 → x＝25。検算 兄70×25＝1750、弟50×35＝1750○。
弟の時間を x にしない（弟は x＋10分）。
こたえ25分後

問31個90円のなしと、1個60円のみかんを合わせて12個買ったら、代金は870円でした。なしは何個買いましたか。
まず何型か考え、表にして式をノートに書こう。
個数・代金型（合計が決まる）。なしを x 個 → みかんは (12−x) 個。
式90x＋60(12−x)＝870。
解90x＋720−60x＝870 → 30x＝150 → x＝5。検算 450＋420＝870○。
60(12−x) を 720−x としない（720−60x）。
こたえなし 5個

問4兄と弟の所持金の比は 5：3 で、2人の合計は4000円です。兄の所持金はいくらですか。
まず何型か考え、「1つぶん」を文字にしてノートに式を書こう。
比型。比 5：3 は「1つぶん a」を使って兄＝5a、弟＝3a と表す。
式合計＝5a＋3a＝4000。
解8a＝4000 → a＝500。兄＝5a＝5×500＝2500。検算 弟3×500＝1500、合計4000○。
兄を5000円のように勝手に置かない（比は「何倍か」だけ。1つぶんを a に）。最後に「兄」を答える（a で止めない）。
こたえ兄 2500円

問5現在、父は45歳、子は15歳です。父の年齢が子の年齢のちょうど2倍になるのは何年後ですか。
まず何型か考え、x 年後の2人の年齢を式にしてノートに書こう。
年齢型（過不足・等式型の仲間）。x 年後は父(45＋x)歳、子(15＋x)歳。父＝子×2。
式45＋x＝2(15＋x)。
解45＋x＝30＋2x → 45−30＝2x−x → x＝15。検算 父60＝子30×2○。
片方だけに＋x しない（父も子も同じだけ年をとる）。2倍は大きいほう（父）＝小×2。
こたえ15年後

問61辺が10cmの正方形ABCDがあります。点Pは頂点Aを出発し、辺AB上を毎秒1cmの速さでBに向かって動きます。出発してから x 秒後、三角形APDの面積が15cm²になるのは何秒後ですか。
まず何型か考え、x 秒後の AP の長さと三角形の面積を式にしよう。
動点型。動いた長さを x で表す。x 秒後 AP＝1×x＝x cm。三角形APDは底辺AP・高さAD(=10cm)。
D┌──────┐C │＼ │ │ ＼ │ AD=10 │ ＼ │ A└─P────┘B AP=x （三角形APDの面積＝AP×AD÷2）
式面積＝AP×AD÷2＝x×10÷2＝5x。これが15。5x＝15。
解x＝3。検算 AP=3、3×10÷2＝15○。
高さは AD＝10（一定）。P が動いても底辺 AP だけが x で変わる。
こたえ3秒後
