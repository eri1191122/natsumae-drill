特訓P6b数学：正負の数の利用★文章→式 L2-L3（模試0/25直撃）

文章を式にする｜正負の数の文章②を「4問・じっくり」で固める

夏・特訓対策　目安：約30分（問1〜問4・通し番号）　対象: 中1　全問に徹底解説つき

全国統一テストで0/25だった「正負の文章題」を、もう一度4問だけで固める回。コツはいつも同じ＝「基準（0）を自分で決めて、それより上か下かを＋−で書く」。数をこなすより、型カード①〜⑤にそって1問ずつていねいに書くのが今日の目標。

  1. ① 基準は何か（平均・通常水位・スタート…を 0 とする。何を 0 にしたか書く）
  2. ② ＋−で表す（上/高い/増える/収入＝＋、下/低い/減る/支出＝−）
  3. ③ 式にする（合計はたし算、差は「大−小」、平均は「基準＋差の平均」）
  4. ④ 解く（＋と−の計算をていねいに）
  5. ⑤ 検算（出た数を文章に戻して合うか確かめる）
問1気温の上下（基準との差・平均）L2 標準目安 7分
問1ある週の最高気温を、月曜の18℃を基準（0）にして、それより高ければ＋・低ければ−で表すと、下の表のようになりました。月〜金の5日間の最高気温の平均は何℃ですか。
曜日月火水木金
基準18℃との差0＋3−1＋4−1
まず「何型か」を考えよう。何を x にするかは自分で決めて、図か表をかいてから式を作ろう。
平均は「基準＋（差の平均）」で速く出せる。全部の気温を足さなくても、基準18℃からの差だけ平均すればよい。差の合計÷日数＝差の平均。
①基準月曜の18℃を基準（0）とする。表は「18℃より何℃高い/低いか」。
②＋−高い＝＋、低い＝−。差は 0, ＋3, −1, ＋4, −1。
③式差の合計＝0＋3−1＋4−1＝＋5。差の平均＝5÷5＝＋1。平均＝18＋1＝19。
⑤検算実際の気温は 18, 21, 17, 22, 17。合計＝18＋21＋17＋22＋17＝95。95÷5＝19。基準で出した19℃と一致○。
差の合計を「日数」で割る（÷5）。最後に基準18を足し忘れない（差の平均＋1だけで終わらない）。
「基準＋差の平均」は特訓選抜 大問2の正負の表・平均の超定番。点数・重さ・身長でも同じやり方が使える。
こたえ平均 19℃

問2水位の上下（基準0からの差）L2 標準目安 7分
問2ある池の水位を、いつもの水位を基準（0）にして、高ければ＋・低ければ−で表します。大雨の日はいちばん高くて＋12cm、かんそうした日はいちばん低くて−9cmでした。いちばん高いときと、いちばん低いときの水位の差は何cmですか。
まず「何型か」を考えよう。何を x にするかは自分で決めて、図か表をかいてから式を作ろう。
差は「大きいほう−小さいほう」で出す。＋と−のへだたりは、数直線で見ると0をまたぐぶんだけ大きくなる。−を引くと、足し算に変わる。
①基準いつもの水位を0とする。高い＝＋、低い＝−。
②＋−いちばん高い＝＋12、いちばん低い＝−9。
③式・④解差＝大−小＝12−(−9)＝12＋9＝21。
⑤検算数直線で＋12から−9まで。0より上に12、0より下に9、合わせて21cm離れている○。
差を 12−9＝3 としない。−9を引く＝＋9。0をまたぐから、上の12と下の9を「足したぶん」が差になる。
「差＝大−小（−を引くと足す）」は気温・標高・点数の差でも全部同じ。正負の数のいちばんの土台。
こたえ21cm

問3★収支（収入＋・支出−の合計）L3 応用目安 8分
問3きみのおこづかい帳を、お金が入るのを＋・使うのを−で表します。1週間の記録は、月曜＋500円（おこづかい）、火曜−300円、水曜−150円、木曜＋200円（お年玉の残り）、金曜−450円でした。この1週間で、お金は増えましたか減りましたか。また、いくら増えた（減った）か求めなさい。
まず「何型か」を考えよう。何を x にするかは自分で決めて、図か表をかいてから式を作ろう。
収支の合計は「＋どうしの合計」と「−どうしの合計」を先に出して、最後に1回だけ引くとミスが減る。答えの符号が＋なら増えた、−なら減った。
①基準「使わず増えも減りもしない」を0とする。入る＝＋、使う＝−。
②＋−＋500, −300, −150, ＋200, −450。
③式・④解＋どうし＝500＋200＝＋700。−どうし＝300＋150＋450＝900だから−900。合計＝700−900＝−200。
⑤検算500−300−150＋200−450を左から：500−300＝200、200−150＝50、50＋200＝250、250−450＝−200○。符号が−だから減った。
符号を落として 500＋300＋150＋200＋450 と全部足さない。使ったお金は−。入った分と使った分を分けてから引く。
収入＋・支出−は中2の「正負の利用」「一次関数の増減」にもつながる、お金・温度・人口の増減を読む基本の型。

問4★仮平均（基準を決めて差の平均から本当の平均）L3 応用目安 8分
問4下の表は、7人の身長を150cmを基準にして、高ければ＋・低ければ−で表したものです。7人の身長の平均を求めなさい。
人ABCDEFG
基準150cmとの差＋4−2＋6−3＋1＋5−4
まず「何型か」を考えよう。何を x にするかは自分で決めて、図か表をかいてから式を作ろう。
仮平均：平均＝基準＋（差の平均）。154や148のような大きい数を全部足さなくても、基準150からの差（小さい数）だけ平均すれば、速く正確に出せる。
①基準150cmを基準（0）とする。高い＝＋、低い＝−。
②＋−差は ＋4, −2, ＋6, −3, ＋1, ＋5, −4。
③式・④解差の合計＝4−2＋6−3＋1＋5−4＝＋7。差の平均＝7÷7＝＋1。平均＝150＋1＝151。
⑤検算実際の身長は 154,148,156,147,151,155,146。合計＝1057、1057÷7＝151○。差で出した151と一致。
差の合計を「人数」で割る（÷7）。最後に基準150を足し忘れない（差の平均＋1だけで答えにしない）。
仮平均は特訓選抜 大問2の正負の表・平均で必ず出る型。テストの平均点・体重・タイムなど、大きい数の平均を速く出す入試必須テク。
