文章を式にする｜割合・食塩水の型を「4問・じっくり」で固める

夏・特訓対策　目安：約30分（問1〜問4・通し番号）　対象: 中1　全問に徹底解説つき

これがきみのいちばんの伸びしろ＝「文章を式にする」力。全国統一テストで0/25だったのもここ。今日は4問だけ。割合のいのちは「もとにする量×割合＝くらべる量」。これ1つを、割引・値上げ・食塩水・人数の4つの場面で使えるようにするのが今日の目標。数をこなすより型を体に入れる。

  1. ① 何を求める？ → それを x とおく（「□を x とする」と書く）
  2. ② 図か表にする（割合は線分図・食塩水は表）＝目で見える形に
  3. ③ 等しい関係を式に（「もとにする量×割合＝くらべる量」で＝を作る）
  4. ④ 解く（移項して x ＝ の形へ）
  5. ⑤ 検算（出た数を文章に戻して合うか確かめる）
問1割引（割引後の代金から定価を求める）L3目安 7分
問1ある品物を定価の3割引で買ったら、代金は1750円でした。この品物の定価を求めなさい。
求めるもの（定価）を x にする。「3割引で買う」＝「定価の何倍をはらった？」を先に決めてから式を作ろう。
割合のいのち：もとにする量×割合＝くらべる量。割引ははらうのは「残り」。3割引で買う＝定価の(1−0.3)＝0.7倍をはらう（3割＝0.3）。
①x定価を x 円とする（求めるもの）。
定価 x 円 ───────────────── ↑3割(0.3x)引く↑ 残り0.7x＝はらった額(1750円)
③式はらった代金＝定価×0.7。0.7x＝1750（3割引＝×(1−0.3)＝×0.7）。
④解x＝1750÷0.7＝2500。
⑤検算定価2500円の3割引：2500×0.7＝1750○。引いた額は2500×0.3＝750円で、2500−750＝1750もOK。
3割引を ×0.3 としない。はらうのは引いた残り＝×0.7。「割」を％とまぜない（3割＝0.3、3%＝0.03）。
「割引＝×(1−割合)」は特訓選抜 大問3（割合の方程式）の超定番。値引き・セール・利益はすべてこの形でつながる。

問2値上げ（○%増しで今の値段になった）L3目安 7分
問2あるバスの運賃が、去年より15%値上げされて、今年は230円になりました。去年の運賃は何円でしたか。
求めるもの（去年の運賃）を x にする。値上げは「もとより増える」。何倍になったかを先に決めよう。
値上げ（増加）はもとに足す。15%値上げ＝もとの(1＋0.15)＝1.15倍（15%＝0.15）。式は「もとにする量×割合＝くらべる量」のまま。
①x去年の運賃を x 円とする（求めるもの＝もとにする量）。
去年 x 円 ────────→ 今年 230円 ＋15%(0.15x)増し → 1.15x＝230
③式今年の運賃＝去年×1.15。1.15x＝230（15%増し＝×(1＋0.15)）。
④解x＝230÷1.15＝200。
⑤検算去年200円の15%増し：200×1.15＝230○。増えた額は200×0.15＝30円で、200＋30＝230もOK。
15%を 1.5 や 0.015 にしない（15%＝0.15）。値上げは×1.15（足す）、値下げ・割引は×0.85など（引く）。増えるのか減るのかを必ず先に確認。
「○%増し＝×(1＋割合)」は値上げ・利益・人口増などに共通。割引の×(1−割合)と＋か−かだけがちがう。

問3★食塩水（水を加えて濃度を下げる）L3 応用目安 8分
問3濃度10%の食塩水が200gあります。これに水を何gか加えたら、濃度が8%になりました。加えた水は何gですか。
食塩水は表で整理。「水を加えても食塩の重さは変わらない」がカギ。食塩の重さを2通りで表して＝で結ぼう。
食塩水のいのち：食塩＝食塩水×濃度（割合の式と同じ）。水を加えても変わらないのは「食塩の重さ」。加える前の食塩＝加えた後の食塩、で＝を作る。
①x加える水を x g とする。加えた後の食塩水は (200＋x) g。
┌──────────┬──────────┬──────────┐ │ │ 加える前 │ 加えた後 │ ├──────────┼──────────┼──────────┤ │ 食塩水(g) │ 200 │ 200＋x │ │ 濃度 │ 10%(0.1) │ 8%(0.08) │ │ 食塩(g) │ 200×0.1 │(200＋x)×0.08│ └──────────┴──────────┴──────────┘ ※水だけ足す＝食塩はそのまま（左の食塩＝右の食塩）
③式食塩の重さは変わらない：(200＋x)×0.08＝200×0.1（8%＝0.08、10%＝0.1）。
④解右辺＝20。0.08(200＋x)＝20 → 16＋0.08x＝20 → 0.08x＝4 → x＝50。
⑤検算水を50g加えると食塩水は200＋50＝250g。食塩は20gのまま。20÷250＝0.08＝8%○。
8%を0.8にしない（8%＝0.08）。水を加えると食塩も増える、と思わない＝足したのは水だけ。食塩は変わらないのが等式のもと。
「変わらない量（食塩）を2通りで表して＝」は過不足算と同じ考え方。中2の食塩水（連立）にもそのままつながる。

問4人数の割合増減（昨年比○割増）L3目安 7分
問4あるクラブの今年の部員は、去年より2割増えて36人になりました。去年の部員は何人でしたか。
求めるもの（去年の部員）を x にする。「2割増えて」＝去年の何倍が今年か、を先に決めてから式を作ろう。
人数でも考え方は同じ：もとにする量（去年）×割合＝くらべる量（今年）。2割増え＝去年の(1＋0.2)＝1.2倍（2割＝0.2）。
①x去年の部員を x 人とする（もとにする量）。
去年 x 人 ────────→ 今年 36人 ＋2割(0.2x)増し → 1.2x＝36
③式今年の人数＝去年×1.2。1.2x＝36（2割増し＝×(1＋0.2)）。
④解x＝36÷1.2＝30。
⑤検算去年30人の2割増し：30×1.2＝36○。増えた人数は30×0.2＝6人で、30＋6＝36もOK。
36人を「去年」と思わない＝36は今年（増えた後）。もとにする量（去年）が x。2割を0.02にしない（2割＝0.2、2%＝0.02）。
人口・売上・部員の「○割増・○%減」は割引・値上げとまったく同じ型。場面がちがっても「もとにする量×割合＝くらべる量」1つで解ける。
