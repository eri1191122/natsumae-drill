問1〜3素数の判定 L1-L2 基礎目安 7分
① 素数（そすう）とは

1とその数自身でしか割り切れない、1より大きい整数のこと。
たとえば7は、7÷1＝7、7÷7＝1 でしか割り切れない。だから7は素数。

② 素因数（そいんすう）とは

ある整数の約数のうち、素数であるものを、その数の素因数という。

③ 素因数分解（そいんすうぶんかい）とは

整数を素数だけのかけ算の形に表すこと。「これ以上バラせない部品（素数）に分解する」というイメージ。

④ 指数（しすう）の書き方

同じ数を何回かけたかを右上に小さく書く。2×2×2＝23、3×3＝32。素因数分解の答えは、この指数の形まで書くのがルール。

【例】39は素数ですか。理由もつけて答えなさい。
手順1小さい素数から順にわってみる。2, 3, 5, 7, …の順。
手順22でわれる？　39は奇数だからわれない。
3でわれる？　39÷3＝13　→ われた！
手順3わり切れたということは、39の約数に 3 と 13 がある。1と39以外の約数があるので素数ではない。
どこまで試す？「わる数×わる数」がもとの数をこえたら、そこでやめてよい。39なら 7×7＝49＞39 なので、7まで試して1つもわれなければ素数と決まる。
こたえ　素数ではない（39＝3×13 とわけられるから）
★素数の判定
問11から20までの整数のうち、素数を小さい順に全部書きなさい。また、それは何個ですか。
まず1から20まで書き出してみよう。手を動かしてから答え合わせをしよう。
考え方1から20を書き出して、素数の定義（1とその数自身でしかわれない・1より大きい）に合うものだけ残す。
ふるい2の倍数（4,6,8,10,12,14,16,18,20）は2でわれるので消える。3の倍数（9,15）も消える。残った15は3でわれるのでこれも消える。1は1より大きくないので最初から入らない。
残るもの2, 3, 5, 7, 11, 13, 17, 19　→ 8個。
1を入れない（1は素数ではない。約数が1しかなく「1とその数自身」が同じになってしまう）。9を入れない（9＝3×3）。15を入れない（15＝3×5）。
20までの素数8個は、これから素因数分解でずっと使う道具。声に出して「に・さん・ご・なな・じゅういち・じゅうさん・じゅうなな・じゅうきゅう」と言えるようにしておくと、計算が速くなる。
こたえ2, 3, 5, 7, 11, 13, 17, 19　／　全部で 8個

問2次の4つの数のうち、素数はどれですか。素数でないものは、素数でない理由（何でわれるか）も書きなさい。
51　　57　　61　　91
1つずつノートで確かめよう。手を動かしてから解説を開こう。
51各位の和が 5＋1＝6 で3の倍数 → 3でわれる。51＝3×17。素数ではない。
57各位の和が 5＋7＝12 で3の倍数 → 3でわれる。57＝3×19。素数ではない。
612×（奇数）でない、3の倍数でない（6＋1＝7）、5でも7でもわれない。8×8＝64＞61 なので7まで確かめれば十分。素数。
912・3・5ではわれないが 91÷7＝13。91＝7×13。素数ではない。
なぜ7でやめていい？もし61が8以上の数でわれるなら、かけて61になる相方は8より小さい数になる（大きい×大きい では61をこえてしまうから）。
その小さい相方は 2・3・5・7 のどれかのはず。でもそこは全部ためして、われなかった。
だから8以上でもわれない。7で止めてよい。
91を素数とまちがえる人がとても多い（＝7×13）。2・3・5でわれないと「もう素数だ」と決めてしまうのがワナ。7も必ず試すこと。
「51・57・91は素数っぽく見えるけど素数ではない」は入試の定番ひっかけ。3の倍数の見分け方（各位の和が3の倍数）はここで武器になる。
こたえ素数は 61　／　51＝3×17、57＝3×19、91＝7×13

問3ゆいさんは「2は偶数だから素数ではない」と言いました。また「1は1でしかわれないから素数だ」とも言いました。この2つは正しいですか。正しくないものは、理由をつけて直しなさい。
定義（ことばの意味）にもどって、自分の言葉で書いてみよう。
2について素数の決まりは「1とその数自身でしかわれない」だけ。偶数か奇数かは関係ない。2の約数は1と2だけ → 2は素数。
ただし2以外の偶数は必ず2でわれてしまう。だから偶数の素数は2ただ1つ。「2だけが特別」と覚えるのではなく、定義から確かめられるようにしよう。
1について素数は「1より大きい整数」。1はそもそも仲間に入れない。1は「1とその数自身」が同じ数になってしまい、約数が1個しかないから、素数ではない。
「偶数＝素数ではない」ではない（2は素数）。「1は素数」ではない。この2つは特訓選抜の正誤問題で毎年ねらわれる。
素数かどうかは、偶数・奇数で決めない。定義（1とその数自身でしかわれない・1より大きい）で決める。
こたえどちらも正しくない。2は素数（偶数の素数は2だけ）。1は素数ではない（素数は1より大きい整数）。

問4〜5素因数分解（指数の形まで書く）L2 標準目安 6分
【例】60を素因数分解しなさい。
手順1小さい素数から順にわっていく。わり切れなくなったら次の素数へ。
2 ) 60 2 ) 30 3 ) 15 5 ← 5は素数。ここで止める たてに並んだ数をかける → 2×2×3×5
手順260÷2＝30、30÷2＝15、15は2でわれないので3へ。15÷3＝5、5は素数なので終わり。
手順3わった素数とさいごに残った素数をすべてかける。60＝2×2×3×5。
手順4同じ数は指数の形にまとめる。2が2個だから 22。
こたえ　60＝22×3×5
問484を素因数分解しなさい。（指数の形まで書くこと）
ノートにすだれ算を書いてから答え合わせをしよう。
2 ) 84 2 ) 42 3 ) 21 7
わる84÷2＝42、42÷2＝21、21は2でわれないので3へ。21÷3＝7、7は素数で終わり。
まとめる84＝2×2×3×7＝22×3×7。
検算22×3×7＝4×21＝84○。かけ算してもとの数にもどるか必ず確かめる。
84＝4×21 で止めない（4も21も素数ではない。まだバラせる）。指数を書き忘れない（2×2×3×7 のままより 22×3×7 が正式な答え）。
コツ2でわれる＝一の位が偶数。3でわれる＝各位の和が3の倍数（84 は 8＋4＝12 だから3でわれる）。5でわれる＝一の位が0か5。
わる前にこの3つを見るだけで、すだれ算の速さが変わる。
こたえ84＝22×3×7

問5360を素因数分解しなさい。（指数の形まで書くこと）
ノートにすだれ算を書いてから答え合わせをしよう。
2 ) 360 2 ) 180 2 ) 90 3 ) 45 3 ) 15 5
わる2で3回われる（360→180→90）。90は2でわれるが45は奇数なのでここで3へ。3で2回われる（45→15→5）。5は素数で終わり。
まとめる360＝2×2×2×3×3×5＝23×32×5。
検算23×32×5＝8×9×5＝360○。
とちゅうでわるのをやめない。「もうこれ以上小さい素数でわれない」ところまで続ける。2で何回わったかを数えまちがえない（メモを縦にそろえて書くとまちがえにくい）。
素因数分解は、このあとの最大公約数・最小公倍数・約数の個数・平方根の全部の土台になる。中3の「√」もここから始まる。
コツ2でわれる＝一の位が偶数。3でわれる＝各位の和が3の倍数（360 は 3＋6＋0＝9 だから3でわれる）。5でわれる＝一の位が0か5。
わる前にこの3つを見るだけで、すだれ算の速さが変わる。
こたえ360＝23×32×5

問6〜7約数の個数（素因数分解から求める）L3 応用目安 8分
【例】72の約数は全部で何個ありますか。
手順1まず素因数分解する。72＝23×32。
手順272の約数は「2を何個か、3を何個か、かけ合わせた数」でできている。
2は 0個・1個・2個・3個 の 4通り（0個＝使わない、も1通り）。
3は 0個・1個・2個 の 3通り。
手順32の選び方と3の選び方は自由に組み合わせられるから 4×3＝12通り。組み合わせ1つが約数1個に対応する。
3を0個 3を1個 3を2個 2を0個 → 1 3 9 2を1個 → 2 6 18 2を2個 → 4 12 36 2を3個 → 8 24 72 計 12個
手順4つまり指数に1をたしてかける。(3＋1)×(2＋1)＝4×3＝12。「1をたす」のは0個のときも数えるから。
こたえ　12個（1,2,3,4,6,8,9,12,18,24,36,72）

問8〜9平方数にする（★首都圏オープンで落とした型）L3-L4 応用目安 9分
【例】72にできるだけ小さい自然数をかけて、ある自然数の2乗にしたい。どんな数をかければよいですか。また、そのときの2乗の数も答えなさい。
そもそも「ある自然数の2乗」＝平方数。36＝62、144＝122 のような数。
平方数を素因数分解すると、36＝22×32、144＝24×32 のように指数が全部偶数になる。同じ部品を2つずつペアにできるからだ。
手順1素因数分解する。72＝23×32。
手順2指数を見る。3の指数は2（偶数・OK）。2の指数が3で奇数＝ここだけペアが作れず1個あまっている。
手順3あまっている2を1個おぎなえばよい。だから2をかける。
72×2＝24×32 → 指数が全部偶数になった。
手順42乗の数は、むずかしく考えずまず計算する。72×2＝144。
つぎに「何を2回かけたら144？」をさがす。10×10＝100、20×20＝400 → 144はその間だから、答えは10と20の間。
12×12＝144○　だから 144＝122。
こたえ　2をかける　／　そのとき 144＝122
問854にできるだけ小さい自然数をかけて、ある自然数の2乗にしたい。かける数を求めなさい。また、そのときの2乗の数も答えなさい。
ノートに手を動かしてから答え合わせをしよう。
素因数分解54＝2×27＝2×33。（検算：2×27＝54○）
指数を見る2の指数は1（奇数）、3の指数は3（奇数）。両方とも奇数なので、どちらも1個ずつおぎなう必要がある。
かける数2×3＝6。54×6＝22×34 → 指数が全部偶数。
2乗の数むずかしく考えず、まず 54×6＝324 を計算する。
つぎに「何を2回かけたら324？」をさがす。10×10＝100、20×20＝400 → 324はその間だから、答えは10と20の間。
18×18＝324○　だから 324＝182。
33 だから3を3個かける、としない。足りないのはペアにあぶれた1個だけ（3をもう1個かければ34になる）。2を見落とさない（2の指数1も奇数）。
こたえ6をかける　／　そのとき 324＝182

問9 ★252をできるだけ小さい自然数でわって、ある自然数の2乗にしたい。わる数を求めなさい。また、そのときの2乗の数も答えなさい。
ノートに手を動かしてから答え合わせをしよう。
2 ) 252 2 ) 126 3 ) 63 3 ) 21 7
素因数分解252＝22×32×7。（検算：4×9×7＝252○）
指数を見る2の指数2（偶数・OK）、3の指数2（偶数・OK）、7の指数だけ1で奇数。
今回は「わる」かけるときは足りない分をおぎなった。わるときは、あまっている分を取り除く。あまっているのは7が1個だから 7でわる。
2乗の数252÷7＝22×32＝(2×3)2＝62＝36○。
かける問題もわる問題も、見るところは同じ＝素因数分解して「指数が奇数の素数」をさがす。
かける＝足りない1個をおぎなう／わる＝あまった1個を取り除く。この対で覚えると、どちらを聞かれても迷わない。
かける問題と同じ答え（7をかける）にしない。聞かれているのは「わる数」。252÷7＝36 で終わらない。「36＝62」と2乗の形まで答える。
この「指数を偶数にそろえる」考え方は、中3の平方根（√の中を簡単にする・√が整数になる条件）でそのまま使う。今ここで身につけると3年分ずっと効く。
こたえ7でわる　／　そのとき 36＝62
