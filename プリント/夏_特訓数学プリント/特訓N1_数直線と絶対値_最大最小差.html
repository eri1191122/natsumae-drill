大問1前にやった問題。今日はどう？｜0からのきょり　L2目安 3分
問1次の5つの数のうち、0に2番目に近い数はどれですか。
−1.6 95 74 −2.1 114
まず5つの数を、ノートの数直線の上に書き出してみよう。
「0に近い」＝ 0からのきょりが小さいということ。きょりに ＋も−も関係ない。プラスかマイナスかを見るのではなく、符号を外した大きさ（＝絶対値）だけで比べる。
①まず全部小数にそろえる。95 ＝ 9÷5 ＝ 1.8　／　74 ＝ 7÷4 ＝ 1.75　／　114 ＝ 11÷4 ＝ 2.75
②0からのきょり（符号を外す）を並べる。
−1.6 → 1.6　／　1.8 → 1.8　／　1.75 → 1.75　／　−2.1 → 2.1　／　2.75 → 2.75
③きょりの小さい順にならべる：1.6 → 1.75 → 1.8 → 2.1 → 2.75
1番近い＝ −1.6（きょり1.6）。2番目に近い＝きょり1.75 → 74
「0に近い」をマイナスだから遠いと思いこまない。−1.6 は 0 から 1.6 しか離れていない＝かなり近い。符号を外したきょりで比べるのがルール。
もうひとつ。74 と 95 を見た目で決めない。分母がちがう分数は、小数に直してからくらべる。1.75 と 1.8 は0.05しか差がない。目分量では絶対に当たらない。
出した順番を、数直線の上で目で確かめる。0の右にも左にも同じきょりで手をのばす感じ。−1.6 が 0 にいちばん近く、そのすぐ次に 74 がいる。図と答えが一致していれば正解。

問2【例】次の問いに答えなさい。
(1)−5 の絶対値は 　　
(2)3 の絶対値は

絶対値 ＝ 0からのきょり。これだけ。むずかしそうな名前だけど、やっていることは「0から何メートル離れてる？」と聞いているだけ。
きょりに マイナスは無い。家から学校まで「−5分」ということが無いのと同じ。きょりは必ず 0 か プラス。
(1) −5 は、0 から左に 5 だけ離れている。だから −5 の絶対値 ＝ 5。記号で書くと |−5| ＝ 5。マイナスを外すだけと覚えてもいい。
(2) 3 は、0 から右に 3 だけ離れている。だから |3| ＝ 3。もともとプラスの数は、そのまま。
(3) ここが山。「0からのきょりが 4 の数」を探す。0 の右にも、0 の左にも、きょり 4 の場所がある。
右に4進むと ＋4。左に4進むと −4。だから答えは ＋4 と −4 の 2つ。
★ここが命。(3) で ＋4 だけ書いて終わらない。絶対値は 0 を中心に左右に同じだけ広がる。だから「絶対値が◯になる数」は、0 のとき以外はいつも2つ。
検算：|＋4| ＝ 4 ○、|−4| ＝ 4 ○。どちらも 4 になった。合っている。
読んだら、同じ数直線をノートに写そう。写しながら「絶対値は0からのきょり」と声に出して。
聞かれているのは 「その数は、0からどれだけ離れているか」 だけ。数の向き（＋か−か）は捨てて、長さだけを見る。だから答えはいつも 0 か プラスになる。
(1)−5 は 0 から左へ 5。きょりは 5 → |−5| ＝ 5
(2)3 は 0 から右へ 3。きょりは 3 → |3| ＝ 3
(3)きょり 4 の場所は2か所（0の右に4、0の左に4）→ ＋4 と −4
(3) で ＋4 だけを答えてしまう。これが最多。絶対値は 0 を中心に左右へ同じだけ広がる。「すべて書きなさい」と言われたら、左側（マイナス）も必ず探す。
もうひとつ。絶対値の答えにマイナスをつけてしまう（|−5| ＝ −5 とする）。きょりにマイナスは無い。「学校まで −5 分」なんて言わない。絶対値の答えが マイナスになったら、その時点でまちがい。
出した答えを、もう一度絶対値に入れ直す。|＋4| ＝ 4 ○、|−4| ＝ 4 ○。両方 4 になった。
そして答えが 0 以上かを最後に見る。(1)は5、(2)は3。どちらもプラス○

(1)「小さい順」＝ 数直線で左にあるものから順。数直線は右へ行くほど大きい。マイナスの世界では、0から遠いほうが小さい（−3 は −1 より小さい）。

◆ ①〜(1)
①分数を小数に直す。数直線に置くには、小数のほうが位置がすぐ分かる。
−95 ＝ −(9÷5) ＝ −1.8　／　−12 ＝ −(1÷2) ＝ −0.5
②5つを小数でならべ直す：−1.8　0.5　−3　2　−0.5
③数直線に置く。左にあるほど小さいので、左から読む。
−3 ＜ −1.8 ＜ −0.5 ＜ 0.5 ＜ 2
(1)小数を、もとの形にもどして答える。
−3、−95、−12、0.5、2
◆ だいじ〜(2)

−3 は、左右で見れば いちばん左（＝いちばん小さい）。きょりで見れば 0から3もはなれている（＝いちばん遠い）。
同じ数が、ものさしを変えると「いちばん小さい」と「いちばん大きい」の両方になる。
(2)0からのきょり（絶対値）を出す：3　1.8　0.5　0.5　2
いちばん大きいきょりは 3 → その数は −3

問4 ★次の問いに答えなさい。
(1)−7 と 5 では、絶対値が大きいのはどちらですか　　　
(2)−7 と 5 では、数として小さいのはどちらですか　　　
(3)−8、3、−5、6 を絶対値の大きい順に並べなさい　　　
数直線をノートにかいて、それぞれの数を置いてから答えよう。
この問題は 「2つのちがう質問」をわざと並べている。
・数として大きい・小さい＝ 数直線の左右で決まる（右が大きい）
・絶対値が大きい・小さい＝ 0からのきょりで決まる（符号は無視）
この2つはまったく別のものさし。どちらを聞かれているかを、毎回たしかめる。
(1)0からのきょりを出す：−7 → 7、5 → 5。7 ＞ 5 だから 絶対値が大きいのは −7
(2)数直線の左右で見る：−7 は 0 の左、5 は 0 の右。左にあるほど小さいので 小さいのは −7
(3)まずきょりだけ書き出す：−8 → 8、3 → 3、−5 → 5、6 → 6
きょりの大きい順：8 ＞ 6 ＞ 5 ＞ 3 → もとの数にもどして −8、6、−5、3
−7 と 5 では、数としては −7 が小さい。でも 0 からのきょりは −7 のほうが大きい。聞かれているのがどっちかを、毎回たしかめる。ここを混ぜると、正しく計算しても答えが逆になる。
直しかた：問題文の「絶対値」という言葉に丸をつける。丸がついていたら符号を消して、きょりだけで勝負。丸がなければ数直線の左右。
もうひとつ。(3) で −8 を「いちばん小さいから最後」に置いてしまう。聞かれているのは絶対値の大きい順。−8 のきょりは 8 でいちばん大きい。だから先頭。
(3) は、並べた順にきょりを書き添えて読み返す。−8(8)、6(6)、−5(5)、3(3)。8 → 6 → 5 → 3 とちゃんと下がっている○　もし上がっている場所があれば、そこがまちがい。
(1)(2) は、答えが両方 −7 になってもあわてない。理由がちがうだけで、たまたま同じ数になることはある。
9月の大問1では「絶対値が大きいのはどちらか」「大小を不等号で表せ」の形で出る。ものさしが2本あると知っているだけで、確実に取れる。
こたえ(1) −7　(2) −7　(3) −8、6、−5、3

問5 ★次の整数をすべて書き、個数も答えなさい。
(1)絶対値が3以下の整数　　　（　個）
(2)絶対値が2以上4以下の整数　　　（　個）
まず数直線をノートにかいて、あてはまる数に●をつけてから書き出そう。
「絶対値が3以下」＝ 0から3までしか離れていない数。0の右にも左にも3ずつ手をのばした、その範囲の中の整数。
「絶対値が2以上4以下」＝ 0から2〜4だけ離れている数。0のすぐ近く（きょり0や1）は入らない。ドーナツみたいに、まん中に穴があく。
(1)-①0からのきょりが 0、1、2、3 の整数を、左右そろえて書き出す。
きょり3 → −3、3／きょり2 → −2、2／きょり1 → −1、1／きょり0 → 0
(1)-②小さい順に並べ直す：−3、−2、−1、0、1、2、3 → 7個
0 を必ず入れる。0 は「0からのきょりが 0」だから、3以下の仲間。
(2)-①きょりが 2、3、4 の整数を、左右そろえて書き出す。
きょり4 → −4、4／きょり3 → −3、3／きょり2 → −2、2
(2)-②小さい順に並べ直す：−4、−3、−2、2、3、4 → 6個
今回は 0、1、−1 は入らない。きょりが 2 より小さいから。まん中に穴があくのがこの形の特ちょう。
(1)で 0 を書き忘れる。これがいちばん多い。テストでも首都圏オープンでも、ここで✕がついた。0 は 0 からのきょりが 0 だから、ちゃんと入る。0は「プラスでもマイナスでもないから仲間外れ」ではない。きょり0 ＝ りっぱに「3以下」。
マイナス側を書き忘れるのも同じくらい多い。1、2、3 だけ書いて終わると、7個のうち3個しか取れない。絶対値は0の左右に同じだけ広がる。
ふせぎかた：「きょり0 → きょり1 → きょり2 → きょり3」の順に、左と右をペアで書く。この書き方なら 0 もマイナスも絶対に落ちない。
(2)では逆に 0 を入れてしまう。今度は「2以上」なので、きょり0の 0 は入らない。条件を1個ずつ声に出して確かめる。
書き出した数を、1個ずつ元の条件に入れ直す。
(1) 0 → きょり0。3以下○（入る）。3 → きょり3。3以下○。4 → きょり4。3をこえる×（入れなくて正解）。
(2) 0 → きょり0。2以上ではない×（入れなくて正解）。−4 → きょり4。2以上4以下○。5 → きょり5。4をこえる×。
最後に指で個数を数える。(1)は7個、(2)は6個。数え忘れがないか必ず確認。
「絶対値が◯以下の整数をすべて」は、9月の大問1でほぼ確実に出る。首都圏オープン4(1)も、必修3(3)も、この形だった。0を入れる・左右をペアで書く。この2つだけで、落としていた点がまるごと戻ってくる。
こたえ(1) −3、−2、−1、0、1、2、3（7個）　(2) −4、−3、−2、2、3、4（6個）

📝 まちがえたら、「きょり0 → 1 → 2 → 3」の順に、左と右をペアで書き直そう（0を忘れずに）
問6次の整数をすべて書き、個数も答えなさい。
(1)−2以上4未満の整数　　　（　個）
(2)−3より大きく2以下の整数　　　（　個）
まず自分で書き出してみよう。個数も数えること。
この問題は計算ではなく、言葉のルールを聞いている。覚えることはたった2行。
「以上」「以下」＝ その数を入れる（＝がついている仲間）
「未満」「より大きい」「より小さい」＝ その数を入れない
① 前半のポイント
(1)-以上「−2以上」＝ −2 をふくむ。−2 から出発する。
(1)-未満「4未満」＝ 4 より小さい。4 はふくまない。だから 4 の一歩手前の 3 まで。
(1)−2 から 3 まで、整数を全部：−2、−1、0、1、2、3 → 6個
(2)-より大きい「−3より大きく」＝ −3 はふくまない。−3 の一歩右の −2 から出発。
② (2)-以下〜コツ
(2)-以下「2以下」＝ 2 をふくむ。2 まで。
(2)−2 から 2 まで：−2、−1、0、1、2 → 5個
「未満」を「以下」と読みちがえて 4 を入れる。未満 ＝ まだ達していない ＝ その数は入らない。この1問で入試の1点が動く。問題文の「以上・以下・未満・より大きい」に丸をつけてから解く。
そして問5と同じ罠。0 を数え忘れる。(1)も(2)も、0 はちゃんと範囲の中にいる。マイナスからプラスへ数えるときは、0 をとばしやすい。「−2、−1、0、1、2、3」と、0 を声に出しながら数える。
もうひとつ。個数を数えまちがえる。「−2 から 3 まで」を「3 − (−2) ＝ 5個」としてしまう。両はしを入れるので、5 ＋ 1 ＝ 6個。書き出して指で数えるのがいちばん確実。
書いた数を1個ずつ条件に入れ直す。
(1) −2 →「−2以上」○（入る）。3 →「4未満」○。4 →「4未満」×（入れなくて正解）。−3 →「−2以上」×。
(2) −3 →「−3より大きく」×（入れなくて正解）。2 →「2以下」○。3 →「2以下」×。
両はしの数を必ずチェックする。まん中はまず間違えない。事故はいつも はし で起きる。
コツ片方だけ入れる範囲は、引き算の答えがそのまま個数。「−2以上4未満」→ 4 − (−2) ＝ 6個。
両はしとも入れるなら ＋1。「絶対値が3以下」→ 3 − (−3) ＋ 1 ＝ 7個。
こたえ(1) −2、−1、0、1、2、3（6個）　(2) −2、−1、0、1、2（5個）

問9 ★たて 60cm、よこ 84cm の長方形の紙があります。この紙を、あまりが出ないように、できるだけ大きい正方形に切り分けます。
(1)正方形の1辺の長さは 　　 cm
(2)正方形は全部で 　　 枚できる
まず自分で考えて、ノートに書いてから答え合わせをしよう。
「あまりが出ないように切る」＝ たて60cm も よこ84cm も、正方形の1辺で ぴったりわり切れるということ。
つまり、1辺の長さは 60 の約数でもあり、84 の約数でもある数。
そのうちできるだけ大きいものを選ぶ ＝ 60 と 84 の最大公約数。
問題文の中に「最大公約数」という言葉はひとつも書いていない。「あまりが出ない」「できるだけ大きい」の2語が、それを意味している。この読みかえができるかどうかが勝負。
◆ ①〜(1)
①2つの数をかけ算の形（素因数分解）にバラす。
60 ＝ 2 × 2 × 3 × 5
84 ＝ 2 × 2 × 3 × 7
②両方に共通しているものだけを集める。
2 が2つ ＝ 共通　／　3 が1つ ＝ 共通　／　5 は60だけ、7 は84だけ → 共通ではない
共通なのは 2 × 2 × 3 ＝ 12 → 最大公約数は 12
(1)1辺の長さは 12cm
◆ (2)-①〜コツ
(2)-①たて方向に何枚ならぶか：60 ÷ 12 ＝ 5 → 5枚
よこ方向に何枚ならぶか：84 ÷ 12 ＝ 7 → 7枚
(2)-②全部の枚数は たて × よこ。5 × 7 ＝ 35 → 35枚
1辺の長さ（12cm）を答えて、枚数を出し忘れる。(2)まで聞かれている。問題は2つある。
次。最大公約数と最小公倍数を取りちがえる。「切り分ける・分ける・配る」＝ 最大公約数（もとの数より小さくなる）。「しきつめる・そろえる・同時に出発」＝ 最小公倍数（もとの数より大きくなる）。切って小さくするのだから、答えは60より小さいはず。ここで見わけがつく。
そして。枚数を「5 ＋ 7 ＝ 12枚」としてしまう。正方形はたてに5列、よこに7列ならぶ。マス目の数だから たし算ではなく、かけ算。5 × 7 ＝ 35。図をかけば一発で分かる。
もうひとつ。共通でないもの（5 や 7）まで かけてしまう。最大公約数は両方に共通しているものだけを集める。5 は 60 にしかない。7 は 84 にしかない。片方にしかないものは、使えない。
出した1辺（12cm）で、本当にわり切れるかを両方たしかめる。
60 ÷ 12 ＝ 5（あまり0）○　84 ÷ 12 ＝ 7（あまり0）○　両方わり切れた。
さらに「これより大きい数では無理」も確かめる。次に大きい共通の約数は 6。12のほうが大きい。もし 15 でためすと 84 ÷ 15 はわり切れない×　だから 12 で正しい。
枚数の検算：面積で確かめる。紙全体は 60 × 84 ＝ 5040 平方cm。正方形1枚は 12 × 12 ＝ 144 平方cm。5040 ÷ 144 ＝ 35。35枚で一致○
「あまりが出ないように、できるだけ大きく分ける」は、9月の文章題でそのまま出る形。問題文に「最大公約数」とは書いてない。書いてないものを見つけるのが、この問題のすべて。
コツすだれ算（連除法）が速い。60 と 84 を、両方わり切れる数で上から順にわる。
2）60　84　→　2）30　42　→　3）15　21　→　のこりは 5 と 7（共通なし・ここで終わり）。
左にならんだ 2 × 2 × 3 ＝ 12 が最大公約数。
