大Bかっこをふくむ（分配して解く）L2→L3 標準目安 6分・4問
かっこを分配して外してから解こう。
問5 3(x−4)＝9
問6 2(x＋5)＝4x−2
問7 4x−3(x−2)＝11

まず分配でかっこを外す→移項→まとめる→わる→検算。−のあとのかっこは中の全部の符号が変わる。
問53x−12＝9 → 3x＝21 → x＝7。検算 3×3＝9○。
問62x＋10＝4x−2 → 10＋2＝4x−2x → 12＝2x → x＝6。検算 22＝22○。

大C小数係数（両辺を10倍・100倍）L3 応用｜9月1(3)目安 6分・4問
小数はまず両辺を10倍・100倍して整数にしてから解こう。
問9 0.3x＋0.4＝1.6
問10 0.6x−0.8＝0.2x＋0.8
問11 0.15x＋0.1＝0.4
問12 1.2x−0.5＝0.7x＋2
小数は両辺をかけて整数に（1/10の位までなら×10、1/100の位があれば×100）。このとき全部の項に同じ数をかける（整数の項にもかける！）。
問9両辺×10：3x＋4＝16 → 3x＝12 → x＝4。検算 1.2＋0.4＝1.6○。
問10両辺×10：6x−8＝2x＋8 → 4x＝16 → x＝4。検算 2.4−0.8＝1.6＝0.8＋0.8○。
問11・★100倍0.15があるので両辺×100：15x＋10＝40 → 15x＝30 → x＝2。検算 0.3＋0.1＝0.4○。
問12両辺×10：12x−5＝7x＋20 → 5x＝25 → x＝5。検算 6−0.5＝5.5＝3.5＋2○。
問9・問12 整数の項（16や2）に×10をかけ忘れない。問11 0.15があるのに×10だけで止めない＝1.5xになって整数にならない。いちばん小さい位に合わせて10倍か100倍を決める。
こたえ問9 x＝4／問10 x＝4／問11 x＝2／問12 x＝5

大D★分数係数（両辺に分母の最小公倍数）L3｜9月1(4)目安 8分・6問
分数は両辺に「分母の最小公倍数」をかけて分母を払ってから解こう。
問13 x2＋1＝x4＋4
問14 x3−2＝x6
問15 x−23＝x＋25
問16 x＋34＝x−32
問17 x＋42−x−13＝3
問18★山 3x−14−x−22＝2
分数は両辺に分母の最小公倍数をかけて分母を払う。分子が2つ以上ある項は、はらった後を必ず( )でくくる。整数の項にも最小公倍数をかける（かけ忘れ最多）。
問13分母2,4の最小公倍数4を両辺に：2x＋4＝x＋16 → x＝12 → x＝12。検算 6＋1＝7＝3＋4○。
問14分母3,6の最小公倍数6：2x−12＝x → x＝12 → x＝12。検算 4−2＝2＝12÷6○。
問15・★かっこ分母3,5の最小公倍数15：5(x−2)＝3(x＋2) → 5x−10＝3x＋6 → 2x＝16 → x＝8。検算 63＝2＝105○。
問16・★かっこ分母4,2の最小公倍数4：(x＋3)＝2(x−3) → x＋3＝2x−6 → 9＝x → x＝9。検算 124＝3＝62○。
問17・★−のかっこ分母2,3の最小公倍数6：3(x＋4)−2(x−1)＝18 → 3x＋12−2x＋2＝18 → x＋14＝18 → x＝4。検算 82−33＝4−1＝3○。
問18・★最重要分母4,2の最小公倍数4：(3x−1)−2(x−2)＝8 → 3x−1−2x＋4＝8 → x＋3＝8 → x＝5。検算 144−32＝3.5−1.5＝2○。
−の前の分数をはらうと、中の全項の符号が変わる：問17 −2(x−1)＝−2x＋2、問18 −2(x−2)＝−2x＋4。右辺の整数（3や2）にも最小公倍数をかける（問17は3×6＝18、問18は2×4＝8）。4×(x−1) を 4x−1 としない＝分子のかたまりは必ずかっこ。
