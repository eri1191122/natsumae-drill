大A同類項をまとめる（文字の表し方）L2 基礎目安 4分・4問
次を計算して、いちばん簡単な形にしよう。
問1 3x＋2x
問2 7a−3a
問3 5y−y
問4 4x−6x＋x
① どんな計算？＝同じ文字どうし（同類項）を足したり引いたりして1つにまとめる計算。
② なぜまとめられる？ x は「正体のわからない同じ数」。3x＋2x は「x が3個＋x が2個＝x が5個」だから 5x。同類項＝文字の部分が同じ仲間だけ足せる。x と数（3 とか）はちがう仲間なので足せない。
問1でやってみる：
step1 3x と 2x は同じ x の仲間 → 数の部分（係数）だけ足す。
step2 3＋2＝5 → 5x。文字 x はそのまま残す。
3x＋2x を 5x² としない（文字はそのまま x。かけ算じゃないので x は増えない）。5y−y を 5 としない（y は 1y。5y−1y＝4y。文字は消えない）。x と数を混ぜない（3x＋2 は 5x にできない＝別の仲間）。
x＝2 を入れて両方が同じ値か確かめる。問1：左 3×2＋2×2＝10、答え 5×2＝10。合う◎。
問13x＋2x ＝（3＋2）x ＝ 5x。〔検算 x＝2：6＋4＝10＝5×2◎〕
問27a−3a ＝（7−3）a ＝ 4a。〔a＝2：14−6＝8＝4×2◎〕
問3・★1は省く5y−y ＝ 5y−1y ＝（5−1）y ＝ 4y。〔y＝2：10−2＝8＝4×2◎〕
問44x−6x＋x ＝（4−6＋1）x ＝ −x。係数の計算 4−6＝−2、−2＋1＝−1 → −1x＝−x。〔x＝2：8−12＋2＝−2＝−1×2◎〕
こたえ問1 5x／問2 4a／問3 4y／問4 −x

大B一次式のたし算・ひき算（かっこを外す）L2→L3 標準目安 6分・4問
かっこを外してから、同類項をまとめよう。
問5 (3x＋2)＋(x−5)
問6 (6x−1)＋(2x＋4)
問7 (5x＋3)−(2x＋7)
問8 (4x−2)−(7x−5)
① どんな計算？＝かっこのついた式どうしのたし算・ひき算。
② ルール（ここが命）： ＋のかっこはそのまま外す（中の符号は変わらない）。−のかっこは、中の全部の項の符号が変わる。−(2x＋7)＝−2x−7、−(7x−5)＝−7x＋5。「−でかけ算する」と思うと符号をまちがえない。外したら文字は文字どうし・数は数どうしでまとめる。
問7でやってみる：
step1 ＋のかっこをそのまま外す → 5x＋3。
step2 −のかっこは全部の符号を変えて外す → −(2x＋7)＝−2x−7。
step3 並べる：5x＋3−2x−7。文字 5x−2x＝3x、数 3−7＝−4 → 3x−4。
−(2x＋7) を −2x＋7 としない（後ろの ＋7 も −7 に変わる）。−(7x−5) を −7x−5 としない（−5 は ＋5 に変わる）。−のかっこは「中の全部」の符号が変わる。1つだけ変えるミスが最多。
x＝1 を入れて両方が同じ値か確かめる。問7：左（5＋3）−（2＋7）＝8−9＝−1、答え 3×1−4＝−1。合う◎。
問53x＋2＋x−5 → 文字 4x、数 −3 → 4x−3。〔x＝1：5−4＝1＝4−3◎〕
問66x−1＋2x＋4 → 文字 8x、数 3 → 8x＋3。〔x＝1：5＋6＝11＝8＋3◎〕
問7・★−のかっこ5x＋3−2x−7 → 文字 3x、数 −4 → 3x−4。〔x＝1：8−9＝−1◎〕
問8・★−のかっこ−(7x−5)＝−7x＋5。4x−2−7x＋5 → 文字 −3x、数 3 → −3x＋3。〔x＝1：2−2＝0＝−3＋3◎〕
こたえ問5 4x−3／問6 8x＋3／問7 3x−4／問8 −3x＋3

大C分配法則（数をかっこの中の全部にかける）L2→L3 標準目安 6分・4問
分配法則でかっこを外してから、同類項をまとめよう。
問9 2(3x−1)
問10 −3(x＋4)
問11 4(2x−3)＋5
問12 3(x＋2)−2(x−4)
① どんな計算？＝かっこの前の数を、中の項にかけて外す計算。
② ルール： a(b＋c)＝ab＋ac。前の数を「中の全部の項」にかける（後ろの項にかけ忘れが最多）。
マイナスをかけるとき： 前が −3 や −2 なら、かけた相手の符号も変わる。−3(x＋4)＝−3x−12、−2(x−4)＝−2x＋8（−2×−4＝＋8）。
問12でやってみる：
step1 3(x＋2)＝3×x＋3×2＝3x＋6。
step2 −2(x−4)＝−2×x−2×(−4)＝−2x＋8。
step3 並べる：3x＋6−2x＋8。文字 3x−2x＝x、数 6＋8＝14 → x＋14。
後ろの項にかけ忘れない：2(3x−1) を 6x−1 としない（−1にも2をかけて −2）。マイナスの分配で符号を落とさない：−3(x＋4) を −3x＋4 としない（＝−3x−12）。−2(x−4) を −2x−8 としない（−2×−4＝＋8 → −2x＋8）。
x＝1 を入れて確かめる。問12：左 3×3−2×(−3)＝9＋6＝15、答え 1＋14＝15。合う◎。
問92×3x＋2×(−1)＝6x−2。〔x＝1：2×2＝4＝6−2◎〕
問10・★−の分配−3×x＋(−3)×4＝−3x−12（＋4にも−3をかけて−12）。〔x＝1：−3×5＝−15＝−3−12◎〕
問114×2x＋4×(−3)＋5＝8x−12＋5＝8x−7。〔x＝1：4×(−1)＋5＝1＝8−7◎〕
問12・★−の分配3x＋6−2x＋8 → 文字 x、数 14 → x＋14。〔x＝1：9＋6＝15＝1＋14◎〕
こたえ問9 6x−2／問10 −3x−12／問11 8x−7／問12 x＋14

大D★分数の係数の一次式（通分してまとめる）L3｜9月大問1目安 8分・4問
分母をそろえて（通分して）から、分子の同類項をまとめよう。
問13 x2＋x3
問14 3x4−x2
問15 x＋12＋x−12
問16★山 2x−13＋x＋43
① どんな計算？＝分数の形の文字式どうしのたし算・ひき算。
② ルール： 分母がちがうときは分母の最小公倍数で通分してから、分子の同類項をまとめる。
分子が「x＋1」のようにかたまりのときは、必ず( )でくくる。これは方程式の「分母をはらう」とはちがう計算（計算式なので分母は消えない。＝で結ばれた方程式だけ分母を消せる）。
問13でやってみる：
step1 分母 2 と 3 の最小公倍数は 6。
step2 通分：x2＝3x6（上下に3をかける）、x3＝2x6（上下に2をかける）。
step3 分子をたす：3x＋2x＝5x → 5x6。
x2＋x3 を x5 としない（分母どうしを足すのは×。通分して 5x6）。計算式なのに分母を勝手に消さない（＝がない式は方程式ではないので分母は残る）。問16 分子のかたまりは( )でくくる：(2x−1)＋(x＋4) を 2x−1＋x＋4 と書ければOK、符号に注意。
x＝6 など分母で割り切れる数を入れて確かめる。問13：左 6/2＋6/3＝3＋2＝5、答え 5×6/6＝5。合う◎。
問13最小公倍数6で通分：3x6＋2x6＝5x6。〔x＝6：3＋2＝5＝5×66◎〕
問14最小公倍数4で通分：x2＝2x4。3x4−2x4＝x4。〔x＝4：3−2＝1＝44◎〕
問15・★分子はかたまり分母が同じ2なので分子をたす：(x＋1)＋(x−1)＝2x → 2x2＝x（＋1と−1が消える、約分できる）。〔x＝3：2＋1＝3＝x◎〕
問16・★最重要分母が同じ3なので分子をたす：(2x−1)＋(x＋4)＝3x＋3 → 3x＋33＝x＋1（分子を3でわって約分）。〔x＝3：5/3＋7/3＝12/3＝4＝3＋1◎〕
こたえ問13 5x6／問14 x4／問15 x／問16 x＋1

大E代入（負の数・分数を x に入れる）L3 応用｜9月大問1目安 4分・2問
x のところに数を入れて（代入して）、式の値を求めよう。
問17 x＝−4 のとき　2x＋5
問18 x＝12 のとき　4x−3
① どんな計算？＝文字 x のところに数を置きかえて、ふつうの計算にして値を出す。
② ルール（命）： 負の数や分数を入れるときは必ず( )で囲んで置きかえる。x＝−4 なら 2x＝2×(−4)。こうすると符号を落とさない。x が「2x」のとき、x をただ −4 とくっつけて「2−4」にしないこと。
問17でやってみる：
step1 x のところを (−4) に置きかえる → 2×(−4)＋5。
step2 先にかけ算：2×(−4)＝−8。
step3 たし算：−8＋5＝−3。
2x を 2−4＝−2 としない（2x は 2×x。2×(−4)＝−8）。かっこを書かずに符号を落とさない（−4 のマイナスを消してしまうミスが多い）。問18 4× 12 を 42 のままにしない（約分して 2）。
出した値は、もとの式に戻すと合っているはず。問17：−3 から逆算 −3−5＝−8、÷2＝−4＝x。合う◎。
問172×(−4)＋5＝−8＋5＝−3。
問184×12−3＝2−3＝−1（4×12＝2）。
こたえ問17 −3／問18 −1
