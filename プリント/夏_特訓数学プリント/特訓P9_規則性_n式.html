規則性｜ならび方のきまりを「nの式」にして、大きい番目も一発で出す

夏・特訓対策　目安：約30分（問1〜問7・通し番号）　対象: 中1　全問に徹底解説つき

特訓選抜 大問4と全国統一テストで出る「規則性」。コツは①まず1・2・3番目を表に書く②差（いくつずつ増える）を見る③「差×n＋調整」でnの式を作る。式さえできれば100番目でも一瞬。図のならびは「変わる所×n＋変わらない所」で考える。

  1. ① 1・2・3番目を表に書く（具体を並べる）
  2. ② 差を見る（毎回いくつ増える？＝この数が n の係数）
  3. ③ n式を作る：差×n＋（1番目−差）。図形は「変わる部分×n＋変わらない部分」
  4. ④ 確かめ：n＝1,2 を入れて表と合うか
問1〜2数の並びを n の式にする L2→L3目安 6分
問1次の数の並びの、n番目の数を n の式で表しなさい。　3, 7, 11, 15, 19, …

問2次の数の並びの、n番目の数を n の式で表しなさい。　5, 8, 11, 14, 17, …
番号と数を表に書き、差を調べてから n の式を考えよう。
等差の並びの公式：n番目＝（差）×n＋（1番目−差）。差は「毎回いくつ増えるか」。最後に n＝1 で確かめる。
問1差は4（3→7→11…）。式＝4×n＋(3−4)＝4n−1。確かめ n＝1：4−1＝3○、n＝2：8−1＝7○。
問2差は3（5→8→11…）。式＝3×n＋(5−3)＝3n＋2。確かめ n＝1：3＋2＝5○、n＝2：6＋2＝8○。
差だけ書いて「4n」で終わらない（1番目に合わせる調整 −1 が要る）。必ず n＝1 で検算。
こたえ問1 4n−1／問2 3n＋2

問5n式を使って「大きい番目」「何番目」を出す L3目安 4分
問5問1の並び 3, 7, 11, 15, …（n番目＝4n−1）について、(1) 20番目の数を求めなさい。(2) 99 は何番目ですか。
(1)は n に数を入れる、(2)は n の方程式にして解く。ノートに式を書こう。
(1) 代入n＝20 を 4n−1 へ：4×20−1＝79。79。
(2) 逆算（方程式）4n−1＝99 → 4n＝100 → n＝25。25番目。検算 4×25−1＝99○。
「n式さえ作れば代入で大きい番目、方程式で何番目」が出せる＝規則性はnの式づくりが命。特訓選抜・模試の(3)で問われる。
こたえ(1) 79／(2) 25番目

問6〜7★碁石を「周」に並べる（角の重複を引く）L4目安 7分
問6碁石を、正方形の形に「周（外側）だけ」並べます。1辺に n 個ずつ並べるとき、碁石は全部で何個ですか。n の式で表しなさい。 問7碁石を、正三角形の形に「周だけ」並べます。1辺に n 個ずつ並べるとき（頂点の石は2辺で共有）、碁石は全部で何個ですか。n の式で表しなさい。
「1辺×辺の数」だと角の石を重複して数えてしまう。重複分をどう引くかをノートで考えよう。
周に並べるコツ：「1辺の数×辺の本数」では角を二重に数える。角の数だけ引く。正方形は角4つ→4n−4、正三角形は角3つ→3n−3。
問6 正方形4辺×n＝4n。でも4つの角を2回ずつ数えている→角4個分を引く：4n−4。確かめ n＝4：16−4＝12（図の○の数）○。
問7 正三角形3辺×n＝3n。3つの角を引く：3n−3。確かめ n＝4：12−3＝9（図の○の数）○。n＝6なら3×6−3＝15。
4n や 3n のままにしない（角の重複を引く）。数えで確かめる（n＝4で12個・9個）。
問6（4n−4）は全国統一テストで×だった型、問7（3n−3）は特訓選抜 大問4のタイル型と同じ。「角を引く」が決め手。
こたえ問6 4n−4／問7 3n−3
