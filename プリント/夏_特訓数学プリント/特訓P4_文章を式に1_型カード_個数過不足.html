これがきみのいちばんの伸びしろ＝「文章を式にする」力。全国統一テストで25点中0点だったのもここ。コツは才能じゃなく手順。下の型カードに毎回そって解けば、誰でも式が作れる。今日はその型を体にしみこませる日。

  1. ① 何を求める？ → それを x とおく（「□を x とする」と書く）
  2. ② 単位をそろえる（円・分・人…バラバラなら直す）
  3. ③ 図にする（線分図・表）＝目で見える形に
  4. ④ 等しい関係を式に（＝で結べる2つの言い方を探す）
  5. ⑤ 解いて検算（出た数を文章に戻して合うか）
大問AL2 基礎問1〜問2・目安 6分
問11個80円のりんごを何個か買ったら、代金は640円でした。りんごは何個買いましたか。
まず「何型か」を考えよう。何を x にするかは自分で決めて、図か表をかいてから式を作ろう。
これは 「個数・代金型」だった。見抜くカギは「1個◯円のものを何個か買って、代金は◯円」という言い方。1つぶんのねだん × 個数 ＝ 代金、これだけ。
①xりんごの個数を x 個とする。
④式代金＝80×個数。これが640。80x＝640。
⑤解両辺を80でわる。x＝640÷80＝8。
⑤検算80×8＝640○。8個。

問21本120円のペンを何本かと、150円のノートを1冊買ったら、代金は750円でした。ペンは何本買いましたか。
まず「何型か」を考えよう。何を x にするかは自分で決めて、図か表をかいてから式を作ろう。
これも 「個数・代金型」だった。ただしノートは1冊と決まっている＝数が変わらない。変わる数（ペンの本数）だけが xで、ノートは150円のまま足すだけ。
①xペンの本数を x 本とする。
④式ペン代＋ノート代＝合計。120x＋150＝750。
⑤解-a150 を右へ移項（＝両辺から150をひく）。120x＝750−150＝600。
⑤解-b両辺を120でわる。x＝600÷120＝5。
⑤検算120×5＋150＝600＋150＝750○。5本。
ノートの150を120にかけない（ノートは1冊だけ＝＋150）。「何本でも変わる数」だけが x につく。
こたえ5本

問350円切手と80円切手を合わせて10枚買ったら、代金は650円でした。50円切手は何枚買いましたか。
まず「何型か」を考えよう。何を x にするかは自分で決めて、図か表をかいてから式を作ろう。
これは 「2種類・合計が決まっている型」だった。見抜くカギは「合わせて10枚」。全体の枚数が分かっているから、片方を x、もう片方は「合計−x」で書ける。文字は1つで済む。
問3 ①表50円を x 枚とすると、80円は残り (10−x) 枚。表にして整理する。
50円切手80円切手合計
枚数（枚）x10 − x10
金額（円）50x80(10 − x)650
問3 ④式金額の合計＝650。50x＋80(10−x)＝650。
問3 ⑤解-aカッコをはずす：50x＋800−80x＝650 → まとめて −30x＋800＝650 → 800 を移項して −30x＝650−800＝−150。
問3 ⑤解-b−30x ＝ −150 のつぎ。両辺を −30 でわる。x ＝ (−150) ÷ (−30)。マイナス ÷ マイナス ＝ プラス → x ＝ 5。
問3 ⑤検算50円が5枚、80円は 10−5＝5枚。50×5＋80×5＝250＋400＝650○。50円は5枚。
80(10−x) を 80×10−x としない（＝800−80x。x にも80をかける）。−の分配は符号注意。
こたえ50円切手 5枚

大問CL3 応用問5〜問6・目安 8分
問5あめを子どもに配ります。1人に4個ずつ配ると7個あまり、1人に5個ずつ配ると3個たりません。子どもの人数を求めなさい。
まず「何型か」を考えよう。何を x にするかは自分で決めて、図か表をかいてから式を作ろう。
これは 「過不足型」だった。見抜くカギは「あまる」「たりない」の2語。変わらないのは全体の数（あめ）。これを2通りの配り方で表して＝で結ぶ。あまる＝＋、たりない＝−（全体＝1人分×人数＋あまり ／ 全体＝1人分×人数−たりない）。
問5 ①x子どもの人数を x 人とする（求めるもの）。
配り方配ったぶんあまり／たりないあめの数
4個ずつ4x 個7個あまる（＋7）4x ＋ 7
5個ずつ5x 個3個たりない（−3）5x − 3
問5 ④式同じあめの数だから ＝ で結ぶ。4x＋7＝5x−3。
問5 ⑤解-a移項してまとめる：7＋3＝5x−4x → 10＝x。
問5 ⑤解-b10＝x のまま終わらない。ふつうは左右を入れかえて x＝10 と書く（x を左に置くのが答えの形）。
問5 ⑤検算4×10＋7＝47個、5×10−3＝47個。どちらも47個で一致○。10人。
「たりない」を ＋ にしない（たりない＝−）。あまる＝＋、たりない＝− を取りちがえると符号がまるごとずれる。
こたえ10人

問6えんぴつを生徒に配ります。1人に3本ずつだと20本あまり、1人に5本ずつだと4本たりません。生徒の人数とえんぴつの本数を求めなさい。
まず「何型か」を考えよう。何を x にするかは自分で決めて、図か表をかいてから式を作ろう。
これも 「過不足型」だった。あめがえんぴつに変わっただけ。変わらないのはえんぴつの本数。2通りで表して＝で結ぶ。あまる＝＋、たりない＝−。
① 問6 ①x〜問6 ⑤解-a
問6 ①x生徒の人数を x 人とする。
配り方配ったぶんあまり／たりないえんぴつの本数
3本ずつ3x 本20本あまる（＋20）3x ＋ 20
5本ずつ5x 本4本たりない（−4）5x − 4
問6 ④式同じえんぴつだから 3x＋20＝5x−4。
問6 ⑤解-a移項してまとめる：20＋4＝5x−3x → 24＝2x。
② 問6 ⑤解-b〜問6 ⑤検算
問6 ⑤解-b左右を入れかえて 2x＝24。両辺を2でわる → x＝24÷2＝12（生徒12人）。
問6 ⑤解-cえんぴつの本数も聞かれている。3×12＋20＝56 → 56本。
問6 ⑤検算5×12−4＝60−4＝56本。もう一方の配り方でも56本で一致○。12人・56本。
「たりない」を ＋ にしない（たりない＝−）。人数だけ答えて本数を答え忘れない（問われたものを最後に確認）。
「全体を2通りで表して＝」は特訓選抜 大問3（方程式の利用）の定番。この発想は比・速さ・割合でもそのまま使える。
こたえ生徒 12人・えんぴつ 56本

問8現在、母は40歳、子は12歳です。母の年齢が子の年齢のちょうど3倍になるのは何年後ですか。
まず「何型か」を考えよう。何を x にするかは自分で決めて、図か表をかいてから式を作ろう。
これは 「年齢型」だった。見抜くカギは「何年後」。x 年後には、2人とも同じだけ ＋x される（片方だけ歳をとることはない）。そのうえで「◯倍」を式にする。
問8 ①xx 年後とする。x年後は母 (40＋x) 歳、子 (12＋x) 歳。
いまx 年後
母40歳40 ＋ x
子12歳12 ＋ x
問8 ④式母＝子の3倍。40＋x＝3(12＋x)。
問8 ⑤解-aカッコをはずす：40＋x＝36＋3x。移項して 40−36＝3x−x → 4＝2x。
問8 ⑤解-b左右を入れかえて 2x＝4。両辺を2でわる → x＝4÷2＝2。
問8 ⑤検算2年後、母は42歳、子は14歳。14×3＝42○。ちょうど3倍。2年後。
「3倍」をどちらにかけるか＝大きいほう（母）が小さいほう（子）の3倍。母＝子×3。両方に＋x するのを忘れない。
こたえ2年後
