夏・特訓対策　目安：約30分（問1〜問8・通し番号）　対象: 中1　全問に徹底解説つき

これがきみのいちばんの伸びしろ＝「文章を式にする」力。全国統一テストで0/25だったのもここ。コツは才能じゃなく手順。下の型カードに毎回そって解けば、誰でも式が作れる。今日はその型を体にしみこませる日。

  1. ① 何を求める？ → それを x とおく（「□を x とする」と書く）
  2. ② 単位をそろえる（円・分・人…バラバラなら直す）
  3. ③ 図にする（線分図・表）＝目で見える形に
  4. ④ 等しい関係を式に（＝で結べる2つの言い方を探す）
  5. ⑤ 解いて検算（出た数を文章に戻して合うか）
問1〜2個数と代金（型カードの練習）L2 基礎目安 6分
問11個80円のりんごを何個か買ったら、代金は640円でした。りんごは何個買いましたか。

問21本120円のペンを何本かと、150円のノートを1冊買ったら、代金は750円でした。ペンは何本買いましたか。
型カードの①〜④（何を x →図→等式）までをノートに書いてから解こう。
問1 ①xりんごの個数を x 個とする。
問1 ④式代金＝80×個数。これが640。80x＝640。
問1 ⑤解x＝640÷80＝8。検算 80×8＝640○。8個。
問2 ①xペンの本数を x 本とする。
問2 ④式ペン代＋ノート代＝合計。120x＋150＝750。
問2 ⑤解120x＝600 → x＝5。検算 600＋150＝750○。5本。
問2 ノートの150を120にかけない（ノートは1冊だけ＝＋150）。「何本でも変わる数」だけが x につく。

問3〜42種類の個数（合計が決まっている）L3 標準目安 7分
問350円切手と80円切手を合わせて10枚買ったら、代金は650円でした。50円切手は何枚買いましたか。

問4大人と子どもが合わせて30人います。大人の入場料は500円、子どもは300円で、合計は11000円でした。大人は何人ですか。
「片方を x、もう片方を（合計−x）」と表にしてからノートで式を作ろう。
2種類で合計が決まっているとき：片方を x、もう片方は「合計−x」で表す。これで文字は1つで済む。
問3 ①表50円を x 枚 → 80円は (10−x) 枚。
┌─────┬─────┬──────┐ │ │ 50円 │ 80円 │ ├─────┼─────┼──────┤ │ 枚数 │ x │ 10−x │ │ 金額 │ 50x │80(10−x)│ └─────┴─────┴──────┘
問3 ④式金額の合計＝650。50x＋80(10−x)＝650。
問3 ⑤解50x＋800−80x＝650 → −30x＝−150 → x＝5。検算 250＋400＝650○。50円は5枚。
問4 ①表大人を x 人 → 子どもは (30−x) 人。
問4 ④式500x＋300(30−x)＝11000。
問4 ⑤解500x＋9000−300x＝11000 → 200x＝2000 → x＝10。検算 5000＋6000＝11000○。大人10人。
300(30−x) を 300×30−x としない（＝9000−300x。x にも300をかける）。−の分配は符号注意。

問5〜6★過不足算（あまる・たりない）L3 応用目安 8分
問5あめを子どもに配ります。1人に4個ずつ配ると7個あまり、1人に5個ずつ配ると3個たりません。子どもの人数を求めなさい。

問6えんぴつを生徒に配ります。1人に3本ずつだと20本あまり、1人に5本ずつだと4本たりません。生徒の人数とえんぴつの本数を求めなさい。
「配り方が2通り→同じ全体の数を2通りで表して＝で結ぶ」。x を決めてノートに2通りを書こう。
過不足のコツ：変わらないのは「全体の数（あめ・えんぴつ）」。これを2通りの配り方で表して＝で結ぶ。あまる＝+、たりない＝−（全体＝1人分×人数＋あまり ／ 全体＝1人分×人数−たりない）。
問5 ①x子どもの人数を x 人とする（求めるもの）。
問5 ④式あめの数を2通りで：4個ずつ＝4x＋7、5個ずつ＝5x−3。同じあめだから 4x＋7＝5x−3。
問5 ⑤解7＋3＝5x−4x → 10＝x。検算 4×10+7＝47、5×10−3＝47○。10人。
問6 ①x生徒の人数を x 人とする。
問6 ④式えんぴつ＝3x＋20（あまる）＝5x−4（たりない）。3x＋20＝5x−4。
問6 ⑤解20＋4＝5x−3x → 24＝2x → x＝12。えんぴつ＝3×12＋20＝56。検算 5×12−4＝56○。12人・56本。
「たりない」を ＋ にしない（たりない＝−）。問6で人数だけ答えて本数を答え忘れない（問われたものを最後に確認）。
過不足は特訓選抜 大問3（方程式の利用）の定番。「全体を2通りで表して＝」は比・速さ・割合でも同じ発想。
こたえ問5 10人／問6 生徒12人・えんぴつ56本

問7〜8特訓型（過不足・年齢）L3 応用目安 7分
問7クラスで画用紙を配ります。1人に5枚ずつ配ると12枚たりず、1人に4枚ずつ配ると9枚あまります。クラスの人数と画用紙の枚数を求めなさい。

問8現在、母は40歳、子は12歳です。母の年齢が子の年齢のちょうど3倍になるのは何年後ですか。
型カードにそって、x を決め→2通り（または等しい関係）をノートに作ってから解こう。
問7 ④式人数を x。画用紙＝5x−12（たりない）＝4x＋9（あまる）。5x−12＝4x＋9。
問7 ⑤解5x−4x＝9＋12 → x＝21。画用紙＝4×21＋9＝93。検算 5×21−12＝93○。21人・93枚。
問8 ①xx 年後とする。x年後は母(40＋x)歳、子(12＋x)歳。
問8 ④式母＝子の3倍：40＋x＝3(12＋x)。
問8 ⑤解40＋x＝36＋3x → 40−36＝3x−x → 4＝2x → x＝2。検算 母42＝子14×3○。2年後。
問8 「3倍」をどちらにかけるか＝大きいほう（母）が小さいほう（子）の3倍。母＝子×3。両方に＋x するのを忘れない。
こたえ問7 21人・93枚／問8 2年後
