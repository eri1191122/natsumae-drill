特訓P5b数学：方程式の利用★文章→式 L3（速さ）

文章を式にする｜速さ②を「4問・じっくり」で固める（往復・出会い・追いつき・トンネル）

夏・特訓対策　目安：約30分（問1〜問4・通し番号）　対象: 中1　全問に徹底解説つき

速さの文章→式はきみの伸びしろのど真ん中。全国統一テストで0/25だったのもこの種類。今日は4問だけ。そのぶん1問ずつ図（線分図）をかいてから式を作る。コツはたった2つ＝①道のり＝速さ×時間②単位をそろえる。数をこなすより「型を体に入れる」のが今日の目標。

  1. ① 何を求める？ → それを x とおく（「□を x とする」と書く）
  2. ② 図にする（線分図・ダイヤ図）＝だれがどこをどれだけ進むかを見える形に
  3. ③ 等しい関係を式に（道のりや時間で「＝で結べる2つの言い方」をさがす）
  4. ④ 解く（移項して x ＝ の形へ。分数は両辺×分母で払う）
  5. ⑤ 検算（出た数を文章に戻して合うか確かめる）
問1往復（行き帰りで速さがちがう・時間の合計）L3 標準目安 7分
問1家から公園まで、行きは分速60m、帰りは分速90mで往復したら、合計で25分かかりました。家から公園までの道のりは何mですか。
行き帰りで「道のりは同じ」（どちらも x m）。ちがうのは速さ→だから時間がちがう。「時間＝道のり÷速さ」で行き・帰りの時間を表し、合計で式を作ろう。
⓪ 速さの3つの関係
  • 道のり＝速さ×時間　時間＝道のり÷速さ　速さ＝道のり÷時間
  • 分速のときは時間は「分」で出る。単位をそろえるのもわすれずに。
往復は道のりが行き帰り同じ（x）。立てる式は「時間の合計」。x を時間にしないで、道のりを x にするのがコツ。
①x家から公園までの道のりを x m とする（行きも帰りも同じ x）。
③式時間＝道のり÷速さ。行き x60 分、帰り x90 分。合計が25分だから x60＋x90＝25。
④解両辺×180（60と90の最小公倍数）：3x＋2x＝4500 → 5x＝4500 → x＝900。
⑤検算行き 900÷60＝15分、帰り 900÷90＝10分。15＋10＝25分○。
x を「時間」にしない。求めるのは道のりだから x は道のり。行き帰りで道のりは同じ、ちがうのは時間、と整理しよう。分数は両辺に分母の最小公倍数をかけて払う（÷ではなく×）。
「往復＝道のりが同じ・時間の合計で式」は特訓選抜 大問6（速さ）の超定番。時間や速さがちがっても、まず線分図で同じ部分をさがす。

問2★出会い算（向かい合って進み出会う）L3 応用目安 7分
問21500mはなれたP地点とQ地点から、AさんはPを分速60m、BさんはQを分速90mで、同時に向かい合って進みました。2人は何分後に出会いますか。
向かい合って進むと、2人の進んだ道のりを合わせると、はじめの間の距離（1500m）になる。出会うまでの時間を x にしよう。
出会い算＝2人の進んだ道のりの合計＝はじめにはなれていた距離。「合計」で式を立てる（差ではない）。同時に出発だから、2人とも進んだ時間は同じ x。
①x出発してから x 分後に出会うとする（2人とも x 分進む）。
③式道のりの合計＝1500：A は 60x m、B は 90x m 進む。60x＋90x＝1500。
④解150x＝1500 → x＝10。
⑤検算A 60×10＝600m、B 90×10＝900m。600＋900＝1500m○。ちょうど出会う。
「合計」を「差」にしない。向かい合うと近づくので、2人の道のりを足すと間の距離になる。引き算にすると追いつき算とまざるので注意。
出会い算は特訓選抜 大問6の定番。次の問3「追いつき算」と「足すか引くか」だけがちがう。図でどちら向きかを必ず確認しよう。

問3★追いつき算（先に出た人を後から追う）L3 応用目安 8分
問3弟が分速50mで家を出発しました。その12分後に兄が分速90mで同じ道を追いかけました。兄が出発してから何分後に弟に追いつきますか。
追いつく＝「2人の進んだ道のりが同じ」になったとき。気をつけるのは時間のずれ。弟は兄より12分多く歩いている。兄が出発してからの時間を x にしよう。
追いつき算＝2人の進んだ道のりが等しくなる。同じ家から同じ道を進むので、追いついた瞬間は「家からの道のり」が一致する。出会い算（合計）とちがって＝で結ぶ。
①x兄が出発してから x 分後に追いつくとする。弟は兄より12分早く出たので、そのとき弟は (x＋12) 分歩いている。
③式道のりが等しい：兄 90x ＝ 弟 50(x＋12)。90x＝50(x＋12)。
④解90x＝50x＋600 → 90x−50x＝600 → 40x＝600 → x＝15。
⑤検算兄 90×15＝1350m。弟 50×(15＋12)＝50×27＝1350m。どちらも1350mで一致○。
弟の時間を x にしない（弟は x＋12 分歩いている）。50(x＋12) を 50x＋12 としない＝かっこの中の全部に50をかける（＝50x＋600）。
追いつき算は特訓選抜 大問6の頻出。「先に出た人は時間が多い」を ＋ で式に入れるのがカギ。問2の出会い（合計）と見くらべて、足すか＝かを図で判断しよう。

問4★列車がトンネルを通る（列車の長さ込み）L3 応用目安 8分
問4長さ120mの列車が、秒速25mで走っています。この列車があるトンネルに入りはじめてから、完全に出おわるまでに32秒かかりました。トンネルの長さは何mですか。
「入りはじめてから出おわるまで」に列車の先頭が進む道のり＝トンネルの長さ＋列車の長さ。これを「道のり＝速さ×時間」で式にしよう。トンネルの長さを x にする。
トンネル・橋の問題は列車の長さを忘れないのがすべて。先頭が進む道のり＝トンネルの長さ＋列車の長さ（入りはじめ＝先頭が入口、出おわり＝最後尾が出口）。
①xトンネルの長さを x m とする。
③式先頭が進む道のり＝トンネル＋列車＝(x＋120) m。道のり＝速さ×時間だから x＋120＝25×32。
④解x＋120＝800 → x＝800−120 → x＝680。
⑤検算進む道のり＝680＋120＝800m。速さ×時間＝25×32＝800m。一致○。
トンネルの長さだけにしない（＝25×32＝800 を答えにしない）。完全に出おわるには、列車の長さ120m分も先頭が余分に進む。だから＋列車の長さ。秒速と秒でそろっているので単位変換は不要。
「通過＝対象の長さ＋列車の長さ」は特訓選抜 大問6の応用パターン。鉄橋・ホームでも同じ考え方。単位がちがう問題（時速→秒速）も出るので、いつも単位をそろえる習慣を。
こたえトンネルの長さ 680m
