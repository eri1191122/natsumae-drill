規則性・総合｜「どの型か自分で見分ける」インターリーブ特訓（本番型ミックス）

夏・特訓対策　目安：約35分（問1〜問4・通し番号）　対象: 中1　全問に7層の徹底解説つき

これまでP9でやってきた型（タイル・碁石・マッチ・三角数・数表・階段）をわざとバラバラに混ぜたのがこのプリント。本番の大問4は「これは○○の型ですよ」と教えてくれない。自分で型を見分ける力こそ9月特訓選抜のカギ。今日は「同じ問題のくり返し」ではなく、毎問ちがう型。まず上の見分け方フローを使って「これは何の型か」を当ててから解こう。式さえ作れれば100番目でも一瞬だよ。

  1. ① 差は一定？ → 1・2・3番目を書いて「差」を計算。差が一定なら 差×n＋(1番目−差) の一次の型。差が一定でない(差がだんだん増える)なら合計の型 or 平方数の型を疑う。
  2. ② 図のどこが増える？ → 「まわり1周」なら囲みの型、「横に1個連結」なら共有辺の型、「段が1段増える」なら三角数・階段の型。増える部分に印をつけて数える。
  3. ③ 聞かれているのは「合計」か「個数(その番目だけ)」か？ → 「ぜんぶで何個」「合計」なら足し算の型(三角数 n(n+1)÷2 など)。「n番目は何個」ならその番目だけの型。数表なら「行と列」で位置を出す型。
見分けの特徴たぶんこの型使う式の形

問1マッチ棒で正三角形を、となりと1辺をくっつけながら横に並べていきます。下の図は1番目・2番目・3番目です。

①なに型?まず見分けフロー。これは「マッチ棒を横に1個ずつ増やす」図形数列。最初に3本、2本3本…と並ぶので、差が一定の等差の型かどうかを確かめにいく。
②数える図を数えると 1番目＝3本、2番目＝5本、3番目＝7本。表に「番目」と「本数」を上下に分けて書く。
③差を見る5−3＝2、7−5＝2。1個増えるごとに2本ずつ増える（差は2で一定）。→ 等差の型だと確定。なぜ2本かというと、三角形を1個足すとき、くっつく1辺は前の三角形と共有して数えないので、新しく増えるのは残り2本だけだから。
④式を作る等差の式＝差×n＋(1番目−差)。差＝2、1番目＝3 → 2×n＋(3−2)＝2n＋1。(1)の答え＝2n＋1。
⑤検算n＝1:2×1+1＝3○、n＝2:2×2+1＝5○、n＝3:2×3+1＝7○。表とぴったり。式は正しい。
⑥代入と逆算(2)代入：n＝31 を 2n+1 へ → 2×31＋1＝62＋1＝63本。(3)逆算：本数＝41 だから 2n＋1＝41 → 両辺−1で 2n＝40 → ÷2で n＝20。20番目。逆算は確かめも：2×20+1＝41○。
⑦なぜこの型と見抜けた?決め手は「差がずっと2で一定」だったこと。差が一定なら必ず 差×n＋(1番目−差) の一次の型。図が三角でもタイルでも碁石でも、差が一定なら全部この型。逆に「合計」を聞かれていたり差がだんだん増えるなら別の型を疑う。今回は「n番目の本数」を聞いていて差も一定だから、迷わずこの型。
「マッチ棒を横に1辺くっつけて並べる」と書いてあったら、まずくっつく辺は共有で1本に数えないを思い出す。だから新しく増える本数が差になる。三角形なら＋2、正方形なら＋3になりやすい。図を見て「増える棒だけ」に印をつけるのがコツ。
「3n」で終わらせない。三角形1個に3本だからと 3n にすると n＝2 で6本になり、図の5本と合わない。くっつく辺を二重に数えているミス。必ず n＝1 で検算する。逆算で 41÷2＝20.5 と即答しない。先に＋1を移項（−1）してから÷2。

問2合計の型：碁石の三角数（足し算の規則）L3-L4目安 9分
問2碁石を三角形の形に、1段目に1個、2段目に2個、3段目に3個…と1段ずつ増やしながら積みます。下の図は3段目までです。
(1) n 段目まで積んだとき、碁石はぜんぶで何個ですか。n の式で表しなさい。
(2) 10段目まで積むと碁石はぜんぶで何個ですか。
(3) 碁石をちょうど55個ぜんぶ使い切ったとき、何段目まで積めましたか。
各段の数（1,2,3…）と、ぜんぶの合計（1, 3, 6…）はちがう。聞かれているのは「ぜんぶで」＝合計
見分けフロー③が決め手。「ぜんぶで何個」＝合計を聞かれている。各段は1,2,3…と増えるが、合計は 1段目1個・2段目まで3個・3段目まで6個。この合計の並びの差を見てみよう（差が一定でないはず）。
段目 n1234…
その段の個数1234…
合計(ぜんぶ)13610…
①なに型?問題文に「ぜんぶで・合計」がある。見分けフロー③で、これは足し算の型（合計の型）だとアタリをつける。各段の個数(1,2,3)ではなく、足し上げた合計(1,3,6,10)を追う。
②合計を書く1段目まで＝1、2段目まで＝1＋2＝3、3段目まで＝1＋2＋3＝6、4段目まで＝1＋2＋3＋4＝10。これが「三角数」とよばれる並び。
③差を見て型を確定合計の差は 3−1＝2、6−3＝3、10−6＝4。差が2,3,4とだんだん増える＝一定でない。だから等差(2n+1のような一次)の型ではない。差が一定でない合計→三角数の型と確定。
④式を作る三角数は 1＋2＋…＋n の合計で、公式の形は n×(n＋1)÷2。これは「1からnまでの数を全部足すと n(n+1)÷2 になる」というもの。(1)の答え＝n(n＋1)÷2。
⑤なぜ n(n+1)÷2?暗記でなく理由：同じ三角を2つ用意してひっくり返してくっつけると、たて n 個・よこ (n＋1) 個の長方形ができる。長方形の個数は n×(n＋1)。その半分が三角1つ分だから ÷2。だから合計＝n(n＋1)÷2。検算：n＝3なら3×4÷2＝6○、n＝4なら4×5÷2＝10○。
⑥代入と逆算(2)代入：n＝10 → 10×(10＋1)÷2＝10×11÷2＝110÷2＝55個。(3)逆算：合計＝55 → n×(n＋1)÷2＝55 → 両辺×2で n×(n＋1)＝110。となり合う2数の積が110になるのを探すと 10×11＝110。だから n＝10、10段目。確かめ：10×11÷2＝55○。
⑦なぜこの型と見抜けた?2つの決め手。1つ目は「ぜんぶで・合計」という言葉（見分けフロー③）。2つ目は「合計の差が一定でない（2,3,4と増える）」こと。この2つがそろったら三角数の型。逆に「n段目だけは何個」なら答えは n でいい（合計ではない）。「合計か、その段だけか」を読み分けるのが全て。今回は合計だから n(n+1)÷2。
「ぜんぶで」「合計」「全部使い切る」「積み上げた総数」の言葉が出たら足し算の型を最優先で疑う。さらに合計の差をとってみて、差が一定でない（だんだん増える）なら三角数 n(n+1)÷2。差がずっと同じなら問1のような一次の型。合計の数列の「差」を見るのが見分けの裏ワザ。
各段の個数(n)と合計をごちゃ混ぜにしない。「10段目」を答えに10と書くのは「その段の個数」。聞かれた合計は55。n(n+1)÷2 を n²÷2 と書かない（＋1を落とすミス）。逆算で ÷2 を忘れて n(n+1)＝55 としない。先に×2して110にしてから2数を探す。

問3数表の型：行と列で場所を出す L3-L4目安 9分
問3下のように、1から順に数を1行に5個ずつ、左から右へ、上から下へと並べていきます。
(1) 4行3列の数はいくつですか。
(2) 数の「48」は何行何列にありますか。
(3) a 行 b 列にある数を、a と b の式で表しなさい。
1列2列3列4列5列
1行12345
2行678910
3行1112131415
4行1617181920
………………
1行に5個ずつ。各行の終わりは 5, 10, 15, 20…（5の倍数）になっている
見分けフロー③。これは「○行○列」と場所を聞く数表の型。カギは「各行のいちばん右（行の最後の数）」。1行の最後は5、2行の最後は10…＝5×(行番号)。そこから列の分を足したり引いたりして場所を出す。
①なに型?「4行3列」「何行何列」と場所(行と列)を聞いている。見分けフロー③で、これは数表の型と確定。等差や三角数ではなく、「行で大きくジャンプ、列で1ずつ進む」と考える。
②しくみをつかむ1行に5個。だから次の行へ行くと数は5だけ大きくジャンプする。各行の左はしは 1, 6, 11, 16…（前の行の左はし＋5）。各行の右はしは 5, 10, 15, 20…＝5×行番号。
③(1)を解く（基準から数える）4行を求める。4行の左はし（1列）は？　3行の右はしが15だから、4行1列は16。そこから 3列までは「16, 17, 18」と進む。4行3列＝18。式で言うと「4行1列(16)＋(3−1)＝16＋2＝18」。
④(2)を解く（48の場所）48が何行目かは「5でいくつ分か」で見る。5×9＝45、5×10＝50。45＜48≦50 だから48は10行目（9行までで45個使い切り、10行目に入る）。10行の左はし（1列）は45＋1＝46。46が1列、47が2列、48が3列。48は10行3列。
⑤(3)一般化の準備a 行 b 列をことばで：まず「a 行の1つ前の行(a−1行)までで何個使ったか」＝5×(a−1)個。その次から a 行が始まる。a 行 b 列はそこから b 個目だから、5×(a−1)＋b。
⑥(3)式にする＆検算a 行 b 列＝5×(a−1)＋b＝5a−5＋b。検算：4行3列なら5×4−5＋3＝20−5＋3＝18○（(1)と一致）。48の場所10行3列なら5×10−5＋3＝50−5＋3＝48○（(2)と一致）。式が(1)(2)の答えと合うので正しい。
⑦なぜこの型と見抜けた?決め手は「行と列という2つの場所」を聞いていること。1つの番目 n ではなく、たて(行)とよこ(列)の2方向がある。これが数表の型のサイン。考え方は「1行の個数×(行−1)＝前の行までの合計」＋「列の分」。1行が5個なら5、6個なら6に変わるだけで、しくみは同じ。
「○行○列」「表の□番目の行」など場所を2方向で聞かれたら数表の型。最初にやるのは「1行に何個並ぶか」を確認すること（ここでは5個）。次に「各行の右はし＝1行の個数×行番号」を出すと、どの数が何行目かがすぐ分かる。行をまたぐジャンプ幅＝1行の個数、というのが裏の仕組み。
a 行を 5a としない。5a は a 行の右はし(最後の数)。1列〜の途中はそこから引く必要がある。だから 5(a−1)＋b と、左はし基準で組む。48の行を 48÷5＝9.6 で「9行」としない。9行までで45個（5×9）使い、48はその次の10行目。「ちょうど5の倍数のときだけその行の右はし」と覚える。
9月大問4で「数を表に並べて、□行△列は何か」「ある数は何行何列か」はよく出る。1行の個数（5か6か7か）が変わっても、5(a−1)＋b の「5」をその数に変えるだけ。仕組みを覚えれば全部解ける。

問4★本番型：階段状にふやす（差が一定でない・2段階）L4目安 9分
問4おはじきを下の図のように、正方形の形になるよう、外側にL字（かぎ形）を1つずつ足して大きくしていきます。1番目は1個、2番目は4個、3番目は9個です。
(1) おはじきの数は1番進むごとに同じ数ずつ増えますか。増え方を調べ、n 番目のおはじきの数を n の式で表しなさい。
(2) 6番目は何個ですか。
(3) おはじきがちょうど100個になるのは何番目ですか。
うすい色＝そのとき新しく足したL字。1番目→2番目で＋3個、2番目→3番目で＋5個…と、足す数がちがう
見分けフロー①が最大のヤマ。1個・4個・9個の差を計算してみよう。差は同じ？　もし差が一定でなく増えていくなら、等差の型(差×n＋…)は使えない。「正方形に並べる」「L字を足す」「差が3,5,7と奇数で増える」がそろったら…？
番目 n12345…
おはじき(個)1491625…
前との差−3579…
①まず差をしらべる（ワナに注意）1番目1個、2番目4個、3番目9個。差は 4−1＝3、9−4＝5、16−9＝7。差が3,5,7…と一定でない！　ここが本番型のひっかけ。「差を出せば2n+1型でしょ」と決めつけると失敗する。差が一定でないので別の型を疑うのが正しい第一歩。
②2段階の見方：差の差差(3,5,7,9)をさらに見ると、5−3＝2、7−5＝2…と差の差が2で一定（奇数が1個ずつ）。これは「奇数を順に足していく」並びのサイン。1, 1＋3＝4, 1＋3＋5＝9, 1＋3＋5＋7＝16…。奇数を順に足すと平方数になる。
③図でも確かめる図を見ると、毎回まわりにL字を足して、つねに正方形になっている。1番目は1×1、2番目は2×2、3番目は3×3、4番目は4×4。だから個数は「1辺×1辺」。これで型が見える。
④式を作るn番目は n×n の正方形になる。だから n番目＝n×n＝n²。(1)の答え＝増え方は一定でない（3,5,7…と2ずつ増える）。式は n²（＝n×n）。
⑤検算n＝1:1×1＝1○、n＝2:2×2＝4○、n＝3:3×3＝9○、n＝4:4×4＝16○、n＝5:5×5＝25○。表とすべて一致。式は正しい。
⑥代入と逆算(2)代入：n＝6 → 6×6＝36個。(3)逆算：個数＝100 → n×n＝100 → 同じ数をかけて100になるのは 10×10＝100 だから n＝10、10番目。確かめ：10×10＝100○。（n²＝100 を「n＝100÷2＝50」などとしないこと。同じ数を2回かけて100になる数を探す。）
⑦なぜこの型と見抜けた?決め手は3つ。(1)差が一定でない（3,5,7と増える）→等差の型ではないと分かる。(2)差の差が2（奇数を足していく）→平方数のサイン。(3)図がいつも正方形（n×n）。この3つがそろえば n² の型。差が一定でないとき、あわてず「差の差」を見る・図の形を見るのが本番で差がつくポイント。差が一定でない＝難しいではなく、合計の型(三角数)か平方数の型のどちらか、と落ち着いて分けよう。
差が一定でないときの分かれ道：「合計・ぜんぶで」の言葉＋差がだんだん増える → 三角数 n(n+1)÷2（問2）。「正方形に並べる・L字を足す」＋差が3,5,7と奇数 → 平方数 n²（問4）。同じ「差が一定でない」でもこの2つは別物。図が正方形かどうか、合計を聞いているかどうかで見分ける。これがインターリーブ特訓のいちばんの収穫。
差を見て3,5,7だから「差×n」と一次の式を作らない。差が一定でないとき一次の式(an+b)は絶対に合わない。逆算 n²＝100 を n＝50 としない（÷2ではない）。同じ数を2回かけて100＝10×10だから n＝10。(1)で「増え方は同じ」と答えない。差は3,5,7…と変わる、が正しい。
