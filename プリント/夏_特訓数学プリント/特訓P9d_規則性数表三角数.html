規則性④｜三角数・数表・階段で「nの式」を作って大きい数を逆算する

夏・特訓対策　目安：約30分（問1〜問4・通し番号）　対象: 中1　全問に「あほでもわかる」徹底解説つき

9月の特訓選抜・大問4はタイル以外の別パターンでも出る：三角数・カレンダー・階段・数表。やることは同じで①数えて表 → ②差や仕組みを見る → ③nの式 → ④大きい数を代入/逆算。きみのいちばんの伸びしろ＝規則性。今日は図と表をたっぷり使って、4つの別パターンを1つずつゆっくり通すよ。式さえ作れれば100番目でも一瞬だよ。

  1. ① 1・2・3番目を表に書く（図を数えて具体を並べる）
  2. ② 仕組みを見る（毎回いくつ増える？　たて×よこ？　中央の何倍？）
  3. ③ nの式を作る：差が一定なら差×n＋（1番目−差）。形がある時は「逆さに重ねる・全体から引く」など図の工夫で式にする
  4. ④ 代入と逆算：「○番目はいくつ？」は代入。「いくつになるのは何番目？」は式＝○の方程式
  5. ⑤ 確かめ：n＝1,2,3 や出た答えを式に入れて、表・図と合うか必ずチェック
問1三角数：●を三角に積む → n(n+1)/2 L3-L4目安 8分
問1下の図のように、●を三角の形に1段ずつ増やして積んでいきます。1段目は1個、2段目までで3個、3段目までで6個…。このように積んだ●の合計を「三角数」といいます。
●を1, 2, 3, 4 …個ずつ下に足していく。合計の数が「三角数」
(1) 5段目までの●の合計は何個ですか。
(2) n段目までの●の合計を n の式で表しなさい。
(3) 10段目までの●の合計は何個ですか。((2)の式を使う)
(1)は表をのばす。(2)が今日のヤマ。三角の●を「もう1つ逆さに重ねる」と長方形になるよ（解説の図を見て！）。(3)は(2)の式に n＝10 を入れるだけ。

① やさしい言い換え「下の段ほど●が1個ずつ多い三角の山。何段かまで積んだとき、●はぜんぶで何個？」を聞いている問題だよ。
② わかってること／求めることわかってる→1段目1個、2段目までで3個、3段目までで6個（毎回足す数が 1→2→3→4 と1ずつ増える）。求めるのは→(1)5段目の合計、(2)n段目の式、(3)10段目の合計。
③ 図でイメージ4段目までの三角（合計10個）を思いうかべて。これとまったく同じ三角を逆さま（上下ひっくり返し）にして、右にぴったりくっつけると、たてが4個・横が5個の長方形になる！4×5＝20個。長方形は三角2つ分だから、三角1つ分は 20÷2＝10個。表と合う。
④ 作戦＋たとえ話三角は数えにくいけど、逆さの三角をもう1つ重ねれば長方形（=たて×よこ）になる。長方形なら一瞬で数えられる。あとで半分にすればいい。＝「めんどうな三角は、ペアにして長方形に化けさせる」作戦。これが三角数の魔法。
⑤ 1ステップずつ（なぜ付き） ・(1) 表を1つのばす。5段目は「4段目まで10個」に「5個」を足す→10＋5＝15個。なぜ5個？ n段目には●が n 個あるから、5段目には5個。
・(2) n段目までの三角を、同じ三角の逆さでペアにすると、たて n 個・横 (n＋1) 個の長方形。なぜ横が n＋1？　逆さに重ねると、どの段も「●の数＋すきま」がそろって n＋1 個になるから。長方形＝n×(n＋1)。三角1つ分はその半分→n(n＋1)÷2。
・(3) (2)の式に n＝10 を代入→10×(10＋1)÷2＝10×11÷2＝110÷2＝55個。
(1)5段目まで＝10＋5＝15個。
(2)逆さ重ねで長方形 n×(n＋1)、その半分→n(n＋1)÷2（個）。
(3)n＝10：10×11÷2＝110÷2＝55個。
⑥ よくある間違い（赤・先出し）「n×n」や「n×n÷2」で答えない。横は n ではなく n＋1（逆さに重ねると1個ずれて長くなる）。÷2を忘れない＝長方形は三角2つ分だから、必ず半分にする。「5段目だけの●の数(=5個)」と「5段目までの合計(=15個)」を混同しない。聞かれているのは合計。
⑦ 検算＋次に見抜くコツ検算→n＝1:1×2÷2＝1○、n＝2:2×3÷2＝3○、n＝3:3×4÷2＝6○。表とぴったり。見抜くコツ＝「1,3,6,10,15…」や「足す数が1ずつ増える並び」を見たら三角数。n(n＋1)÷2 を思い出す。
三角数は「逆さに重ねて長方形 n×(n＋1)、その半分」。公式を丸暗記せず、長方形に化けさせて半分にする仕組みで思い出せる。
9月大問4で「●や○を三角に積む／1,3,6,10…と並ぶ」型が出たら、この逆さ重ねで即 n(n＋1)÷2 を作れる。タイルと並ぶ規則性の最重要パターン。

問2カレンダーの十字の和 = 中央×5 L3目安 7分
問2下のカレンダーで、ある日付を中央にして、その上・下・左・右・中央の5つの数を選び、十字（プラスの形）に囲みます。例として中央が9のとき、十字の5つの数の和を考えます。
赤い十字＝中央(9)と、その上(2)・下(16)・左(8)・右(10)の5つ。カレンダーは横が7列
(1) 中央が9のとき、十字の5つの数の和を求めなさい。
(2) 中央の数を a とするとき、十字の5つの数の和を a の式で表しなさい。
(3) 十字の5つの数の和が80になるのは、中央がいくつのときですか。
カレンダーは横7列。だから「上は中央−7」「下は中央＋7」「左は中央−1」「右は中央＋1」。これを全部たすとどうなる…？
① やさしい言い換え「カレンダーで十字に5つ選ぶと、その合計はいつも中央の何倍になる？」を探す問題だよ。
② わかってること／求めることわかってる→カレンダーは横7列。中央の右どなりは＋1、左は−1、1つ下は＋7、1つ上は−7。求めるのは→(1)中央9の和、(2)中央 a のときの式、(3)和が80になる中央。
③ 図でイメージ中央を a とすると、まわりは：上＝a−7、下＝a＋7、左＝a−1、右＝a＋1。十字に並べると、中央 a を真ん中に、上下・左右がペアになっている。
④ 作戦＋たとえ話5つをそのまま足すと、＋7と−7が打ち消し、＋1と−1も打ち消す。残るのは a が5個ぶん。＝「シーソーのように、＋とーが同じ大きさなら消える」。だから和は中央の5倍。
⑤ 1ステップずつ（なぜ付き） ・(2) 和＝中央＋上＋下＋左＋右＝a＋(a−7)＋(a＋7)＋(a−1)＋(a＋1)。なぜ−7と＋7？　1つ上は1週間前で7小さい、1つ下は1週間後で7大きいから。a を数えると5個。−7＋7＝0、−1＋1＝0で消える→5a。
・(1) (2)に a＝9 を代入→5×9＝45。または9＋2＋16＋8＋10＝45で確認。
・(3) 和＝80 だから 5a＝80、両辺÷5→a＝16。
(1)9＋2＋16＋8＋10＝45（＝5×9）。
(2)和＝a＋(a−7)＋(a＋7)＋(a−1)＋(a＋1)＝5a。
(3)5a＝80 → a＝80÷5＝16（中央が16）。
⑥ よくある間違い（赤・先出し）上下を ±1、左右を ±7 と逆にしない＝1つ上下は「1週間＝7」動く、左右が「1日＝1」。和を a＋5 にしない＝「a が5個」なので 5a（かけ算）。(3)で 80−5 などにしない＝5a＝80 は割り算 80÷5。
⑦ 検算＋次に見抜くコツ検算→中央16なら上9・下23・左15・右17。9＋23＋15＋17＋16＝80○。見抜くコツ＝表のマス目で「となりは＋1、1つ下は＋（列の数）」。十字や四角で囲む問題は、中央を文字にして「まわりを中央＋何」で書けば必ず解ける。
表のマス目は「右へ＋1、下へ＋7（列の数だけ）」。中央を文字にしてまわりを「中央±いくつ」で書くのが数表問題の必殺技。

① やさしい言い換え「1段ごとに2個ずつ多くなる階段を n 段ぜんぶ積んだら、ブロックは合計で何個？」を聞いているよ。
② わかってること／求めることわかってる→各段は1,3,5…（奇数、2個ずつ増える）。合計は1,4,9…。求めるのは→(1)4番目の合計、(2)合計の n の式、(3)合計100になる番目。
③ 図でイメージ3番目の階段（合計9個）をよく見て。じつはこれ、たて3個・横3個の正方形に並べ替えられる（階段の出っぱりを反対側のへこみにはめると四角になる）。3×3＝9。2番目なら2×2＝4。つまり合計は「番目の数を2回かけた数（正方形の数）」。
④ 作戦＋たとえ話「1＋3＋5＋…＋（奇数）」の合計は、いつもピッタリ正方形になる（奇数を足していくと四角がどんどん大きくなる）。＝「奇数を順に足すのは、四角形を1まわりずつ大きくしていくのと同じ」。だから合計＝n×n。
⑤ 1ステップずつ（なぜ付き） ・(1) 4番目は3番目の合計9に「4段目の7個」を足す→9＋7＝16個。なぜ7個？　n段目の個数は奇数 2n−1。4段目は2×4−1＝7。
・(2) 合計の並び 1,4,9,16 は 1²,2²,3²,4²。だから n 番目の合計＝n×n＝n²。
・(3) 合計＝100 だから n²＝100。同じ数を2回かけて100になるのは10×10。だからn＝10（10番目）。
(1)9＋7＝16個。
(2)合計は 1,4,9,16＝1²,2²,3²,4²。よって n²（個）。
(3)n²＝100、10×10＝100 だから 10番目。
⑥ よくある間違い（赤・先出し）合計を「各段の個数 2n−1」と混同しない＝2n−1は「その段だけ」、合計は n²。聞かれているのは合計。(3)を 100÷2 などにしない＝n²＝100 は「2回かけて100」＝10。n²＝100 を n＝50 にしない（50×50は2500）。
⑦ 検算＋次に見抜くコツ検算→n＝1:1²＝1○、n＝2:2²＝4○、n＝3:3²＝9○、n＝4:4²＝16○。見抜くコツ＝合計の並びが「1,4,9,16,25…」なら正方形数 n²。各段が奇数（1,3,5…）で積む形は、ほぼ n² の合図。
奇数 1,3,5,7… を順に足した合計は、いつも n²（正方形）。階段は四角に並べ替えられる、と図で覚える。

問4★本番型：数を表に並べ「○行○列の数」を出す L3-L4目安 8分
問4下の表のように、1, 2, 3, … と数を左から順に、1行に7個ずつ並べていきます。
　1列2列3列4列5列6列7列
1行1234567

(2) r 行 c 列の数を、r と c の式で表しなさい。
(3) 数の50は、何行何列にありますか。
各行は7個ずつ。だから「○行の1列目」は、前の行までで使った個数＋1。たとえば3行1列は、2行ぶん(7×2=14個)の次だから15。式は「7×(行−1)＋列」になりそう…？
① やさしい言い換え「数を7個ずつ折り返して並べた表で、ある場所(行・列)の数を当てる／逆にその数の場所を当てる」問題だよ。
② わかってること／求めることわかってる→1行に7個ずつ。1行目は1〜7、2行目は8〜14…。求めるのは→(1)3行4列の数、(2)r行c列の式、(3)数50の場所。
③ 図でイメージ各行のおわりの数(右はし)に注目：1行目=7、2行目=14、3行目=21…＝7の倍数。だから「r 行の1つ前の行まで」で 7×(r−1) 個つかっている。r 行の c 列目は、そこから c 個進んだ数。
④ 作戦＋たとえ話本のページのよう：「前のページぶんを全部数えて、今のページで何個目か足す」。＝(前の行までの個数)＋(今の行で何列目か)。前の行までは 7×(r−1)、今の行は c。
⑤ 1ステップずつ（なぜ付き） ・(1) 3行1列は「2行ぶん終わった次」＝7×2＝14 の次の15。4列目はそこから3つ進んで 15,16,17,18。式でいうと 7×(3−1)＋4＝14＋4＝18。
・(2) なぜ7×(r−1)？　r 行より前の行は (r−1) 行あり、各行7個だから 7×(r−1) 個つかい終わっている。そこから c 列目なので＋c。式＝7(r−1)＋c。
・(3) 数50の場所。50 を「7でわって、何行ぶんか」を見る。50÷7＝7あまり1。7行ぴったり終わると 7×7＝49。49の次の50は、8行目の1列目。＝8行1列。式で確認：7×(8−1)＋1＝49＋1＝50○。
(1)7×(3−1)＋4＝14＋4＝18。
(2)r行c列の数＝7(r−1)＋c。
(3)49＝7×7（7行目の右はし）。次の50は8行目の1列目＝8行1列。
⑥ よくある間違い（赤・先出し）7×r としない＝前の行は (r−1) 行ぶん。7×r だとその行の右はしを越えてしまう。(3)で「50÷7＝7あまり1 だから7行1列」としない＝7行目の右はしは49で、50はもう次の行。あまりが出たらその行は1つ進む。あまりが0のとき（例:49）は行をくり上げない＝49は7行7列でちょうど。
⑦ 検算＋次に見抜くコツ検算→(1)表の3行4列を見ると18○。(3)8行1列を式へ：7×7＋1＝50○。見抜くコツ＝「1行に□個ずつ並ぶ表」を見たら、式は□×(行−1)＋列。場所→数は代入、数→場所は「□でわってあまりを見る」。
7個ずつの表は 「7×(行−1)＋列」。逆に数から場所は「7でわって商とあまり」。あまりが出たら行が1つ進む、あまり0ならその行の右はし。
