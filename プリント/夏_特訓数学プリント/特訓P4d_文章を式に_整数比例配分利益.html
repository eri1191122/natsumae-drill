問1連続する3つの整数（中央を x にする型）L2 標準目安 7分
問1連続する3つの整数があります。その3つの和は72です。この3つの整数を求めなさい。
まず「何型か」を考えよう。何を x にするかは自分で決めてノートに書こう。
連続する整数は 真ん中を x におくのがコツ。前後が x−1・x+1 になり、足すと −1 と +1 が消えて式がすっきりする。
①x真ん中の整数を x とする → 前は (x−1)、後は (x+1)。
（x−1） ─ （x）─ （x+1） ↑前 ↑真ん中 ↑後 3つの和 ＝ (x−1) ＋ x ＋ (x+1)
③式3つの和＝72。(x−1)＋x＋(x+1)＝72。
④解x−1＋x＋x+1＝72 → 3x＝72 → x＝24。
⑤検算真ん中24なら 23・24・25。23＋24＋25＝72○。
いちばん小さい数を x にしてもよいが、その時は x＋(x+1)＋(x+2)＝72 になる。どこを x にしたか最初に決めて書くこと。真ん中を x にすると計算がいちばんラク。
「連続する数＝真ん中を x」は特訓選抜 大問3の定番。連続する偶数・奇数（2ずつ増える）も同じ考え方でいける。
こたえ23、24、25

問2★2けたの数（位の数・入れかえ）L3 応用目安 8分
問22けたの整数があります。十の位の数と一の位の数の和は9です。十の位と一の位を入れかえてできる数は、元の数より27大きくなります。元の数を求めなさい。
🔧 道具を確認する（まず自分で考えてから開こう）
2けたの数：十の位が a、一の位が b なら 10a＋b
まず「何型か」を考えよう。何を x にするかは自分で決めてノートに書こう。
2けたの数のしくみ：十の位 a・一の位 b の数は 10a＋b。例 36 ＝ 10×3＋6。位の数をそのままならべて書かない（×ab、○ 10a＋b）。
①x十の位を x とする → 和が9だから一の位は (9−x)。
元の数 ＝ 10×x ＋ (9−x) ＝ 9x＋9 入れかえ＝ 10×(9−x) ＋ x ＝ 90−9x （十の位と一の位が入れかわる）
③式入れかえ＝元より27大きい。90−9x＝(9x＋9)＋27。
④解90−9x＝9x＋36 → 90−36＝9x＋9x → 54＝18x → x＝3。
⑤検算十の位3、一の位(9−3)＝6 → 元は36。入れかえると63。63−36＝27○。位の和3＋6＝9○。
「27大きい」のは入れかえた方。元＋27＝入れかえ、と置く。どちらが大きいか文章をよく読む。36 を 3×6 や 3＋6 と書かない＝10×3＋6 が正しい。
「2けた＝10a＋b」は特訓選抜 大問3で出やすい難所。位の表し方を知っているかどうかだけで差がつく。
こたえ元の数 36

問3★比で分ける（比例配分）L2-L3 標準目安 7分
問3きみと妹の所持金の比は 3:5 です。2人の所持金の合計は4800円です。きみと妹の所持金はそれぞれいくらですか。
まず「何型か」を考えよう。何を x にするかは自分で決めてノートに書こう。
比で分けるコツ：1つ分（かたまり）を x におく。3:5 なら 3x と 5x。これなら文字は1つで済み、足すと合計になる。
①x1かたまりを x 円とする → きみ＝3x 円、妹＝5x 円。
きみ |■■■| ＝ 3x 妹 |■■■■■| ＝ 5x 合計 |■■■■■■■■| ＝ 8x ＝ 4800
③式合計＝4800。3x＋5x＝4800。
④解8x＝4800 → x＝600。
⑤検算きみ＝3×600＝1800円、妹＝5×600＝3000円。合計1800＋3000＝4800○。比は1800:3000＝3:5○。
x がそのまま答えではない。x は「1かたまり＝600円」で、聞かれたのは 3x と 5x。最後に必ず 3x・5x を計算する。
「比は1つ分を x」は特訓選抜 大問3の割合・比の問題に直結。a:b でも同じく ax と bx でいける。

問4★原価・定価・利益（割合）L3 応用目安 8分
問4ある品物の原価（仕入れねだん）に3割の利益を見こんで定価をつけました。ところが定価から200円引きで売ったので、利益は400円でした。この品物の原価を求めなさい。
まず「何型か」を考えよう。何を x にするかは自分で決めてノートに書こう。
割合のコツ：3割＝0.3。「3割の利益を見こむ」＝原価＋原価×0.3＝原価×1.3。利益＝売ったねだん−原価。この2つの言葉を式に直すだけ。
①x原価を x 円とする。
原価 ＝ x 定価 ＝ x×1.3 ＝ 1.3x （3割の利益を見こむ） 売りねだん＝ 1.3x − 200 （200円引き） 利益 ＝ 売りねだん − 原価 ＝ 400
③式利益＝400。(1.3x−200)−x＝400。
④解1.3x−200−x＝400 → 0.3x＝600 → x＝2000。
⑤検算原価2000円 → 定価2000×1.3＝2600円。200円引きで2400円で売った。利益＝2400−2000＝400○。
「3割の利益」を ＋3 や ×3 としない＝×1.3（原価の1.3倍）。利益は「売値−原価」で、定価−原価ではない（実際に売ったねだんで考える）。
原価・定価・利益は9月特訓選抜 大問3(4)の「割合・利益」と同じ根っこ。「○割＝×(1＋0.○)」を体に入れておけば、値引きや消費税の問題も同じ手で解ける。
