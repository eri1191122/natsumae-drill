特訓P1b数学：正負の数計算精度ドリル（大問1対策）L1-L2

正負の数 計算精度ドリル｜符号を1つも落とさず「速く・正確に」

夏・特訓対策　目安：約30分（問1〜問18・通し番号）　対象: 中1　全問に簡潔な解説つき

これは計算の正確さドリル。9月特訓選抜の大問1（計算）の土台になる。きみがいちばん落としやすいのは符号（＋−のしるし）。1問ずつ途中式を書く欄に手で書いて、確かめ欄で符号をもう一度声に出してたしかめよう。スピードより「符号を1つも落とさない」が今日の目標。

  1. ① 符号は数とセット。「−4」の − は数の一部。途中で書き写すとき必ず一緒に運ぶ。
  2. ② 計算の順番＝かっこ → 累乗 → ×÷ → ＋−。この順を破ると符号がずれる。
  3. ③ 同符号は＋・異符号は−（かけ算わり算は−が偶数個→＋・奇数個→−）。
  4. ④ 答えが出たら符号だけ最後にもう一度チェック（＋か−か、声に出す）。
大A加減（たし算・ひき算）L1 基礎目安 5分・4問
次の計算をしよう。符号を落とさないこと。
問1 (−7)＋(−5)
問2 (＋6)−(−9)
問3 (−8)＋(＋3)
問4 (−4)−(−10)＋(−6)
たし算は数直線で動くこと。＋の数は右へ、−の数は左へ動く。(−7)＋(−5) は「7だけ左、さらに5だけ左」だから合わせて12だけ左＝−12。ひき算は「反対の数をたす」に直すのが安全：−(−9)＝＋9（マイナスのマイナスはプラス）。
−12 −7 0 ←──●←─────●─────────|→ ←5← ←7← （左へ7、さらに左へ5）
同じ符号どうしのたし算＝符号はそのまま・数はたす。ちがう符号どうし＝符号は大きい方・数はひく。ひき算は −(−▲)＝＋▲ に直してから。
問1同符号(−どうし)＝そのまま−で数をたす：7＋5＝12 → −12。
問2・★符号−(−9)＝＋9 に直す → 6＋9 → ＋15。
問3異符号＝大きい方(−8)の符号・数はひく：8−3＝5 → −5。
問4・★符号−(−10)＝＋10 に直す → −4＋10−6＝(−4−6)＋10＝−10＋10 → 0。
問2 −(−9) を −9 のままにしない＝マイナスのマイナスは＋9。問4 −4 の頭の − を書き写し忘れない。マイナスどうしを先にまとめると落としにくい。0 に＋−はつけない。
こたえ問1 −12／問2 ＋15／問3 −5／問4 0

大B乗除（かけ算・わり算）L1→L2 基礎目安 6分・4問
次の計算をしよう。まず符号を決めてから数を計算。
問5 (−6)×(＋4)
問6 (−8)÷(−2)
問7 (−3)×(−5)×(−2)
問8 (＋24)÷(−6)
かけ算・わり算の符号は「−の個数」で決まる。−を1個かけるたびに向きが反対になる。だから −が偶数個（0,2,4…）なら＋・奇数個（1,3…）なら−。数の大きさ（絶対値）は符号と別に、ふつうにかける・わるだけ。
手順は①先に符号を決める（−の数をかぞえる）→②数だけ計算→③符号をつける。これで符号落ちが激減する。
問5−が1個(奇数)→符号−。6×4＝24 → −24。
問6・★同符号−が2個(偶数)→符号＋。8÷2＝4 → ＋4（−÷−＝＋）。
問7・★3つ−が3個(奇数)→符号−。3×5×2＝30 → −30。
問8−が1個(奇数)→符号−。24÷6＝4 → −4。
問6 (−8)÷(−2)＝−4 としない＝−が2個だから＋4。問7 −が3個あるのを「2個まで」で止めない＝3個全部かぞえて奇数→−。「符号を先に・声に出してかぞえる」が一番落ちない。
こたえ問5 −24／問6 ＋4／問7 −30／問8 −4

大C★累乗（−4² と (−4)² のちがい）L2 ここが山目安 7分・4問
かっこに気をつけて累乗を計算しよう。「何を2回かけるか」を見きわめる。
問9 (−2)²
問10 −4²
問11 −(−3)²
問12 (−2)³
累乗の右上の数（指数）は「すぐ左にあるものを何回かけるか」を表す。かっこがあればかっこごと、なければその数だけが2乗される。
・(−4)²＝(−4)×(−4)＝＋16（−が2個→＋）
・−4²＝−(4×4)＝−16（2乗されるのは4だけ。頭の−は最後まで残る）
このちがいが累乗の符号落ちで一番多いミス。
2乗（指数が偶数）→ かっこ付きなら必ず＋。3乗（指数が奇数）→ −のまま。まず「かっこの中まで2回かけるのか／数だけか」を指さして確認。
問9(−2)²＝(−2)×(−2)。−が2個→＋。2×2＝4 → ＋4。
問10・★最重要−4²＝−(4×4)＝−(16)。2乗されるのは4だけ、頭の−は残る → −16。
問11・★二重−(−3)²。まず(−3)²＝＋9、その前の−をつけて → −9。
問12・★3乗(−2)³＝(−2)×(−2)×(−2)。−が3個(奇数)→−。2×2×2＝8 → −8。
−4²＝16 としない＝かっこが無いので4だけが2乗、答えは−16。(−2)²＝−4 としない＝かっこ付きの2乗は＋4。問11 −(−3)²＝9 としない＝(−3)²は＋9、その前の−で−9。「かっこの有無」を毎回指さして確認。
こたえ問9 ＋4／問10 −16／問11 −9／問12 −8

大D★四則混合（かっこ→累乗→×÷→＋−）L2 応用目安 7分・4問
計算の順番を守ろう。先にやる所に下線を引いてから計算。
問13 5−2×(−4)
問14 (−3)²−2×5
問15 −2²＋(−3)×(−4)
問16 24÷(−4)＋(−2)×3
計算には決まった順番がある：かっこ → 累乗 → ×÷ → ＋−。たとえば 5−2×(−4) を左から順に「5−2＝3、3×(−4)」とやると間違い。先に × をやる：2×(−4)＝−8 → 5−(−8)＝5＋8＝13。×÷でできた数の符号も、そのまま＋−に持ちこむ。
手順：①累乗を先に処理→②×÷を処理（符号つきのまま）→③最後に＋−。×÷で出た−を＋−のところで落とさないのがカギ。
問13・★順番先に×：2×(−4)＝−8 → 5−(−8)＝5＋8 → 13。
問14先に累乗：(−3)²＝＋9。次に×：2×5＝10 → 9−10 → −1。
問15・★累乗の符号−2²＝−4（4だけ2乗）。(−3)×(−4)＝＋12 → −4＋12 → 8。
問16÷を先に：24÷(−4)＝−6。×も先に：(−2)×3＝−6。残った＋−を計算：−6＋(−6)＝−6−6 → −12。
問13 5−2×(−4) を (5−2)×(−4) としない＝×が先。問15 −2² を ＋4 としない＝かっこ無しは−4。問16 ×÷の符号を＋−のところで落とさない。先にやる所に下線を引くと順番ミスが消える。
こたえ問13 13／問14 −1／問15 8／問16 −12

大E分数の正負（通分して符号つきで計算）L2 標準目安 5分・2問
通分して、符号をつけたまま計算しよう。
問17 −12＋−13
問18 −34−−12
分数も整数と同じルール。分母をそろえて（通分して）から、分子どうしを符号つきでたし算・ひき算する。整数のときに学んだ「ひき算は反対の数をたす」「同符号はそのまま・異符号はひく」がそのまま使える。分子の−を通分のとき落とさないのがポイント。
①分母の最小公倍数で通分→②分子を符号つきで計算→③符号をつけて答え。ひき算の −(−▲) は＋▲に直してから。
問17分母2,3の最小公倍数6で通分：−36＋−26。同符号(−どうし)→そのまま−で分子をたす：3＋2＝5 → −56。
問18・★符号後ろの −−12 ＝ ＋12 に直す。分母4,2の最小公倍数4で通分：−34＋24。異符号→大きい方(−)・分子はひく：3−2＝1 → −14。
問17 通分のとき分子の−を落とさない＝−12＝−36（−3、＋3ではない）。問18 −−12 を −12 のままにしない＝マイナスのマイナスで＋。
こたえ問17 −56／問18 −14
