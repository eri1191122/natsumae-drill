特訓P13数学：方程式の利用型あて道場① L2-L3

型あて道場①｜「これは何型？」を自分で見抜く特訓

夏・特訓対策　目安：約30分（第1問〜第6問）　対象: 中1　全問に徹底解説つき

今日は見出しに型名を出さない。きみが自分で「これは何型か」を見抜くのが今日の主役。本番の大問3も、最初に型を当てられるかどうかで決まる。数字はいつも変わるから暗記はきかない。だから「どの言葉でその型と分かるか」を体で覚えよう。最後の第6問はひっかけ。あわてず言葉をよく読もう🌸

  1. ① これは何型？ をまず書く（下の4つから選ぶか、自分の言葉で）
    　⑴ 個数・代金型　⑵ 速さ・往復型　⑶ 過不足型　⑷ 割合・利益型
  2. ② x を置く（「□を x とする」と書く）
  3. ③ 式を作る（＝で結べる2つの言い方を探す）
  4. ④ 解く（移項して x ＝ の形へ）
  5. ⑤ 検算（出た数を文章に戻して合うか確かめる）
  6. 最後に 型が当たったかも ○✕ をつける
第1問L2 標準目安 5分
第1問1個70円のチョコと、1個40円のあめを、合わせて15個買ったら、代金は810円でした。チョコは何個買いましたか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。
個数・代金型。「合わせて15個」と合計の個数が決まっているのが見分け方。片方を x、もう片方は「合計−x」で表すと文字は1つで済む。
①xチョコを x 個とする → あめは (15−x) 個。
③式金額の合計＝810。70x＋40(15−x)＝810。
④解70x＋600−40x＝810 → 30x＝210 → x＝7。
⑤検算チョコ7個、あめ(15−7)＝8個。70×7＋40×8＝490＋320＝810○。
40(15−x) を 40×15−x としない（x にも40をかける＝600−40x）。最後はチョコを答える（x がそのまま答え）。
こたえチョコ 7個（あめ 8個）

第2問L3 標準目安 5分
第2問色紙を子どもたちに配ります。1人に4枚ずつ配ると15枚あまり、1人に6枚ずつ配ると5枚たりません。子どもの人数は何人ですか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。
過不足型。配り方が2通りで、「あまる」「たりない」の言葉が見分け方。変わらないのは色紙の数。これを2通りで表して＝で結ぶ。あまる＝＋、たりない＝−。
①x子どもの人数を x 人とする。
4枚ずつ：4x ＋ あまり15 → 色紙＝4x＋15 6枚ずつ：6x − たりない5 → 色紙＝6x−5
③式色紙の数を2通りで：4枚ずつ＝4x＋15（あまる＝＋）、6枚ずつ＝6x−5（たりない＝−）。同じ色紙だから 4x＋15＝6x−5。
④解15＋5＝6x−4x → 20＝2x → x＝10。
⑤検算子ども10人。4枚ずつ：4×10＋15＝55枚。6枚ずつ：6×10−5＝55枚。両方55枚で一致○。
「たりない5」を ＋5 にしない（足りない＝引く。6x−5）。あまる＝＋、たりない＝−を取りちがえると符号がずれる。
「変わらない全体を2通りで表して＝」は大問3(3)の定番。長椅子・部屋割り・お金でもそのまま使える。
こたえ子ども 10人

第3問L3 標準目安 5分
第3問ある品物に、原価の2割の利益を見込んで定価をつけたところ、定価は1800円になりました。この品物の原価はいくらですか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。
割合・利益型。「○割の利益」「定価」の言葉が見分け方。2割の利益（2割増し）＝×(1＋0.2)＝×1.2。定価＝原価×1.2。
①x原価を x 円とする。
原価 x 円 ↓ 2割の利益を見込む（×1.2） 定価 1.2x 円 ＝ 1800円
③式定価＝原価×1.2だから 1.2x＝1800。
④解x＝1800÷1.2＝1500。
⑤検算原価1500円。1500×1.2＝1800円＝定価○。
1800−2割＝1440 としない（割るのは1.2。1800÷1.2＝1500）。2割の利益は×1.2であって、定価から2割ひくのではない。
「×(1＋割合)で割増し」は大問3(4)の利益問題の核。割合は小数か分数で正確に。
こたえ原価 1500円

第4問L3 標準目安 5分
第4問家から駅まで、行きは分速50m、帰りは分速75mで往復しました。往復にかかった時間は合わせて25分でした。家から駅までの片道の道のりは何mですか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。
速さ・往復型。行きと帰りで速さがちがい、時間の合計が分かっているのが見分け方。等しい関係は「行きの時間＋帰りの時間＝25分」。時間＝道のり÷速さを使う。
①x片道の道のりを x m とする（行きも帰りも同じ道）。
③式行きの時間＝x÷50、帰りの時間＝x÷75。合計が25分だから x/50＋x/75＝25。
④解両辺を150倍（50と75の最小公倍数）→ 3x＋2x＝3750 → 5x＝3750 → x＝750。
⑤検算行き 750÷50＝15分、帰り 750÷75＝10分。15＋10＝25分○。
行きと帰りの速さを足して平均にしない（(50＋75)÷2 は使えない）。時間＝道のり÷速さ。割る向きをまちがえないこと。
「時間の合計＝行き÷速さ＋帰り÷速さ」は大問3(2)で頻出。分数の式は最小公倍数を両辺にかけて整数にすると楽。
こたえ片道 750m

第5問L3 標準目安 5分
第5問生徒が長椅子に座ります。1脚に4人ずつ座ると10人がすわれません。1脚に6人ずつ座ると、ちょうど全員が座れます。長椅子は何脚あり、生徒は何人ですか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。
過不足型。座らせ方が2通りで、「すわれない（あまる）」がある。「ちょうど全員＝過不足ゼロ」も同じ過不足型。変わらない「生徒の人数」を2通りで表して＝で結ぶ。
①x長椅子の脚数を x 脚とする。
4人ずつ：4x ＋ すわれない10 → 生徒＝4x＋10 6人ずつ：6x （ちょうど・あまり無し）→ 生徒＝6x
③式生徒数を2通りで：4人ずつ＝4x＋10（すわれない＝＋）、6人ずつ＝6x（ちょうど）。同じ生徒だから 4x＋10＝6x。
④解10＝6x−4x → 10＝2x → x＝5。
⑤検算脚数5脚。生徒＝4×5＋10＝30人。6人ずつ：6×5＝30人でちょうど一致○。
6人ずつは「ちょうど」だからあまりは無い（6x＋何かにしない。6x のまま）。「すわれない10人」は人があまる＝＋10。
「ちょうど座れた」も過不足ゼロとして同じ型でいける。大問3(3)の長椅子はこの形が定番。
こたえ長椅子 5脚／生徒 30人

第6問 ★L3 標準（ひっかけ注意）目安 5分
第6問1個90円のプリンを何個かと、1本150円のジュースを2本買ったら、代金は1020円でした。プリンは何個買いましたか。
まず①これは何型？を書く（⑴個数・代金 ⑵速さ・往復 ⑶過不足 ⑷割合・利益 から選ぶか自分の言葉で）。型を決めたら②x→③式→④解く→⑤検算。※よく読もう。ひっかけがあるよ。
個数・代金型（ただし合計の個数が無い素直版）。ジュースは2本と決まっているから「合計−x」は使わない。変わる数（プリン）だけが x、ジュースは150×2＝300円で固定。
①xプリンを x 個とする（ジュースは2本で固定）。
プリン 90×x ＝ 90x 円 ジュース 150×2 ＝ 300 円（固定） 合計 90x＋300 ＝ 1020 円
③式代金の合計＝1020。90x＋150×2＝1020 → 90x＋300＝1020。
④解90x＝1020−300＝720 → x＝720÷90＝8。
⑤検算プリン8個。90×8＋150×2＝720＋300＝1020○。
「合計の個数」が与えられているか必ず確認。この問題には「合わせて○個」が無いので、ジュースを「○−x」とは置けない。ジュースは2本固定だから＋300、変わるプリンだけを x にする。
2種類でも、合計の個数があれば「合計−x」、無ければ固定分は数字のまま。どちらかを見抜くのが大問3(1)の落とし穴。
こたえプリン 8個
