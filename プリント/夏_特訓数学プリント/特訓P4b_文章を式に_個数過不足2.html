夏・特訓対策　目安：約30分（問1〜問4・通し番号）　対象: 中1　全問に徹底解説つき

これがきみのいちばんの伸びしろ＝「文章を式にする」力。全国統一テストで0/25だったのもここ。今日は4問だけ。そのぶん1問ずつ型カードにそって途中式をていねいに書こう。数をこなすより「型を体に入れる」のが今日の目標。

🔧 道具を確認する（まず自分で考えてから開こう）
  1. ① 何を求める？ → それを x とおく（「□を x とする」と書く）
  2. ② 図か表にする（線分図・表）＝目で見える形に
  3. ③ 等しい関係を式に（＝で結べる2つの言い方を探す）
  4. ④ 解く（移項して x ＝ の形へ）
  5. ⑤ 検算（出た数を文章に戻して合うか確かめる）
問12種類の個数（合計が決まっている）L2 標準目安 7分
問184円切手と63円切手を合わせて20枚買ったら、代金は1470円でした。84円切手と63円切手をそれぞれ何枚買いましたか。
まず「何型か」を考えよう。何を x にするかは自分で決めて、表か図をかいてから式を作ろう。
2種類で合計枚数が決まっているとき：片方を x、もう片方は「合計−x」で表す。これで文字は1つで済む。
①x84円切手を x 枚とする → 63円切手は (20−x) 枚。
┌──────┬───────┬─────────┐ │ │ 84円 │ 63円 │ ├──────┼───────┼─────────┤ │ 枚数 │ x │ 20−x │ │ 金額 │ 84x │ 63(20−x) │ └──────┴───────┴─────────┘
③式金額の合計＝1470。84x＋63(20−x)＝1470。
④解84x＋1260−63x＝1470 → 21x＝210 → x＝10。
⑤検算84円が10枚、63円が(20−10)＝10枚。84×10＋63×10＝840＋630＝1470○。
63(20−x) を 63×20−x としない（＝1260−63x。x にも63をかける）。−の前のかっこは、中の全部にかける。
「片方を x、もう片方を 合計−x」は特訓選抜 大問3（方程式の利用）の超定番。人数・代金・速さの混合でも同じ。
こたえ84円切手 10枚／63円切手 10枚

問22種類の人数（合計人数と合計料金）L3 標準目安 7分
問2大人と子どもが合わせて25人います。入場料は大人1人600円、子ども1人350円で、全員の合計は12500円でした。大人は何人ですか。
まず「何型か」を考えよう。何を x にするかは自分で決めて、表か図をかいてから式を作ろう。
①x大人を x 人とする → 子どもは (25−x) 人。
┌──────┬────────┬──────────┐ │ │ 大人 │ 子ども │ ├──────┼────────┼──────────┤ │ 人数 │ x │ 25−x │ │ 料金 │ 600x │ 350(25−x) │ └──────┴────────┴──────────┘
③式料金の合計＝12500。600x＋350(25−x)＝12500。
④解600x＋8750−350x＝12500 → 250x＝3750 → x＝15。
⑤検算大人15人、子ども(25−15)＝10人。600×15＋350×10＝9000＋3500＝12500○。
問われたのは「大人の人数」。x がそのまま答え。子どもの数(10人)を答えにしないよう、最後に「何を聞かれたか」を読み直す。
こたえ大人 15人

問3★過不足算（あまる・たりない）L3 応用目安 8分
問3あめを子どもに配ります。1人に6個ずつ配ると8個あまり、1人に8個ずつ配ると6個たりません。子どもの人数を求めなさい。
まず「何型か」を考えよう。何を x にするかは自分で決めて、表か図をかいてから式を作ろう。
過不足のコツ：変わらないのは「あめの全部の数」。これを2通りの配り方で表して＝で結ぶ。あまる＝＋、たりない＝−（全部＝1人分×人数＋あまり ／ 全部＝1人分×人数−たりない）。
①x子どもの人数を x 人とする（求めるもの）。
③式あめの数を2通りで：6個ずつ＝6x＋8（あまる）、8個ずつ＝8x−6（たりない）。同じあめだから 6x＋8＝8x−6。
④解8＋6＝8x−6x → 14＝2x → x＝7。
⑤検算6×7＋8＝42＋8＝50、8×7−6＝56−6＝50。どちらも50個で一致○。
「たりない」を ＋ にしない（たりない＝−）。配る数が多い方ほど、たりなくなる、と図でイメージすると符号をまちがえない。
過不足は特訓選抜 大問3の定番。「変わらない全体を2通りで表して＝」は、椅子・部屋割り・お金の問題でもそのまま使える。
こたえ子ども 7人（あめは50個）

問4★長椅子型（座れない・最後があまる）L3 応用目安 8分
問4長椅子に生徒を座らせます。1脚に4人ずつ座ると15人がすわれません。1脚に6人ずつ座ると、最後の1脚だけ1人になります。長椅子は何脚あり、生徒は何人ですか。
まず「何型か」を考えよう。何を x にするかは自分で決めて、表か図をかいてから式を作ろう。
椅子の問題も過不足と同じ：変わらない「生徒の人数」を、椅子の脚数 x を使って2通りで表して＝で結ぶ。「最後の1脚だけ1人」＝最後の椅子以外（x−1脚）は6人ずつ、最後だけ1人。
①x長椅子の脚数を x 脚とする。
③式4人ずつだと15人すわれない＝4x＋15（人）。6人ずつだと最後の1脚だけ1人＝(x−1)脚に6人ずつ＋最後1人＝6(x−1)＋1（人）。同じ生徒数だから 4x＋15＝6(x−1)＋1。
④解4x＋15＝6x−6＋1 → 4x＋15＝6x−5 → 15＋5＝6x−4x → 20＝2x → x＝10。
⑤検算脚数10脚。生徒＝4×10＋15＝55人。6人ずつ：9脚に54人、最後の1脚に1人で計55人○。
「すわれない15人」は ＋（席が足りず人があまる＝人数の方が多い）。「最後だけ1人」を 6x＋1 としない＝最後の椅子は6人ではなく1人なので、6人席は (x−1) 脚分だけ。
こたえ長椅子 10脚／生徒 55人
