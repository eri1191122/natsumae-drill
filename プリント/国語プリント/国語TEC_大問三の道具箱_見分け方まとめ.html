これは答えを教えるページじゃない。「どう見分けるか」の順番と決め手を集めたページ。まず自分で解く。そのあと、まちがえたところをここで確認する。数学の「特訓TEC」の国語版だよ🌸

  • 解く前に開かない。解いたあと・まちがえたあとに開く。これが道具箱のルール。
  • 先に見分け方を見てしまうと、「自分で考える練習」がまるごとムダになる。初見で自力で解く。解けなかったら解説を読む。そのあとにここへ来る。
  • でも 大問三の知識は「速く正確に」が点になる。読解に時間を残すために、知識は迷わず解きたい。だから 解いたあとに必ず寄ること。
  • 1回で全部覚えようとしないで。1章につき、持ち帰るのは1つでいい。

  2. ② 答え合わせ・解説を読む
  3. ③ このページを開く → 「さっきの問題で使えた見分け方」を1つ探す
  4. ④ 章末の □ にチェックを入れる（次の1問で使うと決める）
  5. ⑤ 次にやる問題で、その見分け方を1回だけ使ってみる。使えたら、それはもう自分の道具🌸

第1章文の成分の見分け方道具2つ＋早見表
9月の大問三(3)は、一文の中の3つの傍線部が「主語・述語・修飾語・接続語・独立語」のどれかを答える形（2023実物は とても＝修飾語／いや＝独立語／寒いのに＝接続語 の型）。きみの確定弱点「修飾語の識別」がここ。だから、まずこの章から。
道具1 ★成分は「この順番」で判定する★9月に直撃
👀 こう書いてあったら使う：傍線部が「文の成分（働き）」を聞かれたら → 上から順にあてはめる。ぜんぶ同時に考えない
手順 1まず述語をさがす（多くは文末。「どうする・どんなだ・何だ」）
2次に主語（「何が・誰が」＝〜が／は／も）
3残った文節は修飾語の候補（後の語をくわしくする）
4文頭で切れて感動・呼びかけ・応答なら独立語（はい・ああ・いや）
5「〜ので・から・が・のに」で終わり理由・逆接でつなぐなら接続語
なぜ効く5つを同時に見ると迷う。述語→主語→…と順番にあてると、消去法で1つに決まる。修飾語は「残ったもの」で見つかる。
道具2 ★2023本番型の3傍線を、実際に見分ける★9月に直撃
👀 こう書いてあったら使う：1つの文に、傍線が2〜3本ひいてある → 1本ずつ、道具1の順番でさばく
例文ああ、／今日は／とても／寒いので、／私は／上着を／着た。（全7文節）
見分けとても＝修飾語（あとの「寒い」をくわしくする＝連用修飾）／寒いので、＝接続語（「〜ので」で終わり、理由でつなぐ）／ああ＝独立語（感動・呼びかけ）
2023実物本番は同じ型で とても＝修飾語／いや＝独立語／寒いのに＝接続語。「いや」は感動詞→独立語、「寒いのに」は逆接（のに）でつなぐ→接続語。言葉がちがっても、見分け方は同じ。
成分はたらき見分けの決め手
主語「何が・誰が」〜が／は／も（例：花が咲く）
述語「どうする・どんなだ・何だ」多くは文末（例：花が咲く）
修飾語後の語をくわしくする連体修飾＝体言を／連用修飾＝用言を（例：赤い花・とてもおいしい）
接続語前後をつなぐ〜ので／から／が／のに、で終わる文節（例：寒いので、着る）
独立語他と直接つながらない感動・呼びかけ・応答・提示（例：はい、そうです）
迷ったら「述語をさがす」から。述語が決まると、そこに向かう「何が（主語）」も、それをくわしくする「修飾語」も芋づる式に見える。
第2章品詞10種の見分けフロー道具1つ＋早見表
道具3品詞は「3つの質問」で決まる
👀 こう書いてあったら使う：「この単語の品詞は？」と聞かれたら → 上から3つの質問を、順番にあてる
手順 1それだけで意味が分かる？ → 自立語（8種）／付いてるだけ → 付属語（助詞・助動詞）
2形が変わる（活用する）？ → 用言 or 助動詞／変わらない → 名詞・副詞など
3言い切りの形を作る → ウ段＝動詞・い＝形容詞・だ＝形容動詞
なぜ効く10種を丸暗記しても本番で迷う。「活用するか」「言い切りは何で終わるか」の2点だけで、用言3種はすぐ決まる。
例「美しく」→ 言い切り「美しい」＝形容詞／「静かに」→「静かだ」＝形容動詞／「読ま（ない）」→「読む」（ウ段）＝動詞
品詞自立/付属活用見分け方（決め手）例語
名詞自立✕物事の名前。「が・は」を付けて主語になれる（体言）花・学校・私・こと
動詞自立○動作・存在。言い切りがウ段走る・書く・見る・ある
形容詞自立○性質・状態。言い切りがい美しい・寒い・高い
形容動詞自立○性質・状態。言い切りがだ・です静かだ・元気だ
副詞自立✕おもに用言を修飾。「どのように」とても・ゆっくり・きっと
連体詞自立✕体言だけを修飾。活用しないこの・その・大きな
接続詞自立✕文と文をつなぐ。単独で接続語しかし・だから・そして
感動詞自立✕感動・呼びかけ・応答。単独で独立語ああ・はい・いや
助動詞付属○意味を付け加える。活用する付属語れる・ない・た・ます
助詞付属✕関係を示す・意味を添える。活用しない付属語が・を・に・は・ので
道具4まちがえやすい「4つの分かれ道」
👀 こう書いてあったら使う：連体詞と形容詞、副詞と連体詞、で迷ったら → 下の決め手を1つあてる
連体詞⇔形容詞言い切りが「い」なら形容詞。「〜な」の形しかなければ連体詞（大きな箱＝連体詞／大きい箱＝形容詞）
副詞⇔連体詞修飾する相手が用言なら副詞／体言なら連体詞（ゆっくり歩く＝副詞／この本＝連体詞）
形容動詞⇔名詞＋だ「〜な＋名詞」と言えれば形容動詞（静かな海＝形容動詞／×学生な人＝名詞＋だ）
助動詞⇔助詞活用する（形が変わる）なら助動詞（ない・た・れる＝助動詞／が・を・ので＝助詞）
用言（活用する自立語）は動詞・形容詞・形容動詞の3つだけ。付属語で活用するのは助動詞だけ。ここさえ押さえれば「活用する？」の質問で半分決まる。
第3章熟語の構成（二字熟語）早見表＋道具1つ
道具5二字熟語は「下から上に読んでみる」
👀 こう書いてあったら使う：「この熟語の組み立ては？」（ア〜カで選ぶ） → まず下の字から上へ読んでみる
決め手下→上に読んで「〜を・〜に」が入れば目的（エ）。「〜が〜する」になれば主語述語（オ）。読み方を変えるだけで型が見える。
例登山→「山を登る」＝エ／地震→「地が震える」＝オ／急行→「急いで行く」＝ウ（上が下をくわしく）
型組み立て見分けのコツ例（漢検4級以内）
ア似た意味の字を重ねる2字が同じような意味永久・親切・思考
イ反対の意味の字を重ねる2字が反対売買・高低・寒暖
ウ上の字が下の字を修飾上→下で「どんな／どう」急行・青空・細道
エ下が上の目的・対象〈〜を・〜に〉下→上で「〜を」が入る登山・読書・帰宅
オ主語・述語〈〜が〜する〉下→上で「〜が〜する」地震・頭痛・雷鳴
カ上の字が下を打ち消す上が不・無・非・未不足・無害・未定
迷ったらエとオだけ「下→上に読む」テストをする。「〜を」ならエ、「〜が」ならオ。この2つが一番まちがえやすい。
第4章四字熟語・故事成語の覚え方道具3つ
道具6四字熟語は「仲間（群）」でまとめて覚える
👀 こう書いてあったら使う：四字熟語を1つずつバラバラに覚えようとして忘れる → 仲間ごとにまとめる
数字系数字の字が入る。一石二鳥（一・二）／危機一髪（一）。数字に注目すると空欄が埋まる。
反対系反対の字がヒント。異口同音（異⇔同）＝「口をそろえて同じことを言う」。反対の字を手がかりにする。
由来系物語（由来）で覚える。温故知新（古きをたずね新しきを知る）／因果応報（行いに応じた報い）。意味の理由がわかると忘れない。
道具7 ★6月に落とした3語を、道具で覚え直す
6月の特訓で、異口同音・危機一髪・舌先三寸を落とした。バラバラに覚えようとすると、また忘れる。仲間（群）と字のヒントでつかまえ直す。
異口同音いくどうおん＝おおぜいが口をそろえて同じことを言う。反対系（異⇔同）。「口」を「異」に合わせる。
危機一髪ききいっぱつ＝あぶない、ぎりぎりの場面。数字系（一）。末字は髪（かみの毛1本の差）。「発」と書くとバツ。
舌先三寸したさきさんずん＝口先だけのうまい言葉。心がこもっていないこと。体の部位「舌」＋数字「三」でつかまえる。
道具8故事成語は「由来（もとの話）」で意味を出す
👀 こう書いてあったら使う：故事成語の意味を選ぶ／説明する問題 → もとの話を思い出す
矛盾むじゅん＝つじつまが合わない。どんな盾も貫く矛と、どんな矛も防ぐ盾を同時に売った商人の話から。
五十歩百歩ごじっぽひゃっぽ＝大した差はない。五十歩逃げた兵が、百歩逃げた兵を笑った話から。
漁夫の利ぎょふのり＝二者が争う間に、第三者が利益を得る。シギとハマグリが争い、漁師が両方取った話から。
第5章対義語・類義語のパターン早見表2つ
道具9対義語は「どのパターンで反対か」を見る
👀 こう書いてあったら使う：「〜の対義語は？」で思い出せないとき → どのパターンの反対かを考える
なぜ効くやみくもに思い出すより、「打ち消し」「向き」「程度」「正反対」のどれかを決めると、相手の語がすっと出る。
パターン見分け方代表の対（漢検4級以内）
打ち消し「有／無」「不」など打ち消しの字有⇔無・可能⇔不可能・有利⇔不利
向き・増減反対の向き・数量の増減拡大⇔縮小・増加⇔減少・上昇⇔下降
程度（大小）量や度合いが逆大⇔小・多⇔少・長⇔短・高⇔低
正反対の意味概念そのものが逆複雑⇔単純・絶対⇔相対・形式⇔内容
道具10類義語は「特訓頻出の12組」を先に覚える
👀 こう書いてあったら使う：「〜と似た意味の言葉は？」 → この12組から探す
12組進歩≒向上／容易≒簡単／関心≒興味／方法≒手段／欠点≒短所／原因≒理由／大切≒重要／必要≒不可欠／明白≒明らか／変化≒変動／目的≒目標／工夫≒考案
なぜ効く類義語は出る組が決まっている。この12組を先に固めれば、大問三の類義語はほぼ取れる。
第6章大問三の時間配分道具3つ
大問三は漢字読み書き4＋四字熟語3＋文節（または対義語）3＝10問の固定型。ここは取り切る大問。1問に時間をかけず、読解に時間を残すのが作戦。
道具11知識は「1問30秒」で進む
👀 こうなったら使う：大問三を解きはじめるとき → 1問30秒のペースを頭に入れる
やり方知識は思い出せるか、出せないか。30秒で出なければ、悩んでも出にくい。10問＝5分で通過して、残りを読解にまわす。
なぜ効く知識で10分悩むと、配点の大きい読解・記述の時間が消える。国語はきみの強み科目。強みで点を取るために、知識は速く。
道具12迷ったら「△」をつけて次へ
👀 こうなったら使う：「これ、思い出せない…」と思った瞬間 → 問題番号に △ をつけて、次へ行く
やり方迷った問題は△をつけて飛ばす。全部見てから戻ると、あとの問題がヒントになって思い出せることが多い（数学TECと同じ作戦）。
なぜ効く1問に止まると、解けたはずの問題まで時間切れで落とす。飛ばすのは、あきらめじゃなくて作戦。
道具13 ★見直しは「へん・つくり」だけ
👀 こうなったら使う：大問三を書き終えたあと（1〜2分） → 全部読み直さない。字の部品だけ見る
やり方書き取りはへん・つくりの部品ミスが最頻。部品だけを見直す。四字熟語は空欄の字が別字になっていないかだけ（発⇔髪・故⇔古）。
なぜ効く全部読み直すと時間がない。いちばん落とす「部品ちがい」だけを最短で見直せば、書けていた点を守れる。
やること目安ねらい
漢字 読み書き4問各30秒出れば書く。出なければ △。送り仮名まで書く
四字熟語3問各30秒群と数字でさがす。空欄の字が別字でないか
文節／対義語3問各30秒成分は「述語→主語→…」の順（第1章）
見直し1〜2分へん・つくりだけ。△の問題に戻る。ここを残す
13個の道具、ぜんぶ覚えなくていい。覚えるのは、今日の1つだけ。
まず ★9月に直撃 のついた章（第1章の文の成分・第4章の6月失点3語・第6章の見直し）から固めよう。
道具は「読んで覚えるもの」じゃなくて、「1回使って、体で覚えるもの」。だから 解いたあとに開く。👀 の行を読んで「あ、さっきの問題これだ」と思ったら、それが持ち帰る1つ。
ここに書いてある道具は、ぜんぶ きみが実際に落とした場所と、9月にそのまま出るところから作った。つまり、ここは「これから取れるようになる点」ばかり。
