⚙️ さいしょの設定

おとなが 1回だけ やればOK。上から順に押すだけ。
この設定をすると → 子はネットに行けず、ドリルだけ使える状態になります
1正しいアドレスを開くネットあり

Safari（青いコンパス）を開いて、このアドレスを入れます。後ろの「/」まで全部が大事。

https://eri1191122.github.io/natsumae-drill/
⚠️ eri1191122.github.io だけだと 404（開けない）になります。/natsumae-drill/ まで入れてください。
2お気に入りに登録する毎日タップする入口
  1. ドリルが開いたら、共有ボタン（iPadは右上の □に↑）を押す
  2. 「お気に入りに追加」を押す → そのまま「追加」
  3. もし前に作った古いブックマーク（短いアドレスの方）があれば削除する
→ 次からは Safari → お気に入り → これをタップ。404になりません。
※「ホーム画面に追加」は出れば押してもOK。出なくてもお気に入りで十分です。
3子がネットに行けないようにロックこれが本命
  1. 設定アプリ → スクリーンタイム
  2. 「スクリーンタイム・パスコードを使用」で4桁を決める
    （⚠️ ロック解除番号とは違う番号にして、子に教えない）
  3. 「コンテンツとプライバシーの制限」をオン
  4. その中の 「コンテンツ制限」→「Webコンテンツ」
  5. 「許可されたWebサイトのみ」を選ぶ
  6. 「Webサイトを追加」に eri1191122.github.io を入れる
    （最初から入っているApple等の行は左スワイプで削除OK）
→ これで WiFiが繋がっていても、ドリル以外のサイトは全部ブロック。
パスコードが要るので子は外せません。機内モードは不要です。
4結果の受け取り方を知っておく数日に1回
  1. カレンダー上の 「📋記録ボード」を開く（○✕は自動で集まっています）
  2. 「📲結果を送る」を押す → AirDropで親のiPhone/Macへ送る（ネット不要）
  3. AirDropが無ければ 「📁保存してUSBへ」 → USBに保存 → PCでダブルクリック
→ 毎日の確認だけなら、記録ボードを画面で見るだけでもOK。送るのは記録を残したい時だけ。
❓ こまったら

・404になる → アドレスの後ろ /natsumae-drill/ が抜けています。STEP1のコピーを使う。

・サイト内の「もどる」で開けない / 前の画面が古い → 設定 → Safari → 詳細 → Webサイトデータ で github.io を削除 → もう一度開く（最新版になります）。

・「ホーム画面に追加」が出ない → プライベートブラウズを切る。出なくてもお気に入りでOK。

・子のドリルまでブロックされた → STEP3-6で eri1191122.github.io がリストに入っているか確認。
